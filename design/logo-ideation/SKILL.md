# logo-ideation v3.0

Logo设计概念头脑风暴技能。2026年集成AI辅助创意生成、品牌DNA分析、跨文化Logo设计能力。

## 功能说明

帮助进行Logo设计的头脑风暴，梳理品牌DNA，提炼设计方向，提供多个创意概念供选择。

---

## 假设声明（Assumptions）

- 品牌基本信息已知（品牌名称、行业、定位）
- 目标用户群体已明确
- 品牌调性和价值观已清晰

---

## 输入输出（I/O Format）

### 标准输入格式

```
【Logo概念头脑风暴请求】
## 品牌信息
- 品牌名称：[名称]
- 所属行业：[行业]
- 品牌定位：[高端/中端/大众]
## 品牌关键词
- 核心价值：[3-5个关键词]
- 品牌个性：[描述]
```

### 标准输出格式

```
【Logo设计方向建议】
## 品牌DNA分析
[AI分析结果]
## 创意方向（3个）
### 方向1：[名称]
- 核心概念：[描述]
- 视觉元素：[建议]
- 色彩方向：[推荐色系]
## 推荐方向
[首选推荐及理由]
```

## 评审流程（Workflow）

### 阶段1：品牌理解
1. 解析品牌名称内涵
2. 提炼核心价值关键词
3. 定义目标用户特征

### 阶段2：创意发散
1. 列出品牌相关视觉元素
2. 确定风格范围
3. 凝练3-5个创意方向

### 阶段3：概念输出
1. 撰写概念描述
2. 提供延伸设计建议
3. 风险评估
