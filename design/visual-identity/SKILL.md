# visual-identity v3.0

视觉识别系统设计技能。2026年集成AI品牌视觉分析、VI自动生成、品牌一致性检测能力。

## 功能说明

帮助建立完整的品牌视觉识别系统，包括Logo、色彩、字体、图形元素等。

---

## 假设声明（Assumptions）

- 品牌定位和价值观已明确
- 目标受众已定义
- 已有Logo设计或方向

---

## 输入输出（I/O Format）

### 标准输入格式

```
【VI设计请求】
## 品牌信息
- 品牌名称：[名称]
- 品牌定位：[定位描述]
- 目标受众：[受众群体]
## 已有资产
- Logo：[已有/待设计]
- 品牌色：[如有]
```

### 标准输出格式

```
【VI设计手册】
## 视觉识别核心
- Logo使用规范
- 色彩系统
- 字体系统
## 应用规范
- 名片/信纸
- 包装/物料
- 数字媒体
## 设计示例
[应用效果展示]
```

## 2026年新功能

### AI辅助VI设计
| 功能 | 说明 |
|------|------|
| **视觉分析** | AI分析品牌视觉一致性 |
| **VI自动生成** | 从Logo自动生成完整VI |
| **应用预览** | 实时预览各场景应用效果 |
| **一致性检测** | 自动检测VI使用规范 |

### VI系统增强
- 增加动态VI指南
- 增加数字媒体VI规范
- 增加跨文化VI适配

## VI设计流程（Workflow）

### 阶段1：品牌理解
1. 梳理品牌DNA
2. 定义视觉关键词
3. 确定设计方向

### 阶段2：VI系统构建
1. Logo系统设计
2. 色彩系统定义
3. 字体系统选择
4. 图形元素开发

### 阶段3：应用规范
1. 制定应用场景
2. 设计应用示例
3. 编写使用指南
