# slide-deck v3.0

演示文稿规划与制作技能。2026年集成AI幻灯片生成、内容优化、视觉建议能力。

## 功能说明

帮助规划和制作演示文稿，从受众分析到AI辅助生成，提供完整的演示文稿解决方案。

---

## 假设声明（Assumptions）

- 演示目标已明确
- 目标受众已了解
- 演示时长已确定
- 主题内容已准备

---

## 输入输出（I/O Format）

### 标准输入格式

```
【演示文稿规划请求】
## 演示信息
- 演示目标：[获得融资/说服客户/团队对齐]
- 目标受众：[受众背景、关注点]
- 演示时长：[X分钟]
- 演示场景：[线下/线上/录播]
## 主题内容
- 主题：[演示主题]
- 核心信息：[1-3个核心要点]
- 支持材料：[数据、案例]
```

### 标准输出格式

```
【演示文稿规划：XXX】
## 受众分析
- 受众背景：[受众是谁]
- 核心关注点：[关心什么]
- 演讲策略：[如何打动受众]
## 结构规划
| 页码 | 内容 | 时长 | 备注 |
|------|------|------|------|
## 每页要点
### [页码]
- 内容：[内容建议]
- 视觉：[设计建议]
## 演讲建议
- 开场技巧：[...]
- 收尾技巧：[...]
- 时间把控：[...]
```

## 核心框架（Core Frameworks）

| 框架类型 | 适用场景 | 结构特点 |

## 2026年新功能

### AI辅助幻灯片制作
| 功能 | 说明 |
|------|------|
| **AI幻灯片生成** | 根据描述自动生成幻灯片内容 |
| **内容优化** | AI分析并优化文字内容 |
| **视觉建议** | AI提供配色、布局、图表建议 |
| **智能配图** | AI推荐相关图表和图片 |

### 使用示例

```
【AI生成幻灯片】
主题：2026年Q1产品汇报
风格：商务简洁
页数：10页

→ AI自动生成完整PPT结构和内容
→ 提供可编辑的幻灯片大纲
→ 附带视觉设计建议
```

## 工作流程（Workflow）

### 阶段1：需求分析
1. 解析演示目标和受众
2. 确定演示框架
3. 规划页面数量

### 阶段2：内容设计
1. 设计每页核心信息
2. 确定视觉层次
3. 准备支持材料

### 阶段3：AI增强（可选）
1. 使用AI生成幻灯片
2. AI内容优化
3. 获取视觉建议
