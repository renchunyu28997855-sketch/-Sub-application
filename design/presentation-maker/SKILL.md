# presentation-maker v3.0

演示文稿制作技能。2026年集成AI幻灯片生成、智能布局建议、演讲动画设计能力。

## 功能说明

提供演示文稿的设计建议，包括布局、配色、动画、演讲节奏等。

---

## 假设声明（Assumptions）

- 演示目的和受众已明确
- 核心内容已准备好
- 演示时长已确定

---

## 输入输出（I/O Format）

### 标准输入格式

```
【演示文稿请求】
## 演示信息
- 演示目的：[说服/汇报/培训]
- 目标受众：[受众群体]
- 演示时长：[X分钟]
## 内容概要
- 核心主题：[主题]
- 关键要点：[3-5个]
```

### 标准输出格式

```
【演示设计建议】
## 演示结构
[开场/主体/结尾建议]
## 幻灯片设计
### 布局建议
[每页布局指导]
### 配色方案
[推荐色系]
### 动画建议
[过渡动画建议]
```

## 2026年新功能

### AI辅助演示设计
| 功能 | 说明 |
|------|------|
| **智能生成** | 基于内容自动生成幻灯片结构 |
| **布局推荐** | 根据内容类型推荐最佳布局 |
| **演讲动画** | AI设计流畅的演示动画 |
| **节奏优化** | 分析并优化演讲节奏 |

### 演示增强
- 增加数据可视化建议
- 增加互动环节设计
- 增加远程演示优化

## 演示设计流程（Workflow）

### 阶段1：内容规划
1. 确定演示目标
2. 梳理核心内容
3. 设计演示结构

### 阶段2：视觉设计
1. 选择配色方案
2. 设计幻灯片布局
3. 添加视觉元素

### 阶段3：动画设计
1. 设计页面过渡
2. 设计元素动画
3. 演练演讲节奏
