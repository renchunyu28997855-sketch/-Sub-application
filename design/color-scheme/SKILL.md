# color-scheme v3.0

配色方案建议技能。2026年集成AI配色生成、色彩情感分析、可访问性检查能力。

## 功能说明

提供专业配色方案建议，帮助建立视觉统一的设计语言。

---

## 2026年新功能

### AI配色能力
| 功能 | 说明 |
|------|------|
| **AI配色生成** | 基于品牌关键词智能生成配色方案 |
| **色彩情感分析** | 分析色彩传达的情感倾向 |
| **可访问性检查** | WCAG 2.2对比度与色盲友好验证 |

### 评审维度增强
- 增加色彩心理学深度分析
- 增加微梯度设计指导
- 增加跨平台色彩一致性检查

---

## 输入格式

```
【配色方案请求】

## 产品信息
- 产品类型：[App/网站/品牌/运营活动]
- 目标用户：[用户群体描述]

## 配色需求
- 品牌关键词：[科技感/温暖/专业/活力]
- 主色偏好：[已有主色或偏好色系]
- 技术要求：[暗色模式/渐变/色盲友好]
```

---

## 输出格式

```
【配色方案：XXX】

## 品牌关键词
[关键词1]、[关键词2]、[关键词3]

## 配色方案
### 主色
- 色值：#XXXXXX
- 感受关键词：[描述色彩传达的情感]

### 辅色/强调色
[辅色和强调色的色值与用途]

### 中性色
| 用途 | 色值 |
|------|------|
| 深背景 | #XXXXXX |
| 浅背景 | #XXXXXX |
| 深文字 | #XXXXXX |
| 浅文字 | #XXXXXX |

### 功能色
| 状态 | 色值 |
|------|------|
| 成功 | #XXXXXX |
| 警告 | #XXXXXX |
| 错误 | #XXXXXX |

## 渐变方案
```
background: linear-gradient(XXdeg, #XXXXXX 0%, #XXXXXX 100%)
```

## 可访问性检查
- [ ] 对比度达标（≥4.5:1）
- [ ] 色盲友好

## 使用规范
- 主色：70-80%
- 辅色：15-25%
- 强调色：5%
```

---

## 快速工作流

### 1. 需求理解
1. 提取品牌关键词（3-5个）
2. 确认使用场景和平台
3. 确定是否需要暗色模式

### 2. AI配色生成
1. 基于关键词生成3-5个候选方案
2. 情感分析筛选
3. 可访问性验证

### 3. 方案输出
1. 提供完整色板
2. 包含渐变方案
3. 提供使用比例指导

---

## 质量验证

- [ ] 主色、辅色、强调色已定义
- [ ] 功能色（成功/警告/错误/信息）已定义
- [ ] 中性色（背景、文字、边框）已定义
- [ ] 对比度≥4.5:1（WCAG AA）
- [ ] 色盲友好检查通过
- [ ] 渐变色定义完整
