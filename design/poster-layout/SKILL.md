# poster-layout

## 功能说明

提供海报设计的布局和构图建议，提升视觉效果。涵盖构图原则、信息层级、色彩搭配、尺寸规范等内容，帮助创建有效的视觉传播物料。

---

## 假设声明（Assumptions）

### 前提条件
- 海报目的和主题已明确
- 目标受众已了解
- 核心信息已准备好
- 使用场景已确定

### 适用范围
- 商业促销海报
- 活动宣传海报
- 品牌宣传海报
- 招聘海报
- 产品海报
- 社交媒体配图

### 不适用场景
- 高度专业化的艺术创作
- 需要版权素材的完整设计
- 涉及特殊印刷工艺的设计

## 详细工作流程（Detailed Workflow）

### 阶段1：信息梳理

**步骤1：信息重要性排序**
```
检查点：
□ 主标题已确定
□ 关键信息已列出
□ 视觉元素已准备
□ 受众已了解
```

1. 识别核心信息（必须传达的）
2. 区分必要信息和可选项
3. 确定信息优先级
4. 考虑受众最关心的

**步骤2：视觉元素准备**
1. 收集可用素材（图片、Logo等）
2. 评估素材质量
3. 确定是否需要额外素材
4. 准备素材使用计划

### 阶段2：布局规划

**步骤1：选择构图方式**

| 构图类型 | 特点 | 适用场景 |
|----------|------|----------|
| 中心聚焦 | 视觉中心明确 | 人物/产品特写 |
| 分割式 | 信息分区清晰 | 多信息海报 |
| 放射式 | 动感、活力 | 活动宣传 |
| 对角线 | 动态、有张力 | 运动/科技主题 |
| 对称式 | 稳定、正式 | 正式/官方 |
| 留白式 | 简洁、高端 | 品牌/艺术 |

**步骤2：信息区域划分**
1. 确定视觉焦点位置
2. 划分文字区域
3. 确定图片区域
4. 规划视觉动线

### 阶段3：细节设计

**步骤1：标题设计**
1. 字号与层次
2. 字体选择
3. 颜色与背景对比
4. 位置与排列

**步骤2：视觉主体设计**
1. 图片/插画处理
2. 视觉风格的统一
3. 图片与文字的关系

**步骤3：辅助元素**
1. 装饰元素使用
2. 边框/线条使用
3. QR码/联系方式

## 边界情况（Edge Cases）

### 情况1：信息过多
**场景：** 需要传达的信息很多，但海报空间有限
**处理：**
```
【信息过多时的处理策略】

核心挑战：信息过载导致视觉混乱

策略：
1. 信息筛选
   - 绝对必要：必须传达的（1-2个）
   - 重要信息：可以传达的（2-3个）
   - 可选信息：可以省略的

2. 层级简化
   - 主标题：1个核心卖点
   - 副标题：补充说明
   - 详细信息：用QR码引导

3. 视觉简化
   - 减少配色数量
   - 简化图形元素
   - 增加留白

4. 策略转移
   - 考虑分张海报系列
   - 或用主海报+详细页

原则：少即是多
```

### 情况2：仅有文字无图片
**场景：** 没有合适图片素材
**处理：**
```
【纯文字海报设计策略】

无图片时的设计方法：

1. 字体设计
   - 使用装饰性字体
   - 字体大小对比强烈
   - 尝试手写/艺术字体

2. 色块设计
   - 用色块代替图片
   - 几何图形装饰
   - 渐变背景

3. 排版设计
   - 文字即视觉
   - 大小对比创造焦点
   - 留白创造呼吸感

4. 创意字体组合
   - 多种字体搭配
   - 方向/角度变化
   - 描边/阴影效果

参考风格：
- 极简主义
- 孟菲斯风格
- 复古海报风格
```

### 情况3：超大海报（户外广告）
**场景：** 户外大型广告牌
**处理：**
```
【户外广告设计要点】

户外广告特点：
- 观看距离远
- 观看时间短
- 环境复杂

设计要点：
1. 信息极简化
   - 核心信息1-2个
   - 远距离可读
   - 5秒内传达

2. 视觉冲击力
   - 大图/强视觉
   - 高对比度
   - 品牌色突出

3. 环境融合
   - 考虑安装环境
   - 避免与环境融合
   - 确保远距离可辨

4. 字号要求
   - 最小字：30cm+
   - 主标题：60cm+
   - 确保5米外可读

尺寸参考：
-  billboard（广告牌）：14m x 5m
-  候车亭：1.2m x 1.8m
```

### 情况4：社交媒体配图
**场景：** 微信/微博/小红书配图
**处理：**
```
【社交媒体配图策略】

平台特点：
- 屏幕小、缩略图
- 信息流快速划过
- 需要吸引停滑

设计要点：
1. 关键信息前置
   - 在缩略图中也能看到
   - 标题要醒目
   - 品牌标识明显

2. 尺寸适配
   - 微信配图：1:1 或 16:9
   - 微博配图：16:9 或 4:3
   - 小红书：3:4（竖版）
   - 考虑@2x/@3x

3. 留白重要
   - 周围留出安全区
   - 避免边缘放重要信息
   - 考虑平台UI遮挡

4. 品牌一致性
   - 使用品牌色
   - 使用统一模板
   - 建立视觉识别
```

## 错误处理（Error Handling）

### 错误类型1：信息层级混乱
**识别：** 看不出主次，哪个都突出
**处理：**
```
【信息层级诊断】

层级混乱表现：
- 所有信息一样大
- 没有视觉焦点
- 信息争抢注意力

解决方案：
1. 明确层级
   - H1（最大）：主标题
   - H2（中等）：副标题
   - Body（普通）：正文
   - Caption（小）：辅助说明

2. 字号对比
   - 主标题 vs 正文：至少3:1
   - 推荐：H1 48-72px，Body 14-18px

3. 位置安排
   - 视觉焦点区域有限
   - 主信息放焦点区
   - 次信息放边缘
```

### 错误类型2：留白不足
**识别：** 海报看起来拥挤、压抑
**处理：**
```
【留白优化建议】

留白不足问题：
- 所有元素挤在一起
- 视觉呼吸感差
- 信息难以突出

优化方法：
1. 增加边距
   - 最小边距：海报宽度的5%
   - 推荐边距：10-15%

2. 减少信息
   - 必要信息减到最少
   - 其他引导到其他渠道

3. 分散布局
   - 元素间保持距离
   - 用留白分隔区域

留白不是浪费，留白是呼吸。
```

### 错误类型3：色彩过于复杂
**识别：** 使用了太多颜色，不协调
**处理：**
```
【配色简化建议】

配色问题：
- 超过4种主色
- 颜色之间不协调
- 视觉噪音大

简化方法：
1. 颜色数量
   - 主色：1种
   - 辅色：1-2种
   - 中性色：1种
   - 总计：3-4种

2. 色彩来源
   - 从Logo/品牌提取主色
   - 辅色与主色协调
   - 避免高饱和色堆砌

3. 简化技巧
   - 使用渐变减少颜色数
   - 使用黑白灰过渡
   - 考虑单色系设计
```

## FAQ（常见问题）

### Q1：海报的标准尺寸有哪些？
**A：** 常见尺寸：1）A系列（A4/A3/A2/A1）；2）宣传海报（60x90cm、50x70cm）；3）户外广告（ billboards 14x5m、候车亭 1.2x1.8m）；4）社交媒体（1080x1080、1200x628）。根据使用场景选择。

### Q2：如何让海报在远处也能被看到？
**A：** 远处可见的关键：1）主信息字号要大（标题30cm+）；2）使用高对比度配色；3）信息极简化（最多2-3个）；4）品牌色要突出。测试方法：打印后在实际距离查看。

### Q3：如何平衡设计和信息量？
**A：** 平衡原则：1）核心信息≤3个；2）详细信息引导到其他渠道（QR码）；3）设计为信息传达服务，不是炫技；4）如果信息多，考虑系列海报。

### Q4：手写字体可以用在正式海报上吗？
**A：** 可以，但要注意：1）手写体要保证可读性；2）选择风格正式一些的；3）不要全篇使用，增加层级对比；4）正式场合避免过于潦草的风格。

### Q5：海报需要留安全区吗？
**A：** 是的，建议留5-10%的安全区：1）避免重要信息被裁切；2）考虑装裱/悬挂的边缘；3）社交媒体要留出可能被遮挡的区域。

### Q6：如何检查海报的打印效果？
**A：** 打印前检查：1）色彩模式转为CMYK；2）确认分辨率≥300dpi；3）检查黑色文字是否为纯黑（C0 M0 Y0 K100）或叠印；4）如果有渐变，检查是否有渐变断层。
