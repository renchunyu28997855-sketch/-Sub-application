# poster-layout v3.0

宣传海报布局建议技能。2026年集成AI构图建议、动态海报适配、智能信息层级优化能力。

## 功能说明

提供海报设计的布局和构图建议，提升视觉效果。涵盖构图原则、信息层级、色彩搭配等。

---

## 假设声明（Assumptions）

- 海报目的和主题已明确
- 目标受众已了解
- 核心信息已准备好

---

## 输入输出（I/O Format）

### 标准输入格式

```
【海报设计请求】
## 海报信息
- 海报类型：[促销/活动/品牌]
- 主题：[海报主题]
- 目标受众：[受众群体]
## 核心信息
- 主标题：[主要信息]
- 关键信息：[时间/地点/价格]
```

### 标准输出格式

```
【海报设计建议】
## 信息优先级
1. [最重要信息]
2. [次重要]
## 布局方案
### 方案A：[布局名称]
- 构图：[描述]
## 设计要点
- 标题字号建议：[大小]
- 色彩建议：[色值]
```

## 布局流程（Workflow）

### 阶段1：信息梳理
1. 识别核心信息
2. 区分必要/可选项
3. 确定信息优先级

### 阶段2：布局规划
1. 选择构图方式
2. 划分文字/图片区域
3. 规划视觉动线

### 阶段3：细节设计
1. 标题设计（字号/字体/颜色）
2. 视觉主体处理
3. 辅助元素添加
