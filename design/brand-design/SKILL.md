# brand-design v3.0

品牌视觉系统建议技能。帮助建立完整的品牌视觉系统，确保品牌视觉的一致性和可复用性。

## 功能说明

提供品牌视觉系统的专业建议，包括Logo规范、色彩系统、字体系统、图像风格、图标系统、组件规范等。

## 假设声明（Assumptions）

- 品牌定位和核心价值已明确
- 目标用户群体已定义
- 主要应用场景已了解

## 输入输出（I/O Format）

### 标准输入格式

```
【品牌视觉系统请求】
## 品牌信息
- 品牌名称：[名称]
- 所属行业：[行业]
- 品牌定位：[高端/中端/大众]
- 目标用户：[用户群体]
## 品牌资产（已有）
- Logo：[是否完成]
- 配色：[是否有指定颜色]
- 字体：[是否有指定字体]
## 视觉系统需求
- 系统复杂度：[基础/标准/完整]
- 应用场景：[包装/数字产品/空间/广告等]
```

```
【品牌视觉规范手册】

## 3. 字体系统
## 4. 图像风格
## 5. 图标规范
## 6. 组件规范
## 7. 应用场景
```

## 工作流程（Workflow）

### 阶段1：品牌理解
1. 收集现有品牌资产（Logo、配色、字体等）
2. 梳理品牌定位和核心价值
3. 了解目标用户特征
4. 确认主要应用场景

### 阶段2：规范制定
1. Logo规范（版本、安全空间、禁止事项）
2. 色彩系统（主色、辅色、功能色、中性色）
3. 字体系统（字体选择、层级、间距）
4. 图像风格（图片风格、图片使用）
5. 图标规范（风格、尺寸、颜色）
6. 组件规范（按钮、卡片、表单）

### 阶段3：输出整理
1. 按规范模块整理文档
2. 添加图示和示例
3. 提供应用指南
