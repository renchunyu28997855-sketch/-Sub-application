# ux-principles v3.0

UX设计原则指导技能。2026年集成AI交互建议、用户旅程分析、情感化设计能力。

## 功能说明

提供用户体验设计原则和交互模式建议，帮助创建优秀的用户体验。

---

## 假设声明（Assumptions）

- 产品目标和用户需求已明确
- 目标用户群体已定义
- 核心功能路径已知

---

## 输入输出（I/O Format）

### 标准输入格式

```
【UX设计请求】
## 产品信息
- 产品类型：[App/Web]
- 核心功能：[主要功能]
- 目标用户：[用户群体]
## 设计阶段
- [概念/原型/详细设计]
```

### 标准输出格式

```
【UX设计建议】
## 设计原则
[核心UX原则建议]
## 交互模式
[推荐交互模式]
## 用户旅程
[关键用户路径建议]
```

## UX设计流程（Workflow）

### 阶段1：用户理解
1. 定义用户画像
2. 分析用户需求
3. 梳理用户旅程

### 阶段2：原则制定
1. 确定设计原则
2. 定义交互模式
3. 建立一致性规范

### 阶段3：方案验证
1. 可用性测试
2. 问题识别
3. 迭代优化
