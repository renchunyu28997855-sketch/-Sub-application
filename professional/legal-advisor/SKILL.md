# legal-advisor v3.0

法律风险提示技能。2026年集成AI法律文书审查、智能案例匹配、风险量化评估能力。

## 功能说明

提供基础法律风险识别和提示，帮助用户了解常见的法律问题及应对策略。

## 重要声明

**免责声明**：
- 本技能仅提供基础法律信息和风险提示，不能替代专业律师的法律意见
- 涉及重大权益或复杂法律问题，请及时咨询专业律师
- 法律随地区和时间变化，本信息基于中国大陆法律框架

---

## 假设声明（Assumptions）

- 用户处于中国大陆法域范围内
- 用户询问的是民商事领域的基础法律问题
- 问题涉及个人权益保护或轻度法律风险评估

## 工作流程（Workflow）

### 阶段一：问题分类与信息收集
```
检查点：是否明确问题类型
├── 合同风险：识别合同类型、关键条款
├── 劳动法：劳动关系、权利义务
├── 知识产权：权利类型、侵权认定
└── 消费者权益：知情权、退换货
```

### 阶段二：法律分析
```
检查点：是否识别相关法律依据
├── 适用法律条文
├── 权利义务关系
└── 举证责任分配
```

### 阶段三：风险评估
```
检查点：风险等级判断
├── 高风险：涉及重大财产损失
├── 中风险：可通过协商解决
└── 低风险：权利受轻微影响
```

## 问题诊断框架

| 问题类型 | 特征 | 处理方向 |
|----------|------|----------|
| 合同纠纷 | 违约、履行争议 | 证据固定、协商或诉讼 |
| 劳动争议 | 解聘、薪酬、社保 | 劳动仲裁前置 |
| 知识产权 | 侵权、专利、商标 | 保全证据、发送警告函 |
| 消费者维权 | 欺诈、产品质量 | 消协投诉或诉讼 |
| 公司合规 | 资质、监管合规 | 自查整改或专业咨询 |
