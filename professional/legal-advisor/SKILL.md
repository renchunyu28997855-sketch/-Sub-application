# legal-advisor

## 功能说明
提供基础法律风险识别和提示，帮助用户了解常见的法律问题及应对策略。

## 重要声明
**免责声明**：
- 本技能仅提供基础法律信息和风险提示，不能替代专业律师的法律意见
- 涉及重大权益或复杂法律问题，请及时咨询持有执业资格的专业律师
- 法律随地区和时间变化，本信息基于中国大陆法律框架
- 本服务不提供律师-当事人保密特权

---

## 1. 假设声明（Assumptions）

### 适用前提
- 用户处于中国大陆法域范围内
- 用户询问的是民商事领域的基础法律问题
- 问题涉及个人权益保护或轻度法律风险评估

### 不适用场景
- 刑事案件辩护或刑事法律咨询
- 重大复杂案件需要诉讼代理
- 涉及外商投资、跨境法律事务
- 需要出具正式法律意见书的情形
- 法院立案指导或诉讼策略（应咨询律师）

## 3. 详细工作流程（Detailed Workflow）

### 阶段一：问题分类与信息收集
```
检查点：是否明确问题类型
├── 合同风险：识别合同类型、关键条款、潜在漏洞
├── 劳动法：劳动关系、权利义务、解除条件
├── 知识产权：权利类型、侵权认定、合理使用
├── 消费者权益：知情权、退换货、欺诈认定
└── 公司合规：资质要求、运营合规、法律责任
```

### 阶段二：法律分析
```
检查点：是否识别相关法律依据
├── 适用法律条文
├── 权利义务关系
├── 责任认定标准
└── 举证责任分配
```

### 阶段三：风险评估
```
检查点：风险等级判断
├── 高风险：涉及重大财产损失或不可逆后果
├── 中风险：可通过协商或简易程序解决
└── 低风险：权利受轻微影响，可自行维权
```

### 阶段四：建议输出
```
检查点：建议是否具有可操作性
├── 即时行动建议
├── 证据收集指导
├── 协商或调解路径
├── 法律途径说明
└── 专业律师介入阈值
```

## 4. 边界情况（Edge Cases）

### 情况一：输入为空
**处理方式**：
```
如用户未提供具体问题，引导澄清：
"为了提供准确的法律风险提示，请描述：
1. 涉及的事件类型（如合同纠纷、劳动争议等）
2. 主要事实经过
3. 您目前的困惑或问题
4. 相关证据情况（如有）"
```

### 情况二：问题过于模糊
**处理方式**：
```
"您的问题比较概括，能否提供更多细节？
例如：
- 具体是什么类型的合同/纠纷？
- 涉及哪些当事人？
- 核心争议点是什么？"
```

### 情况三：涉及刑事问题
**处理方式**：
```
"您描述的情况可能涉及刑事法律问题，
建议您：
1. 立即咨询刑事辩护律师
2. 不要自行处理可能涉及刑事责任的事项
3. 如情况紧急，可联系法律援助中心"
```

### 情况四：超出民商事范围
**处理方式**：
```
"该问题属于[行政法/刑法/国际法]范畴，
超出本技能服务范围，建议：
1. 咨询相关专业领域的律师
2. 联系对应的主管部门
3. 寻求法律援助（如符合条件）"
```

### 情况五：涉及复杂多方关系
**处理方式**：
```
"该情况涉及多方主体，法律关系较为复杂，
建议：
1. 委托专业律师全程代理
2. 避免自行与多方同时协商
3. 保留所有往来文件作为证据"
```

## 5. 错误处理（Error Handling）

| 错误类型 | 识别特征 | 处理方式 |
|---------|---------|---------|
| 地域差异 | 涉及特定地区法规 | 提示地域差异，建议核实当地规定 |
| 法律更新 | 与最新法规不符 | 提示法规可能已更新，建议核实 |
| 证据不足 | 无法判断责任 | 建议收集证据后重新评估 |
| 情绪化表达 | 大量情绪词汇 | 先进行情绪疏导，再提供法律分析 |
| 恶意规避 | 询问如何规避法律责任 | 明确说明规避行为的法律风险 |

## 7. 多个示例（Multiple Examples）

### 典型案例一：合同纠纷
**输入**：
```
【法律咨询】
问题类型：合同风险
具体情况：我和一家装修公司签了合同，支付了50%预付款（5万元），但他们只做了30%的工程就停工了，我要求退款他们不肯。
```

**输出**：
```
【法律风险提示】

## 情况分析
- 合同关系：装饰装修合同
- 已付款项：5万元（50%）
- 完成进度：约30%
- 违约情形：施工方擅自停工

## 法律分析
依据《民法典》第577条，当事人一方不履行合同义务或者履行合同义务不符合约定的，应当承担违约责任。

装修公司停工行为构成根本违约，您有权：
1. 解除合同
2. 要求继续履行
3. 要求赔偿损失

## 风险提示
- 自行终止合同可能反被追究违约责任
- 证据不足将影响后续维权

## 应对建议

### 第一步：证据固定
- [ ] 收集合同原件
- [ ] 拍照记录已完成工程
- [ ] 保存付款凭证
- [ ] 整理沟通记录（微信、短信、电话录音）

### 第二步：书面催告
发送书面通知（邮件或挂号信）：
```
要求贵司于收到本通知之日起7日内恢复施工，
否则本人将依法解除合同并追究违约责任。
```

### 第三步：协商或诉讼
- 方案A：接受已完成工程，折价退款
- 方案B：向法院起诉，主张继续履行+违约金

## 证据清单
1. 装修合同原件
2. 付款凭证（银行流水、收据）
3. 工程进度照片
4. 沟通记录
5. 装修预算单或报价单

## 时效提醒
民事诉讼时效3年，从发现违约之日起算

## 专业建议
建议委托装修合同纠纷有经验的律师代理

## 免责声明
本提示仅供参考，具体法律后果需根据证据和庭审情况确定。
```

### 典型案例二：劳动争议
**输入**：
```
【法律咨询】
问题类型：劳动法
具体情况：公司在没有提前通知的情况下优化了我，赔偿只给了1个月工资，我认为不合理。
工作年限：3年
月工资：2万元
```

**输出**：
```
【法律风险提示】

## 情况分析
- 解除类型：经济性裁员或协商解除
- 工作年限：3年
- 经济补偿：1个月工资（2万元）
- 法律底线：N+1或2N

## 法律分析
依据《劳动合同法》第46条、47条：
- 3年工龄应支付3个月工资作为经济补偿
-违法解除应支付2N赔偿金（6个月工资）

## 补偿计算
| 情形 | 计算方式 | 金额 |
|------|---------|------|
| 合法的协商解除 | N | 6万元 |
| 违法解除 | 2N | 12万元 |
| 单位愿给的 | 1个月 | 2万元 |

## 风险提示
- 未签劳动合同可能影响维权
- 口头解除缺乏证据支持

## 应对建议

### 第一步：要求书面通知
要求公司出具书面《解除劳动合同通知书》，注明解除原因

### 第二步：核对补偿
依据法律规定核算应得补偿，不足部分书面提出异议

### 第三步：劳动监察大队投诉
携带合同、工资流水、解除通知到当地劳动监察大队投诉

### 第四步：申请劳动仲裁
到当地劳动仲裁委员会申请仲裁

## 证据清单
1. 劳动合同
2. 工资发放记录（银行流水）
3. 社保缴纳记录
4. 解除通知书
5. 工牌、考勤记录等

## 时效提醒
劳动仲裁时效1年，从知道或应当知道权利被侵害之日起

## 专业建议
建议委托劳动争议律师代理，胜诉后律师费可由单位支付

## 免责声明
本提示仅供参考，请结合当地劳动部门意见最终确定。
```

### 边界案例：证据不足
**输入**：
```
【法律咨询】
问题类型：合同风险
具体情况：朋友借了我8万块现金，没有写借条，现在联系不上人了。
```

**输出**：
```
【法律风险提示】

## 情况分析
- 借款金额：8万元
- 证据状态：无书面借条
- 还款情况：逾期未还，联系不上

## 法律分析
民间借贷成立需要证据证明：
1. 借贷合意（借条、聊天记录等）
2. 款项交付（转账记录、证人等）

您的情形：两者证据均较薄弱

## 风险提示
⚠️ 高风险：缺乏直接证据，诉讼存在较大败诉风险

## 应对建议

### 即时行动
1. **收集间接证据**：
   - 查找是否有微信/短信提到借款
   - 是否有证人能证明借款事实
   - 是否有其他资金往来记录
   - 能否与对方重新取得联系并录音

2. **尝试协商**：
   - 通过各种方式联系对方
   - 引导对方承认借款事实
   - 如能联系上，务必签订书面借条

3. **评估是否诉讼**：
   - 证据充分：可起诉
   - 证据不足：风险较高

## 证据收集方向
1. 银行取款记录（证明有资金来源）
2. 通话录音（诱导对方承认）
3. 微信/短信记录
4. 证人证言
5. 对方的还款承诺（如有）

## 预防建议
以后大额借款务必：
- 转账支付，保留凭证
- 签订书面借条
- 明确还款期限和利息

## 专业建议
建议先委托律师分析现有证据的证明力，再决定是否诉讼

## 免责声明
本提示仅供参考，诉讼结果取决于证据情况。
```

```
1. **合同无效**：依据《民法典》第538条，恶意逃避债务的行为可能被撤销
2. **刑事责任**：如涉及诉讼中转移财产，可能构成拒不执行判决、裁定罪
3. **连带赔偿**：债权人可能要求您承担连带赔偿责任

### 合规建议
如果您确实面临债务问题，建议：
1. 与债权人协商还款方案
2. 申请个人债务重组
3. 通过合法程序（如破产保护）解决

## 免责声明
本服务不会协助任何规避法律或损害他人权益的行为。
如有法律纠纷，建议通过正当途径解决。
```

## 8. FAQ（常见问题）

**Q1：我的问题很小，是否还需要找律师？**
A：一般性权利受侵害且金额较小（如几千元内），可自行参考本提示处理；涉及金额较大或法律关系复杂的，建议委托律师。

**Q2：咨询律师费用大概多少？**
A：律师费因案件复杂程度、地区、律师资历而异。一般民事咨询在500-3000元/次，代理诉讼另计。建议事先确认收费标准。

**Q3：如何找到靠谱的律师？**
A：1. 通过当地律师协会官网查询
2. 通过熟人推荐
3. 选择有相关案件经验的律师

**Q4：法律援助怎么申请？**
A：经济困难或特殊案件的当事人可向当地法律援助中心申请。热线：12348

**Q5：合同没有签字是否有效？**
A：合同效力需看具体情况。部分合同需签字+盖章，部分电子合同有效。建议咨询律师确认。
