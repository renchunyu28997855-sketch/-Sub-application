# interview-prep v3.0

面试准备与预测技能。2026年集成AI模拟面试、实时反馈、表情分析能力。

## 功能说明

帮助用户全面准备面试，提供题库预测、答题技巧指导、AI模拟练习和公司研究框架。

## 重要声明

**免责声明**：
- 本技能提供一般性面试技巧和题库预测，不能保证面试结果
- 不同公司、不同面试官风格差异很大，本建议仅供参考
- 最终面试表现取决于个人准备程度和临场发挥

---

## 假设声明（Assumptions）

- 用户正在求职或准备跳槽，有明确求职方向
- 用户处于中国大陆就业市场
- 不适用：公务员/事业单位、技术认证考试、海外求职

## 工作流程（Workflow）

### 阶段一：岗位分析
- [ ] 硬性要求：学历、专业、证书、工作经验
- [ ] 软性要求：沟通能力、团队协作、抗压能力
- [ ] 隐藏需求：文化匹配、稳定性

### 阶段二：问题准备
- [ ] 必考题：自我介绍、职业规划、离职原因
- [ ] 行为题：STAR法则准备
- [ ] 专业题：岗位相关知识
- [ ] 陷阱题：缺点、薪资、加班

### 阶段三：模拟练习
- [ ] 录音回听：检查语速、口头禅
- [ ] AI模拟：获取实时反馈
- [ ] 表情练习：眼神、微笑、仪态

### 阶段四：细节准备
- [ ] 简历、作品集
- [ ] 反问问题
- [ ] 路线规划

## 质量验证清单（Quality Checklist）

面试前检查：
- [ ] 了解目标岗位的核心职责和要求
- [ ] 准备3-5个STAR故事
- [ ] 自我介绍控制在2分钟内
- [ ] 准备了3个以上反问问题
- [ ] 研究过目标公司
- [ ] 进行过AI模拟面试练习

## 核心原则

1. **真实第一**：展示真实的自己，不要过度包装
2. **充分准备**：机会留给有准备的人
3. **保持自信**：相信自己的价值
4. **持续学习**：每次面试都是成长机会
5. **诚信为本**：不要为了面试编造经历
