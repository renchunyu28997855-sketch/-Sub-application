# legal-doc

## 功能说明
帮助用户撰写和理解基础法律文书，提供合同模板、协议起草指南和文书结构参考。

## 重要声明
**免责声明**：
- 本技能提供的是基础法律文书模板和参考，不能替代专业律师的撰写和审核
- 重大法律文书（如涉及大额交易、复杂权利义务）务必委托专业律师
- 法律文书具有法律效力，签署前请仔细阅读并理解全部条款
- 不同地区、不同案件情况可能需要不同的法律文书
- 本模板基于中国大陆法律框架

---

## 1. 假设声明（Assumptions）

### 适用前提
- 文书类型属于常见的民事合同或协议
- 金额不大、关系不复杂
- 当事人双方地位相对平等
- 用户能够理解文书内容

### 不适用场景
- 涉及重大利益（金额较大、涉及重要权利）
- 复杂法律关系（多方当事人、多层嵌套）
- 需要公证或见证的文书
- 诉讼相关的法律文书（起诉状、答辩状等）
- 知识产权、婚姻家庭等复杂领域
- 企业经营相关的合同（建议律师起草）

### 服务边界
- 仅提供基础模板参考
- 不提供具体法律风险评估
- 不保证文书在所有情况下的法律效力
- 不替代律师的专业判断

## 3. 详细工作流程（Detailed Workflow）

### 阶段一：明确文书类型
```
检查点：是否清楚用户的具体需求
├── 合同类：买卖、租赁、借贷、劳务
├── 协议类：和解、调解、保密、竞业
├── 函件类：律师函、催告函、通知函
├── 授权类：委托授权、代理授权
└── 承诺类：保证函、承诺函
```

### 阶段二：收集关键信息
```
检查点：是否获取了必要信息
├── 当事人信息：姓名/名称、身份证/营业执照
├── 标的物/服务：具体是什么
├── 金额/对价：多少、如何支付
├── 时间节点：何时开始、何时结束
├── 权利义务：各自负责什么
└── 违约责任：违反约定怎么办
```

### 阶段三：起草文书
```
检查点：结构是否完整、条款是否清晰
├── 首部：标题、当事人信息
├── 鉴于条款：背景描述
├── 定义条款：关键词定义
├── 正文条款：权利义务约定
├── 违约责任：违反约定后果
├── 争议解决：纠纷如何处理
├── 附则：生效、变更、终止
└── 签署栏：签字盖章位置
```

### 阶段四：风险提示
```
检查点：是否说明了风险点
├── 关键条款的解释
├── 潜在风险点
├── 建议律师审核的情形
└── 签署注意事项
```

## 4. 边界情况（Edge Cases）

### 情况一：用户需求不明确
**处理方式**：
```
"为了帮你起草合适的文书，请说明：
1. 你想签署的是什么类型的文书？（合同/协议/函件）
2. 是和谁之间签署？（个人/公司）
3. 主要内容是什么？（买卖什么、提供什么服务等）
4. 有没有什么特别需要约定的事项？"
```

### 情况二：涉及多方当事人
**处理方式**：
```
"这个情况涉及[两方/多方]当事人，法律关系较为复杂。

我建议：
1. 多方协议需要更详细的条款约定各方的权利义务
2. 建议委托律师起草，以确保各方的利益得到保护
3. 如果坚持自己起草，需要特别注意各方的签字盖章顺序

是否需要我提供一个多方协议的框架？"
```

### 情况三：涉及金额较大
**处理方式**：
```
"该合同涉及金额较大，建议：

1. 务必委托专业律师起草和审核
2. 签订前仔细审核每一条款
3. 考虑是否需要公证
4. 保留所有交易凭证

我不是律师，无法对这类重大交易的法律风险负责。"
```

### 情况四：用户要求写免责声明
**处理方式**：
```
"我不帮你撰写可能用于规避法律责任或损害他人权益的文书。

例如：
- 免除己方全部责任的条款（可能被认定无效）
- 侵犯他人合法权益的约定
- 明显不公平的格式条款

合法的免责条款应当：
- 合理分配风险
- 不违反法律法规强制性规定
- 不侵害他人合法权益
```

### 情况五：用户要修改他人起草的合同
**处理方式**：
```
"如果你有一份已有的合同需要审核，请提供：
1. 完整的合同文本
2. 你的主要顾虑是什么
3. 你想修改哪些条款

我会帮你：
1. 解释条款的含义
2. 指出可能的风险点
3. 建议修改方向

但最终的修改建议需要律师根据具体情况给出。"
```

## 5. 错误处理（Error Handling）

| 错误类型 | 识别特征 | 正确处理方式 |
|---------|---------|-------------|
| 条款模糊 | 关键条款定义不清 | 明确具体表述，避免歧义 |
| 权利义务不对等 | 一方明显吃亏 | 提示并建议修改 |
| 缺少关键条款 | 无争议解决、违约责任等 | 补充必要条款 |
| 格式不规范 | 缺少必要要素 | 按照标准格式补充 |
| 违反法律 | 约定违反强制性规定 | 指出并建议删除 |

```
2. 押金退还：租赁期满，乙方结清所有费用并交还商铺后，甲方无息退还押金

1. 水电费：由乙方承担，按实际使用量结算

4. 其他费用：____________________

1. 甲方负责商铺主体结构及附属设施的正常维修
2. 因乙方使用不当造成的损坏，由乙方负责修复并承担费用
3. 乙方进行装修或改造，须事先征得甲方书面同意，租赁期满或解除合同，装修物归甲方所有

**第八条 合同解除**
1. 乙方逾期支付租金超过____日，甲方有权解除合同
2. 乙方擅自转租，甲方有权解除合同
3. 乙方利用商铺从事违法活动，甲方有权解除合同
4. 甲方提前收回商铺，需提前____日书面通知乙方，并退还剩余租金

**第九条 违约责任**
1. 甲方提前解除合同，需退还剩余租金并支付一个月租金作为违约金
2. 乙方提前退租，需支付两个月租金作为违约金
3. 任何一方违约，违约方应赔偿守约方因此遭受的全部损失

**第十条 争议解决**
本合同履行过程中发生争议，双方应协商解决；协商不成的，向商铺所在地人民法院提起诉讼。

**第十一条 其他约定**
1. 乙方如需在商铺外悬挂招牌，须事先征得甲方及相关部门同意
2. 甲方应保证商铺的权属清晰，不存在权利瑕疵
3. 本合同未尽事宜，双方可另行签订补充协议，补充协议与本合同具有同等法律效力

**第十二条 合同生效**
本合同一式两份，甲乙双方各执一份，自双方签字（或盖章）之日起生效。

## 条款说明

### 装修期
本示例约定了装修期免租金，这是商铺租赁中的常见条款。装修期一般1-3个月，具体视商铺状态和租赁期限而定。

### 押金条款
押金的作用是担保乙方履行合同。建议押金金额为1-3个月租金。

### 转租限制
如需转租，必须在合同中明确约定，否则擅自转租可能构成违约。

## 风险提示

### 甲方风险
- 租客经营不善可能欠租
- 提前退租可能造成空置期损失
- 建议核实租客的经营资质和信用状况

### 乙方风险
- 商铺权属问题
- 无法办理经营资质的风险
- 房东提前收房

## 后续建议

### 强烈建议
1. 签订前让租客提供营业执照复印件
2. 商业租赁可能需要缴纳房产税等税费，建议咨询税务部门
3. 重要条款可适当调整，建议找律师审核

### 可以考虑的补充条款
- 优先续租权
- 租金递增条款
- 不可抗力条款
```

**输出**：
```

## 保密协议

---

**保密协议**

甲方：__________公司
统一社会信用代码：__________
地址：__________

乙方：__________公司
统一社会信用代码：__________
地址：__________

鉴于甲乙双方拟进行[业务合作/技术交流/项目对接]（以下简称"合作项目"），在此过程中一方或双方可能向对方披露涉及商业秘密的保密信息。为保护双方的合法权益，经友好协商，达成如下协议：

**第一条 保密信息定义**
本协议所称"保密信息"包括但不限于：
1. 商业计划、市场策略、客户名单
2. 技术方案、设计图纸、源代码
3. 财务数据、经营状况
4. 双方约定或以书面形式标注为保密的信息
5. 保密信息包括但不限于以书面、口头、电子或其他形式存在的信息

**第二条 保密义务**
1. 接收方应对披露方提供的保密信息予以保密
2. 接收方仅可将保密信息用于本合作项目的目的
3. 接收方不得向任何第三方披露保密信息
4. 接收方仅可向有知悉必要的人员披露保密信息，且应要求其承担同等的保密义务
5. 接收方应采取合理的安全措施保护保密信息，措施等级不低于保护自有保密信息的等级

**第三条 例外情形**
以下信息不属于保密信息：
1. 非因接收方原因已成为公开信息
2. 接收方能证明在收到保密信息前已合法拥有
3. 接收方从有权披露的第三方合法获取
4. 法律法规要求披露的信息（但应及时通知披露方）

**第四条 保密期限**
1. 本协议项下的保密义务自保密信息披露之日起____年内有效
2. 保密期限届满后，接收方仍应对已获取的保密信息承担保密义务
3. 涉及商业秘密的保密信息，保密义务不受期限限制

**第五条 返还或销毁**
合作终止或一方提出要求时，接收方应在____日内：
1. 返还所有保密信息的原件及复制件
2. 或销毁所有保密信息的原件及复制件，并提供书面销毁证明

**第六条 违约责任**
1. 任何一方违反本协议约定，应向守约方支付违约金人民币____元
2. 违约金不足以弥补守约方损失的，违约方应赔偿守约方因此遭受的全部损失
3. 守约方有权采取法律手段阻止进一步的违约行为

**第七条 争议解决**
因本协议引起的争议，双方应协商解决；协商不成的，向甲方所在地人民法院提起诉讼。

**第八条 其他**
1. 本协议自双方签字（盖章）之日起生效
2. 本协议一式两份，甲乙双方各执一份，具有同等法律效力
3. 本协议未尽事宜，双方可另行签订补充协议

甲方（盖章）：__________  乙方（盖章）：__________
代表签字：__________      代表签字：__________
日期：____年____月____日  日期：____年____月____日

---

## 条款说明

### 保密期限
一般商业保密期限为2-5年。但商业秘密本身没有固定期限，只要信息仍属于商业秘密，保密义务就持续。

### 违约金设定
违约金不宜过高也不宜过低。过低无法起到约束作用，过高可能被认定为无效。一般设定为可能损失的合理倍数。

## 风险提示

### 证明困难
保密信息泄露往往难以证明。建议：
- 采取适当的安全措施
- 记录保密信息的接收和使用
- 明确标注哪些信息是保密的

### 范围过宽
保密范围约定过宽可能导致协议难以执行。建议明确具体的保密信息范围。

## 后续建议
1. 涉及核心技术或重要商业机密的保密协议，建议律师起草
2. 可以考虑增加竞业限制条款（但需注意合法性）
3. 重要合作可以考虑签订双向保密协议
```

```
### 事实概述

### 法律依据
```

```
## 重要提示

### 劳动合同是特殊合同
劳动合同不同于一般民事合同，《劳动合同法》对用人单位有较多强制性规定，很多条款不能自由约定。

### 强烈建议
1. 劳动合同的起草和审核务必由专业HR或律师进行
2. 各地劳动法规可能有差异
3. 竞业限制条款有严格的法律要求

### 试用期规定
- 劳动合同期限3个月以上不满1年的，试用期不超过1个月
- 1年以上不满3年的，试用期不超过2个月
- 3年以上固定期限和无固定期限，试用期不超过6个月
- 试用期包含在合同期限内

## 风险提示

### 条款效力
以下条款可能无效：
- 无理由约定试用期
- 约定工伤自负
- 约定加班费包干
- 竞业限制无补偿
- 违约金约定过高

### 员工权利
《劳动合同法》对劳动者的保护较强，违法条款对劳动者没有约束力，但可能给用人单位带来法律风险。

2. 涉及竞业限制、保密等特殊条款，建议律师起草
```

## 8. FAQ（常见问题）

**Q1：网上的合同模板能用吗？**
A：一般性合同模板可以作为参考，但需要注意：1）是否符合当前法律规定；2）是否适合自己的具体情况；3）关键条款是否完整。涉及重大利益的，建议找律师审核。

**Q2：口头协议有法律效力吗？**
A：部分口头协议是有法律效力的（如买卖合同），但举证困难。建议重要事项采用书面形式，保留证据。

**Q3：签合同要注意什么？**
A：1）仔细阅读每一条款，不懂的要问；2）确认对方身份和资质；3）关键条款要明确（金额、期限、违约责任）；4）不要签空白合同；5）保留合同原件。

**Q4：合同签了还能改吗？**
A：可以，但需要双方协商一致。建议在签署前就把条款谈清楚。

**Q5：对方违约了我怎么办？**
A：1）保留证据；2）发书面催告函；3）协商解决；4）协商不成的，可以起诉或仲裁。

## 核心原则

1. **完整性**：条款要完整，不遗漏必要要素
2. **明确性**：表述要清晰，避免歧义
3. **公平性**：双方权利义务要对等
4. **合法性**：条款不能违反法律规定
5. **可执行性**：违约责任要实际可行
6. **专业把关**：重要文书务必律师审核
