# mediator v3.0

冲突调解与协调技能。2026年集成AI冲突分析、方案生成、谈判策略能力。

## 重要声明

- 本技能提供一般性沟通技巧和冲突调解方法，不构成专业法律或心理咨询意见
- 对于涉及法律问题的冲突（家庭暴力、严重人身伤害威胁），建议寻求专业帮助
- 冲突调解需要双方意愿，单方面努力可能无法完全解决问题

---

## 假设声明（Assumptions）

- 冲突属于一般性人际矛盾（家庭、朋友、同事、邻居）
- 双方有继续维持关系的意愿（或至少不反对）
- 用户有沟通的意愿和时间
- 冲突不涉及严重的法律问题

### 不适用场景
- 涉及家庭暴力、虐待 → 建议报警或专业机构
- 涉及严重人身伤害威胁 → 建议报警
- 涉及法律诉讼 → 建议 legal-advisor
- 一方完全拒绝沟通 → 参考 references/examples.md 边界情况
- 严重心理问题 → 建议 therapy-chat

## 核心原则

1. **理解先行**：先理解对方，再表达自己
2. **区分人和行为**：批评行为，不攻击人格
3. **聚焦解决**：目标是解决问题，不是争输赢
4. **尊重边界**：自己的底线和对方的底线都需要尊重
5. **选择权在己**：你无法控制对方，但可以选择自己的回应
6. **不委曲求全**：不是所有关系都值得修复
7. **安全第一**：如果涉及人身安全，优先保护自己

## 冲突类型（详细见 references/conflict-types.md）

| 类型 | 特征 | 处理方向 |
|------|------|---------|
| 利益冲突 | 资源、权力、地位争夺 | 寻求双赢方案 |
| 价值观冲突 | 信念、原则、生活方式差异 | 尊重边界，寻求共识 |
| 沟通冲突 | 误解、表达方式差异 | 改善沟通方式 |
| 角色冲突 | 角色期待不匹配 | 澄清角色定义 |
| 情感冲突 | 积怨、不满、失望 | 情感疏导，修复信任 |
| 身份冲突 | 被尊重、被认可的需求被忽视 | 重建认同感 |

## 解决框架（详细见 references/resolution-frameworks.md）

**利益分析法**：区分立场与利益，寻找共同利益，创造双赢方案，引入客观标准。

**情感修复法**：承认伤害和情绪，理解对方感受合理性，表达真诚歉意，重建信任的小步骤。

**边界确立法**：明确自己的底线，清晰表达边界，一致性执行边界，对方越过时的应对。

## 工作流程（Workflow）

**阶段一：了解冲突全貌**
- [ ] 时间线、触发事件、双方立场、情绪状态、之前尝试

**阶段二：分析冲突结构**
- [ ] 表面冲突、关系层面、身份层面、深层需求

**阶段三：方案设计**
- [ ] 用户能做什么、对方需要做什么、共同利益、妥协空间、第三方帮助

**阶段四：沟通准备**
- [ ] 选择合适时机、准备说什么、预期对方反应、应对负面反应、后备方案
