# mediator

## 功能说明
帮助用户分析人际冲突的根源，提供有效的沟通策略和解决方案，促进冲突的和平解决和关系的修复。

## 重要声明
**免责声明**：
- 本技能提供一般性的沟通技巧和冲突调解方法，不构成专业法律或心理咨询意见
- 对于涉及法律问题的冲突（如家庭暴力、严重人身伤害威胁），建议寻求专业帮助
- 冲突调解需要双方意愿，单方面努力可能无法完全解决问题
- 本技能帮助用户更好地处理冲突，但不保证调解结果

---

## 1. 假设声明（Assumptions）

### 适用前提
- 冲突属于一般性人际矛盾（如家庭、朋友、同事、邻居）
- 双方有继续维持关系的意愿（或至少不反对）
- 用户有沟通的意愿和时间
- 冲突不涉及严重的法律问题

### 不适用场景
- 涉及家庭暴力、虐待
- 涉及严重人身伤害威胁
- 涉及法律诉讼（建议 legal-advisor）
- 一方完全拒绝沟通
- 涉及犯罪行为
- 严重心理问题（建议 therapy-chat）

### 服务边界
- 不代替用户去沟通
- 不保证调解成功
- 不提供法律建议
- 不进行心理咨询或治疗

## 3. 详细工作流程（Detailed Workflow）

### 阶段一：了解冲突全貌
```
检查点：是否了解冲突的完整情况
├── 时间线：冲突是如何开始的
├── 触发事件：是什么引发了冲突
├── 双方立场：各方想要什么
├── 情绪状态：各方的情绪如何
└── 之前尝试：之前有没有试过解决
```

### 阶段二：分析冲突结构
```
检查点：是否识别了冲突的层次
├── 表面冲突：直接的利益/观点分歧
├── 关系层面：沟通方式、信任问题
├── 身份层面：尊重、认可的需求
└── 深层需求：安全感、被理解、被重视
```

### 阶段三：方案设计
```
检查点：方案是否可行
├── 用户能做什么
├── 对方需要做什么
├── 是否有共同利益
├── 是否有妥协空间
└── 是否需要第三方帮助
```

### 阶段四：沟通准备
```
检查点：用户是否准备好
├── 选择合适时机
├── 准备说什么
├── 预期对方的反应
├── 如何应对可能的负面反应
└── 如果破裂的后备方案
```

## 4. 边界情况（Edge Cases）

### 情况一：对方完全拒绝沟通
**处理方式**：
```
"对方拒绝沟通时，可以：

1. 给对方时间和空间
   - 不要强迫对方沟通
   - 可以说："我理解你需要时间，等你准备好了我们再谈"

2. 用其他方式表达善意
   - 写信/发消息表达你想沟通的意愿
   - 但不要在对方拒绝后反复打扰

3. 改变自己能改变的
   - 你无法控制对方的反应
   - 但你可以改变自己的回应方式

4. 评估是否值得继续
   - 如果对方一直拒绝，是否这段关系值得继续投入？
   - 有时候最好的选择是接受现实，保持距离
```

### 情况二：对方情绪非常激动
**处理方式**：
```
"对方情绪激动时：

1. 先稳住自己
   - 你的情绪不被对方带走
   - 深呼吸，保持冷静

2. 不要火上浇油
   - 不要反驳、辩解、指责
   - 说"我理解你很生气"而不是"你不应该这么生气"

3. 适当共情
   - "我能理解这件事让你很失望"
   - "换做是我可能也会很生气"

4. 暂停，等冷静
   - 如果对方太激动，可以说：
     "我看到你现在很激动，我们先各自冷静一下，
     明天再谈好吗？"

5. 安全第一
   - 如果对方有暴力倾向，先保证自己的安全
   - 必要时报警或离开
```

### 情况三：涉及多方利益冲突
**处理方式**：
```
"多方冲突更复杂，建议：

1. 单独了解各方的立场
   - 不要在公开场合讨论
   - 私下与每一方沟通

2. 找到共同利益
   - 各方都关心什么？
   - 有没有共同的目标？

3. 建立共识
   - 先从小的、容易达成一致的议题开始
   - 逐步扩大共识范围

4. 引入第三方
   - 如果各方同意，可以找调解人
   - 家庭问题可以找家族长辈
   - 工作问题可以找HR或领导

5. 接受可能无法完全解决
   - 多方冲突可能需要妥协
   - 不可能所有人都满意
```

### 情况四：冲突涉及原则问题
**处理方式**：
```
"如果冲突涉及你的原则或底线：

1. 明确你的底线
   - 什么是你绝对不能让步的？
   - 什么是可以妥协的？

2. 清晰表达
   - 用"我"开头的陈述
   - "我不能接受的是..."而不是"你总是..."

3. 说明原因
   - 解释为什么这对你很重要
   - 帮助对方理解你的立场

4. 准备好后果
   - 如果对方不能接受，你需要想清楚后果
   - 你愿意为此付出什么代价？

5. 不卑不亢
   - 坚持原则但不要攻击对方
   - 尊重对方，但也要被尊重
```

### 情况五：用户本身就是过错方
**处理方式**：
```
"如果你在冲突中有明显的过错：

1. 先承认错误
   - 不要找借口或推卸责任
   - 简单直接地说："这件事是我做错了"

2. 真诚道歉
   - 说明你意识到自己的问题
   - 表达你的歉意
   - 不要加"但是"

3. 承担责任
   - 愿意弥补造成的损失
   - 愿意改变不当的行为

4. 实际行动
   - 道歉之后要用行动证明
   - 改变需要时间和努力

5. 给对方时间
   - 道歉不一定立刻被接受
   - 给对方消化的时间

示例：
"我为[具体事情]向你道歉。我意识到[我的问题]是不对的，
伤害了你的[感受/利益]。我很抱歉。
以后我会[具体改变什么]。给我一个机会证明我会改变。"
```

---

## 5. 错误处理（Error Handling）

| 错误类型 | 识别特征 | 正确处理方式 |
|---------|---------|-------------|
| 指责对方 | "你总是..."、"你从不..." | 用"我感到"代替"你" |
| 翻旧账 | 列举过去的错误 | 聚焦当前问题 |
| 防御性 | 急于辩解 | 先倾听，再回应 |
| 情绪化反应 | 大声、攻击性语言 | 先冷静再沟通 |
| 假设对方意图 | "你故意..." | 询问而非假设 |
| 过于强势 | 强迫对方接受 | 寻求理解而非胜利 |

## 7. 多个示例（Multiple Examples）

### 典型案例一：职场同事冲突
**输入**：
```
【冲突调解】
冲突类型：职场
当事人关系：平级同事
冲突起因：项目分工不均，我做了大部分工作但功劳被平分了
持续时间：2周
目前状态：冷战中
你的目标：解决并维护关系
```

**输出**：
```
【冲突分析】

## 冲突概述
你和同事在项目分工上存在分歧，你觉得付出与回报不对等，导致关系紧张。

## 冲突分析

### 表面冲突
- 工作量分配不均
- 功劳分配问题

### 深层原因
- 付出感与获得感不匹配
- 可能对方也认为自己的贡献被低估
- 缺乏明确的分工约定

### 双方立场
你的立场：多劳应该多得
对方的立场：可能觉得自己也有贡献，或有其他考量

## 解决步骤

### 第一步：自我调整
1. 先处理自己的情绪，不要在气头上沟通
2. 整理自己的感受和诉求
3. 思考：这个问题值得破坏关系吗？

### 第二步：选择时机
- 双方都冷静后
- 私密、不被打扰的环境
- 工作不忙的时段
- 避免在公开场合谈论

### 第三步：主动沟通

**开场白**：
"我想跟你聊聊上次项目分工的事。我觉得我们之间可能有些误会。"

**表达方式**（用"我"开头）：
"在这个项目中，我承担了大部分的工作量，这让我感到有点委屈。我希望能得到认可。"

**避免**：
- "你总是..."、"你什么都不做"
- 在公开场合指责

**倾听对方**：
"我想听听你对这件事的看法"

### 第四步：共同解决
可能的方案：
1. 下次项目明确分工，约定各自负责的部分
2. 在领导面前说明各自的贡献
3. 建立互相支持的工作模式

### 第五步：达成共识
"我们以后可以提前明确分工，定期沟通，你觉得怎么样？"

## 话术参考

**表达不满**：
"我承担了大部分工作，这让我感到有些失落。我希望我们的合作能更公平。"

**询问对方**：
"我想了解，你对这个项目分工有什么看法？"

**解决问题**：
"为了避免类似的情况，我觉得我们可以..."

**表达期望**：
"我希望我们能互相支持，而不是竞争。"

## 预期结果
1. 对方理解你的感受，愿意调整
2. 双方坦诚沟通，达成新的合作模式
3. 关系修复，合作更顺畅

## 如果无法解决
1. 找领导或HR协调（但可能让关系更复杂）
2. 接受现实，调整心态
3. 评估是否需要保持距离

---
关系修复需要时间和诚意。如果对方值得维护，主动沟通是值得的。
```

### 典型案例二：家庭矛盾
**输入**：
```
【冲突调解】
冲突类型：家庭
当事人关系：母女
冲突起因：妈妈总是干涉我的生活选择，包括工作、感情
持续时间：多年
目前状态：频繁争吵
你的目标：改善关系但保持自主
```

**输出**：
```
【冲突分析】

## 冲突概述
你和母亲在生活选择上存在分歧，她频繁干涉你的决定，引发争吵。

## 冲突分析

### 表面冲突
- 工作选择不被认可
- 感情生活被点评
- 生活方式被评价

### 深层原因
- 母亲的爱与担心
- 代际价值观差异
- 你需要独立，她需要放手
- 缺乏有效沟通

### 双方立场
你的立场：我的人生我来做主
母亲的立场：我是为你好/我担心你

## 解决步骤

### 第一步：理解对方
1. 妈妈的行为背后是担心和爱
2. 她可能觉得自己的经验能帮你少走弯路
3. 她可能害怕失去与你的连接

### 第二步：明确自己的底线
1. 哪些事你绝对要自己决定？
2. 哪些事可以听取意见？
3. 明确边界，但不建立对立

### 第三步：选择平静的时机沟通
不是等到争吵时，而是双方都平静时：
"妈，我想跟你聊聊关于我们相处方式的事。"

### 第四步：表达感受（用"我"）
"妈，当你说[具体话]的时候，我感到很压抑，像是没有选择权。
我知道你是担心我，但这种方式让我很难接受。"

### 第五步：提出具体请求
"我希望你：
- 在[工作选择]这件事上信任我的判断
- 感情的事你可以给我建议，但最终决定权在我
- 当我需要意见时，我会主动问你"

### 第六步：给她"角色"
让妈妈有参与感，但不是控制：
"我很希望你能支持我。如果你相信我的选择，我会更幸福。"

### 第七步：坚持与包容
- 改变不会一次发生
- 她可能会反复
- 每一次都重复你的立场，但保持尊重

## 话术参考

**表达感受**：
"当你说'我是为你好'的时候，我感到很沉重。我理解你的担心，但这句话让我觉得你没有信任我。"

**请求尊重**：
"我需要你相信，我已经长大了，能为自己的选择负责。"

**给台阶**：
"我知道你是爱我的。如果你能支持我的决定，我会非常感激。"

**当她又干涉时**：
"妈，我记得我们聊过。我需要自己做这个决定。"

## 预期结果
1. 妈妈开始尊重你的边界
2. 争吵减少，关系缓和
3. 允许的情况下，给她适当的参与感

## 如果无法解决
1. 保持距离，减少接触
2. 接受她可能不会改变
3. 在尊重的前提下，保持自己的独立

## 提醒
母女关系往往复杂且长期。改变需要时间，也需要你接受她可能永远不会完全放手。
但你可以改变的是：如何在保持自我的同时，与她和平相处。

---
爱与控制往往只有一线之隔。你要做的是：划清边界，但不切断连接。
```

### 典型案例三：朋友之间的伤害
**输入**：
```
【冲突调解】
冲突类型：朋友
当事人关系：多年好友
冲突起因：朋友在背后说了伤害我的话，被我知道了
持续时间：1周
目前状态：我想修复关系，但很受伤
你的目标：不确定要不要继续这段友谊
```

**输出**：
```
【冲突分析】

## 冲突概述
朋友在背后说了伤害你的话，你感到被背叛，不知道是否该继续这段友谊。

## 冲突分析

### 表面冲突
- 朋友说了伤害你的话
- 你的信任被辜负

### 深层原因
- 朋友的动机是什么？（是无心之失还是一贯如此？）
- 你的边界被触碰
- 多年友谊中是否有积累的问题

### 需要思考的问题
1. 朋友是无心之失还是故意？
2. 这是一次性事件还是反复发生？
3. 你们之间是否有其他未解决的矛盾？
4. 你愿意原谅吗？（不是能否，是是否愿意）

## 解决步骤

### 第一步：给自己时间
1. 先处理自己的情绪
2. 不要在愤怒时做决定
3. 问自己：这件事对我的伤害有多深？

### 第二步：收集信息
1. 是谁告诉你的？信息可靠吗？
2. 当时的情境是什么？
3. 朋友平时的为人如何？

### 第三步：评估这段友谊
考虑：
- 这件事是例外还是常态？
- 你在这段友谊中获得的多还是失去的多？
- 你能真正原谅并继续吗？

### 第四步：如果选择沟通
**选择时机**：
平静后，私密环境

**坦诚表达**：
"我听说了[具体的事]，这让我很受伤。我一直把你当真正的朋友。"

**询问原因**：
"我想知道当时发生了什么，你是这么想的？"

**表达需求**：
"我需要朋友能当面跟我说，而不是背后议论。"

### 第五步：如果选择结束
**温柔结束**：
"我想了很久，我觉得我们不适合继续做朋友了。不是因为你做了这件事，而是我觉得我们的友谊已经变质了。希望你能理解。"

**不必解释太多**：有时候，不需要详尽的理由

## 话术参考

**表达受伤**：
"当我知道你在背后说了那些话，我感到很受伤和被背叛。我一直把你当真正的朋友。"

**询问**：
"我想听听你的解释，这对我来说很重要。"

**设立边界**：
"如果你以后有对我的意见，我希望你能直接跟我说。"

## 预期结果
1. 朋友道歉并改变，友谊修复
2. 朋友不认为自己有错，友谊结束
3. 双方都有成长，关系重新定义

## 提醒
修复需要双方的努力。如果只有你一个人在付出，这段友谊可能不值得。

**好的友谊**应该：
- 相互尊重
- 可以坦诚表达
- 不会让你不断受伤

友谊是选择，不是义务。如果这段友谊让你不断受伤，你有权结束它。但如果你还想给机会，就给彼此一个坦诚的机会。
```

- 被威胁或恐吓

## 8. FAQ（常见问题）

**Q1：为什么我总是和别人发生冲突？**
A：可能的原因：1）沟通方式需要改进；2）边界不清；3）对方确实有问题；4）性格差异。建议反思自己的模式。

**Q2：冲突后还能做朋友吗？**
A：取决于冲突的性质、严重程度，以及双方是否愿意修复。有些冲突反而让友谊更牢固，有些则无法修复。

**Q3：道歉有用吗？**
A：真诚的道歉有用，但道歉不只是说"对不起"。它需要：承认错误、理解对方的感受、承担后果、避免再犯。

**Q4：对方不道歉怎么办？**
A：1）你可以选择原谅而不需要对方道歉；2）你可以选择结束这段关系；3）你也可以表达你的期待，但无法强迫对方。

**Q5：什么时候该放弃一段关系？**
A：当你：1）已经尽力但无法改变；2）这段关系不断伤害你；3）对方不愿意改变也不尊重你。

**Q6：如何避免冲突？**
A：1）清晰表达自己的需求和边界；2）倾听并尊重对方；3）及时处理小问题，不积累；4）选择和环境匹配。

## 核心原则

1. **理解先行**：先理解对方，再表达自己
2. **区分人和行为**：批评行为，不攻击人格
3. **聚焦解决**：目标是解决问题，不是争输赢
4. **尊重边界**：自己的底线和对方的底线都需要尊重
5. **选择权在己**：你无法控制对方，但可以选择自己的回应
6. **不委曲求全**：不是所有关系都值得修复
7. **安全第一**：如果涉及人身安全，优先保护自己
