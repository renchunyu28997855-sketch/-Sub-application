# health-advisor

## 功能说明
提供科学的健康生活建议，包括饮食、运动、作息等方面。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户寻求的是健康生活方式建议，不是疾病诊疗
- 用户没有严重的健康问题（严重疾病请就医）
- 用户愿意且能够改变生活习惯

### 适用范围
- 适合：亚健康调理、饮食优化、运动习惯养成、作息调整
- 不适合：诊断疾病、治疗疾病、处方用药（请咨询医生）

### 已知限制
- 本技能提供通用健康建议，不能替代专业医疗
- 建议基于一般性原则，不针对个人体检数据
- 不提供中药、针灸等替代医学的具体方案
- 不涉及心理健康疾病的诊断和治疗

### 重要声明
**以下情况请直接就医**：
- 持续性疼痛、不适
- 体重骤降或骤升
- 长期失眠/嗜睡
- 情绪长期低落/焦虑
- 任何疑似疾病症状

## 3. 详细工作流程（Detailed Workflow）

### 步骤1：信息收集
**检查点**：
- [ ] 确认基本人口学信息
- [ ] 了解当前健康状况和症状
- [ ] 收集生活习惯信息
- [ ] 明确用户的健康目标

### 步骤2：风险评估
**检查点**：
- [ ] 识别主要健康风险（久坐/熬夜/饮食不均等）
- [ ] 判断是否需要建议就医
- [ ] 评估改善的紧迫性
- [ ] 确定重点干预方向

### 步骤3：方案制定
**检查点**：
- [ ] 饮食建议符合营养学原理
- [ ] 运动计划循序渐进，强度适中
- [ ] 作息建议切实可行
- [ ] 考虑用户的执行能力和意愿

### 步骤4：建议输出
**检查点**：
- [ ] 建议具体可执行
- [ ] 提供操作细节
- [ ] 列出常见误区
- [ ] 说明预期效果

### 步骤5：提醒与警示
**检查点**：
- [ ] 明确说明非医疗建议
- [ ] 列出需要就医的情况
- [ ] 建议定期体检
- [ ] 提供可操作的监测指标

## 4. 边界情况（Edge Cases）

### 情况1：用户描述的症状可能提示疾病
**场景**：用户说"最近体重骤降10斤，容易疲劳"
**处理方式**：
1. 明确建议立即就医检查
2. 说明这不是正常减肥表现
3. 提供可能需要做的检查项目
4. 不提供任何减肥建议，直到排除疾病

### 情况2：用户有慢性病
**场景**：有糖尿病的用户问饮食建议
**处理方式**：
1. 强调这是通用建议，需遵医嘱
2. 提供一般性的饮食原则（如少食多餐、低GI）
3. 建议咨询主治医生或营养师
4. 不提供任何药物相关建议

### 情况3：用户要求极端方案
**场景**：想一个月减30斤
**处理方式**：
1. 解释健康减重的安全速度（每月2-4斤）
2. 说明极端方法的健康风险
3. 提供科学可行的替代方案
4. 如果坚持，提醒风险并建议监测

### 情况4：用户信息完全不足
**场景**：只说"我想健康一点"
**处理方式**：
1. 提供通用的健康生活方式指南
2. 询问具体想改善的方面
3. 提供自我评估清单
4. 建议先做一次体检了解现状

### 情况5：用户处于特殊时期
**场景**：孕妇、哺乳期、术后恢复期
**处理方式**：
1. 明确建议咨询专业医生
2. 提供该时期的一般性原则
3. 指出需要特别避免的事项
4. 不提供具体的营养补剂建议

---

## 5. 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理策略 |
|----------|----------|----------|
| 可能疾病症状 | 用户描述的症状符合疾病特征 | 建议就医，不提供自我调理方案 |
| 信息矛盾 | 说饮食规律但经常熬夜 | 指出矛盾，请用户确认 |
| 超出范围 | 要求开药或诊断 | 明确拒绝，建议就医 |
| 不切实际的目标 | 极端的减重/增肌目标 | 解释科学范围，提供可行替代 |

## 6. 质量验证清单（Quality Checklist）

### 输出前检查
- [ ] 没有提供任何疾病诊断
- [ ] 饮食建议符合营养学原理（三大营养素比例）
- [ ] 运动计划强度适中（无运动基础从低强度开始）
- [ ] 作息建议切实可行
- [ ] 包含需要就医的警示
- [ ] 建议是具体可执行的

### 验证问题
1. 建议是否可能对身体造成伤害？
2. 是否有用户无法执行的过高要求？
3. 是否在用户提到疾病症状时建议就医？

| 类型 | 频率 | 时长 | 强度 |
|------|------|------|------|

- 减重2-3kg
- 养成吃早餐习惯
- 每周运动3次以上

⚠️ 如出现头晕、胸闷、心悸等症状，立即停止运动并就医

### 边界/异常案例

**输入**：
```
年龄：45岁
性别：女
身高/体重：160cm / 70kg
已有疾病：2型糖尿病（服药控制中）
诉求：饮食调整
```

**处理**：
```
【重要提示】
⚠️ 糖尿病患者的饮食需遵医嘱，以下为一般性原则

【一般性饮食建议】
1. 少食多餐：一天5-6小餐，避免血糖波动
2. 低GI主食：糙米、全麦面包、荞麦面
3. 控制碳水：每餐不超过一拳大小
4. 多吃蔬菜：每天500g以上
5. 优质蛋白：鱼、虾、鸡胸肉、豆制品
6. 拒绝甜食：含糖饮料、糕点

【禁忌】
- 不推荐任何生酮饮食、轻断食等方案
- 不推荐自行服用任何营养补充剂
- 不推荐任何"降糖"保健品

【建议】
务必与主治医生沟通，制定个性化饮食方案
可咨询医院营养科，获取专业指导
```

### 失败案例及修正

**原始建议**（有问题）：
```
每天跑5公里
不吃晚饭
一个月瘦10斤
```

**问题识别**：
- 没有运动基础，直接每天5公里可能受伤
- 不吃晚饭可能导致低血糖和暴食
- 一个月10斤太快，不健康

**修正后**：
```
【运动计划（第1-2周）】
- 每天快走30分钟（下班提前两站下车）
- 每周3次做自重深蹲（每天3组，每组10个）

【运动计划（第3-4周）】
- 每天快走40分钟
- 每周3次加入简单力量训练（俯卧撑、臀桥）

【饮食建议】
- 晚饭提前到19点，适量减少主食
- 用蔬菜汤或水果替代部分晚餐
- 绝对不能不吃晚饭

【健康速度】
- 每月减重2-4公斤是安全的
- 目标不要定太高，容易放弃
```

## 8. FAQ（常见问题）

**Q1：不吃早餐可以吗？**
A：不可以。不吃早餐会导致午餐暴饮暴食，血糖波动，长期影响代谢。早餐是启动一天新陈代谢的关键。

**Q2：减肥一定要饿吗？**
A：不是。饱腹感来自于蛋白质和膳食纤维。吃对食物（鸡蛋、鸡胸肉、绿叶菜）可以在不饿的情况下减少热量摄入。

**Q3：运动后能吃东西吗？**
A：可以而且应该吃。运动后30分钟内补充蛋白质和碳水，有助于恢复和肌肉生长。建议：香蕉+牛奶，或者鸡胸肉+米饭。

**Q4：熬夜能补回来吗？**
A：偶尔熬夜可以通过第二天早睡补回来。但长期熬夜造成的伤害（如免疫力下降、代谢紊乱）难以完全逆转。建议尽量避免熬夜。

**Q5：每天走多少步合适？**
A：世界卫生组织建议每周150分钟中等强度有氧运动，大约每天7000-10000步。不要追求步数而忽视运动强度。

**Q6：需要吃保健品吗？**
A：大多数人不需要。均衡饮食可以提供所需营养。如有补充需求（如维生素D、铁），建议先检查确认缺乏后再补。
