# schedule-planner v3.0

日程管理与时间规划技能。2026年集成AI智能排程、弹性计划、跨时区协调能力。

## 功能说明

帮助用户规划日程，优化时间分配，提升工作和生活效率。

---

## 假设声明（Assumptions）

- 用户拥有基本的日程管理意识
- 用户能够区分任务的优先级（紧急/重要矩阵）
- 用户有相对固定的工作/生活节奏可供规划参考

---

## 输入输出（I/O Format）

### 标准输入格式
```
【基本信息】
- 日期：[具体日期或"今天"/"明天"/"本周"]
- 角色：[学生/职场/自由职业等]
- 可用时间：[总时长]
- 特殊约束：[如有]

【任务信息】
- 待办任务：[任务列表]
- 预计时长：[每个任务预估]
- 截止时间：[如有]
- 优先级：[高/中/低]
```

### 标准输出格式
```
【今日/本周概览】
- 重点任务数：X
- 预计总时长：X小时
- 完成率目标：X%

【时间块规划】
| 时间 | 任务 | 类型 | 状态 |
|------|------|------|------|
| HH:00-HH:00 | 任务名 | 工作/休息 | □ |

【待办清单】
- [ ] 任务1（优先级：高，截止：XX:XX）
- [ ] 任务2（优先级：中）
```

## 工作流程（Workflow）

### 步骤1：信息收集与解析
- [ ] 确认日期和时间范围
- [ ] 收集所有待办任务
- [ ] 识别截止时间要求

### 步骤2：任务分析与分类
- [ ] 标记有截止时间的任务（紧急）
- [ ] 按优先级排序（高/中/低）
- [ ] 识别任务类型（深度工作/事务性/休息）

### 步骤3：时间块分配
- [ ] 预留缓冲时间（通常10-15%）
- [ ] 将高优先级任务安排在精力最佳时段
- [ ] 确保休息时间合理分布

### 步骤4：输出格式化
- [ ] 时间块无重叠
- [ ] 任务名称清晰
- [ ] 包含复盘提示

---

## 质量验证清单（Quality Checklist）

- [ ] 所有任务都在可用时间内
- [ ] 时间块无重叠冲突
- [ ] 高优先级任务在黄金时段
- [ ] 每个时间块长度合理（15-90分钟）
- [ ] 包含足够的休息间隔
