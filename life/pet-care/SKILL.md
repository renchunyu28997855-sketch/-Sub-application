# pet-care

## 功能说明
提供宠物养护建议，包括喂养、训练、健康护理等方面的指导。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户已经养了宠物或者打算养宠物
- 用户能够为宠物提供基本的生活条件
- 用户关心宠物的健康和福利

### 适用范围
- 适合：常见宠物（猫、狗、兔子、仓鼠、鱼、鸟等）
- 不适合：特殊宠物（爬虫类、异宠等需要专业知识的动物）
- 不适合：宠物疾病诊断（需要兽医）

### 已知限制
- 本技能提供通用养护建议，不能替代专业兽医诊断
- 不提供具体宠物食品品牌的购买建议
- 不提供宠物交易的建议
- 不保证建议对所有个体宠物都适用

### 重要声明
**以下情况请立即就医**：
- 宠物不吃不喝超过24小时
- 严重的呕吐、腹泻
- 呼吸困难
- 受伤出血
- 中毒迹象
- 精神萎靡、意识不清

## 4. 边界情况（Edge Cases）

### 情况1：新手准备养宠物
**场景**：想养猫但完全不了解
**处理方式**：
1. 说明养猫的基本责任
2. 提供养猫前的准备工作
3. 介绍猫咪的基本需求
4. 建议从多大开始养
5. 提醒可能遇到的挑战

### 情况2：宠物不吃东西
**场景**：猫咪2天没吃东西了
**处理方式**：
1. **⚠️ 这需要重视**，2天不吃对猫可能有危险
2. 询问是否有其他症状（呕吐、腹泻、精神差）
3. 建议可能的原因（应激/生病/换粮）
4. **建议立即就医**
5. 提供临时措施（加热食物、换口味）

### 情况3：宠物行为问题
**场景**：狗狗总是拆家/猫咪乱叫
**处理方式**：
1. 分析可能的原因（无聊/焦虑/精力过剩/发情）
2. 提供纠正方法
3. 建议正向训练
4. 考虑是否需要专业帮助
5. 建议耐心和时间

### 情况4：多宠物家庭
**场景**：家里有猫和狗，如何和平相处
**处理方式**：
1. 提供逐步介绍的方法
2. 建议分开喂食和如厕
3. 强调各自的空间
4. 提醒注意的事项
5. 提供和谐相处的建议

### 情况5：老年宠物护理
**场景**：狗狗10岁了，该怎么照顾
**处理方式**：
1. 说明老年宠物的特殊需求
2. 建议增加体检频率
3. 调整饮食（老年粮）
4. 注意关节和牙齿问题
5. 提供舒适的养老环境建议

## 5. 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理策略 |
|----------|----------|----------|
| 可能需要就医 | 描述的症状可能严重 | 建议立即就医 |
| 危险物品 | 宠物可能接触到有毒物质 | 立即提醒就医并说明紧急措施 |
| 不当饮食 | 喂食对宠物有毒的食物 | 明确说明禁忌，提供安全替代 |
| 行为问题复杂 | 简单的建议无法解决 | 建议寻求专业训导师帮助 |

### 典型案例

**输入**：
```
宠物类型：猫
品种：英短
年龄：2岁
性别：母
是否绝育：是
健康状况：健康
饮食情况：正常，但有点挑食

```

**输出**：
```
【宠物概况】
- 类型：猫
- 品种：英国短毛猫
- 年龄：2岁（成年）
- 当前状态：健康，轻微挑食

【喂养指南】
饮食建议：
- 主食：选择优质猫粮（粗蛋白>30%）
- 喂食量：约50-70g干粮/天，分2次喂
- 零食：每周不超过总热量的10%
- 饮水：保持新鲜水源，每天约200ml

喂食频率：
- 成年猫：每天2次（早晚各一次）
- 定时定量，避免自由采食

禁忌食物：
- ❌ 洋葱、大葱、韭菜（导致贫血）
- ❌ 巧克力（中毒）
- ❌ 葡萄、葡萄干（肾损伤）
- ❌ 酒精
- ❌ 咖啡因
- ❌ 骨头（尤其是鸡骨）

【日常护理】
| 项目 | 频率 | 方法 |
|------|------|------|
| 梳毛 | 每天1次 | 使用钢梳，顺毛发方向 |
| 指甲修剪 | 每2-3周 | 猫用指甲剪，只剪尖端 |
| 耳朵清洁 | 每1-2周 | 滴耳液+棉签轻柔清洁 |
| 眼睛清洁 | 每天 | 湿毛巾擦拭分泌物 |
| 刷牙 | 每天 | 猫用牙刷和牙膏 |
| 洗澡 | 每月1次或不洗 | 猫自我清洁能力强 |

【体重管理】
- 英短容易发胖
- 定期称重（每月1次）
- 理想体重：4-5kg
- 胖了就要减少零食，增加运动

【健康提示】
疫苗接种：
- 猫三联（核心疫苗）：首年2-3针，之后每年加强
- 狂犬疫苗：每年一次

寄生虫预防：
- 体内驱虫：每3个月一次
- 体外驱虫：每月一次（春夏秋季）

绝育：
- 已绝育，好处：避免发情、降低某些疾病风险

【用品推荐】
必需品：
- 猫粮盆和水碗（陶瓷或不锈钢）
- 猫砂盆（大一些，至少身体1.5倍）
- 猫砂（豆腐砂或膨润土）
- 猫抓板（磨爪子用）
- 猫窝（也可以不要，猫喜欢高处）

建议购买：
- 逗猫棒（互动玩具）
- 猫爬架（提供活动空间）
- 梳子（钢梳）
- 伊丽莎白圈（备用）
```

### 失败案例及修正

**原始建议**（有问题）：
```
推荐给猫喂牛奶
理由：猫喜欢喝奶
```

**问题识别**：
- 大多数成年猫有乳糖不耐症
- 喝牛奶可能导致腹泻
- 这是常见的养猫误区

**修正后**：
```
【饮食禁忌】

❌ 不要给猫喝牛奶：
- 大多数猫有乳糖不耐症
- 会导致腹泻、呕吐

✅ 如果想给猫喝奶：
- 选择猫咪专用奶粉/羊奶粉
- 幼猫需要专用猫奶粉

✅ 替代零食：
- 猫罐头（适量）
- 煮鸡胸肉（撕小块）
- 猫薄荷饼干
```

## 8. FAQ（常见问题）

**Q1：猫/狗应该多久洗一次澡？**
A：猫：每1-2个月或更少（猫自我清洁能力强）；狗：根据品种和活动量，通常每月1-2次。频繁洗澡会破坏皮肤油脂。

**Q2：宠物不吃东西怎么办？**
A：1. 观察是否有其他症状；2. 检查食物是否新鲜；3. 尝试换一种食物；4. 如果超过24小时不吃，**请就医**。

**Q3：如何判断宠物是否生病了？**
A：观察以下迹象：精神萎靡、食欲下降、呕吐腹泻、呼吸异常、体重骤变、行为改变。有任何异常建议就医。

**Q4：应该什么时候带宠物去打疫苗？**
A：幼猫/犬：6-8周开始，之后每3-4周加强，直到16周。成年后每年加强。具体请咨询兽医。

**Q5：宠物应该绝育吗？**
A：建议绝育。好处：避免发情问题、降低某些疾病风险、减少流浪动物。需要在合适年龄进行（通常6个月以上）。

**Q6：猫狗能吃人的食物吗？**
A：大部分人类食物不适合。绝对不能吃：洋葱、巧克力、葡萄、酒精、咖啡因。可以少量吃：去骨煮鸡肉、白米饭。

## 1. 必需品准备
| 物品 | 说明 |

| 项目 | 频率 |
|------|------|

- **呼叫名字**：用零食建立条件反射

- 封窗！封窗！封窗！
- 收好小物件（防止误食）
- 定期更换猫砂
- 给予足够关注
