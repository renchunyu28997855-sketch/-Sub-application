# music-playlist

## 功能说明
根据用户需求生成音乐播放列表，适合不同场景和心情。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户有听音乐的需求
- 用户能够使用音乐播放平台
- 用户愿意尝试新的音乐风格

### 适用范围
- 适合：流行音乐、经典老歌、轻音乐等主流音乐类型
- 不适合：极其小众的音乐类型（资料有限）、专业古典音乐（需要系统性推荐）

### 已知限制
- 本技能基于公开音乐信息进行推荐
- 不提供VIP歌曲的免费获取方式
- 不推荐非法音乐资源
- 音乐口味因人而异，推荐仅供参考

## 3. 详细工作流程（Detailed Workflow）

### 步骤1：需求分析
**检查点**：
- [ ] 确认听歌场景
- [ ] 了解音乐风格偏好
- [ ] 识别用户的心情状态
- [ ] 了解是否有语言偏好

### 步骤2：歌单生成
**检查点**：
- [ ] 选择符合场景的音乐风格
- [ ] 确保歌曲多样性
- [ ] 考虑情绪匹配
- [ ] 优先推荐经典和口碑之作

### 步骤3：曲目说明
**检查点**：
- [ ] 说明每首歌的特点
- [ ] 解释推荐理由
- [ ] 指出适合的收听时机
- [ ] 提供平台收听方式

### 步骤4：延展建议
**检查点**：
- [ ] 提供风格相似的其他推荐
- [ ] 建议可以尝试的新风格
- [ ] 提供相关艺术家推荐

## 4. 边界情况（Edge Cases）

### 情况1：不知道想要什么
**场景**：只是想听音乐，但不知道想听什么
**处理方式**：
1. 询问当前心情或想达成的效果
2. 提供热门歌单推荐
3. 推荐经典入门的歌曲
4. 建议尝试新的音乐风格

### 情况2：只喜欢某一首特定的歌
**场景**：只喜欢某首歌，想找到类似的
**处理方式**：
1. 分析那首歌的风格特点
2. 推荐相同风格的歌曲
3. 推荐相同歌手的其他歌曲
4. 推荐相同制作人的作品

### 情况3：要求特定年代的音乐
**场景**：只喜欢90年代的歌
**处理方式**：
1. 尊重偏好
2. 找出90年代各风格的代表作品
3. 推荐经典和代表之作
4. 可以适度推荐受该年代影响的后续作品

### 情况4：需要特定语言的音乐
**场景**：想要日语歌曲
**处理方式**：
1. 确认是否喜欢J-pop/J-rock/民谣等
2. 推荐各类型的代表艺人
3. 提供日文歌曲平台歌单
4. 可以推荐动漫OP/ED（如有兴趣）

### 情况5：健身时听歌
**场景**：跑步/健身时听，需要有节奏
**处理方式**：
1. 推荐BPM在140-180之间的歌曲
2. 选择节奏稳定、激励人心的
3. 提供跑步/健身专用歌单
4. 可以推荐热身/放松不同阶段的音乐

---

## 5. 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理策略 |
|----------|----------|----------|
| 需求模糊 | 不知道想听什么 | 引导询问场景或心情 |
| 口味太狭窄 | 只听某种类型 | 尊重但可适度拓展 |
| 推荐不符合口味 | 用户不喜欢 | 了解原因并重新推荐 |
| 歌曲无法找到 | 平台没有 | 提供替代选择 |

### 失败案例及修正

**原始推荐**（有问题）：
```
推荐了100多首歌曲
理由：丰富选择
```

**问题识别**：
- 歌单太长，不利于选择和专注
- 歌曲过多会让选择困难
- 不符合工作/学习场景的需求

**修正后**：
```
推荐8-12首歌曲
- 精选每个场景最经典的曲目
- 确保多样但不过于庞杂
- 提供"更多推荐"作为延展
```

## 8. FAQ（常见问题）

**Q1：哪里可以听这些歌？**
A：推荐使用正规平台：网易云音乐、QQ音乐、酷狗音乐、Spotify、Apple Music等。

**Q2：推荐的都是VIP歌曲怎么办？**
A：国内平台很多歌曲都有免费听的部分。VIP歌曲可能需要会员。可以选择其他免费歌曲替代，或者支持正版。

**Q3：我不喜欢推荐的歌怎么办？**
A：可以告诉我你不喜欢的具体原因（比如太快/太慢/歌词不喜欢），我会根据反馈重新推荐。

**Q4：什么是白噪音/棕噪音/粉噪音？**
A：- 白噪音：所有频率均匀混合的声音（如收音机雪花声）
- 棕噪音：低频更多的噪音（如雷声、瀑布声），比白噪音更柔和
- 粉噪音：介于白噪音和棕噪音之间（如雨声）

**Q5：工作学习听歌会影响效率吗？**
A：因人而异。研究表明，适当的背景音乐可以帮助一些人提高效率，但如果你发现自己经常被歌词或旋律吸引，可能需要换成纯音乐。

**Q6：可以推荐特定语言的歌吗？**
A：当然可以！告诉我你想听什么语言（中文、英文、日文、韩文等）和场景，我会推荐合适的歌曲。
