# fitness-plan v3.0

## 核心功能

**AI健身计划**：根据用户目标、身体数据和训练条件，自动生成个性化训练方案。

---

## 快速开始

### 输入格式
```
【基本信息】
- 性别：男/女
- 年龄：XX岁
- 身高/体重：XXXcm / XXkg
- 体脂率（如知道）：XX%

【健身目标】
- 主要目标：[减脂/增肌/塑形/提升体能]
- 每周可训练次数：X次
- 每次可训练时长：XX分钟

【训练条件】
- 场地：[家用/健身房/户外]
- 可用器械：[哑铃/杠铃/弹力带/无器械]
```

### 输出格式
```
【健身评估】+ 【每周训练计划】+ 【动作示例】+ 【饮食建议】+ 【注意事项】
```

---

## AI能力详解

### 1. AI训练计划生成
根据用户输入的**目标类型、身体数据、训练条件、时间安排**，自动计算：
- 训练频率和周期安排
- 有氧与力量比例
- 动作选择和进阶路径

### 2. 身体状态评估
分析用户提供的数据，评估：
- 当前体能水平（初/中/高级）
- 体脂状况和建议目标
- 肌肉平衡状况（上下身/推拉平衡）
- 主要体态问题

### 3. 智能动作指导
每个动作包含：
- **动作要领**：目标肌群、发力感、呼吸节奏
- **常见错误**：错误姿势及后果
- **调整建议**：如何修正

---

## 输入输出详情

### 完整输入格式
```
【基本信息】性别/年龄/身高/体重/体脂率
【健身目标】主要目标/期望时间/理想体重
【身体状况】运动基础/伤病史/体态问题
【训练条件】场地/器械/每周次数/每次时长
【饮食情况】饮食习惯/蛋白质摄入认知
```

### 完整输出格式
```
【健身评估】当前状态/主要问题/改善重点
【目标分析】目标类型/预计周期/关键指标
【每周训练计划】星期/内容/时长/重点
【力量训练动作】动作/组数/次数/休息
【有氧训练建议】类型/强度/时长
【饮食建议】蛋白质/碳水/热量原则
【注意事项】训练前/中/后安全提示
【进度评估】每X周评估/调整信号
```

## 工作流程

### 步骤1：信息收集
- [ ] 确认基本信息和健身目标
- [ ] 评估运动基础和身体状况
- [ ] 确认训练条件和时间

### 步骤2：计划生成
- [ ] 计算训练频率和强度
- [ ] 分配有氧和力量比例
- [ ] 选择适合的动作

### 步骤3：方案输出
- [ ] 提供每周计划表
- [ ] 详细动作说明
- [ ] 饮食和恢复建议

### 步骤4：进度跟踪
- [ ] 说明评估周期
- [ ] 提示调整信号
- [ ] 进阶路径指引

---

## 边界情况

| 情况 | 处理 |
|------|------|
| 零基础 | 从最低强度开始，培养习惯为主 |
| 时间有限 | HIIT短时高效训练 |
| 目标不现实 | 解释科学范围，设定可达成的阶段目标 |
| 场地受限 | 提供家用/徒手替代方案 |
| 有伤病 | 建议先咨询医生，提供低冲击选项 |

---

- [ ] 饮食建议与目标匹配

## 示例

**输入**：想减肥，健身小白，一周能练3次，每次45分钟，家用哑铃

**输出**：
```
【健身评估】初学者，体脂偏高，需要减脂
【每周计划】
| 星期一 | 有氧+核心 | 45分钟 |
| 星期三 | 上肢力量 | 50分钟 |
| 星期五 | 下肢+有氧 | 50分钟 |

【动作示例】(详见references/examples.md)
【饮食建议】热量缺口300-500大卡，蛋白质1.6g/kg
【注意事项】循序渐进，注意动作标准
```
