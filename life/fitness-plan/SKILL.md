# fitness-plan

## 功能说明
帮助制定个性化的健身计划，实现增肌、减脂或提升体能目标。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户没有严重的运动禁忌症（如心脏病、严重高血压等）
- 用户有一定的自理能力，能够独立完成训练
- 用户有基本的训练场地或器械可用（家用器械/健身房/户外）

### 适用范围
- 适合：初学者到中级健身者、增肌/减脂/塑形
- 不适合：专业运动员（有专业教练）、有严重伤病者、康复训练

### 已知限制
- 本技能不提供医疗性质的康复训练
- 不包含需要专业设备的极端训练方案
- 不保证具体的训练成果（因人而异）
- 不提供营养补剂的专业建议

### 重要声明
**以下情况请先咨询医生**：
- 心脏疾病、高血压、糖尿病
- 严重的关节问题（腰椎间盘突出、膝关节损伤）
- 近期手术恢复期
- 其他医生嘱咐不能运动的疾病

## 3. 详细工作流程（Detailed Workflow）

### 步骤1：信息收集与评估
**检查点**：
- [ ] 确认健身目标和期望
- [ ] 评估运动基础和身体状况
- [ ] 确认训练条件和时间
- [ ] 了解饮食情况

### 步骤2：计划制定
**检查点**：
- [ ] 训练频率符合用户可支配时间
- [ ] 训练强度循序渐进
- [ ] 动作选择符合用户能力和器械条件
- [ ] 平衡有氧和力量训练

### 步骤3：热身/放松设计
**检查点**：
- [ ] 热身动作针对当天训练内容
- [ ] 拉伸覆盖主要肌群
- [ ] 包含关节活动度练习

### 步骤4：饮食指导
**检查点**：
- [ ] 蛋白质摄入量合理
- [ ] 配合目标给出饮食建议（减脂需热量缺口，增肌需热量盈余）
- [ ] 提供简单可行的饮食原则

### 步骤5：进阶说明
**检查点**：
- [ ] 说明训练如何进阶
- [ ] 告知评估周期
- [ ] 提示常见错误

## 4. 边界情况（Edge Cases）

### 情况1：用户完全零基础
**场景**：从未运动过，想开始健身
**处理方式**：
1. 从最低强度开始，以培养习惯为主
2. 前2周以熟悉动作为主，不追求强度
3. 强调动作标准比重量更重要
4. 建议从自重训练开始

### 情况2：时间极度有限
**场景**：每天只能抽出20分钟
**处理方式**：
1. 提供高强度短时训练（HIIT）
2. 动作精简为复合动作
3. 强调训练质量而非时间
4. 建议利用碎片时间活动

### 情况3：目标不现实
**场景**：想一个月练出马甲线/腹肌
**处理方式**：
1. 解释体脂率和肌肉的关系
2. 说明健康减脂的速度（每月2-4斤）
3. 提供可达成的阶段目标
4. 设定合理的期望

### 情况4：训练环境受限
**场景：只有弹力带和瑜伽垫
**处理方式**：
1. 提供仅用弹力带和自重的计划
2. 动作选择适合家用的
3. 增加徒手训练（深蹲、俯卧撑、平板支撑）
4. 提供无需器械的有氧方式（原地跑、跳绳）

### 情况5：用户有伤病
**场景**：有腰间盘突出，但想减肥
**处理方式**：
1. 明确建议先咨询医生
2. 如果医生允许，提供低冲击运动
3. 避免所有弯腰和负重的动作
4. 强调康复优先，减肥其次

## 5. 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理策略 |
|----------|----------|----------|
| 训练过度 | 每天练、强度过大 | 强调休息的重要性，重新规划频率 |
| 目标冲突 | 既要增肌又要减脂 | 解释两个目标的不同阶段策略 |
| 动作错误 | 用户描述动作不标准 | 提供详细的动作描述或建议看视频 |
| 进度太快 | 想一周就看到变化 | 解释肌肉生长需要时间，设定合理周期 |

### 边界/异常案例

**输入**：
```
性别：女
年龄：35岁
身高/体重：160cm / 55kg
健身目标：塑形（练出马甲线、翘臀）
运动基础：每周瑜伽课1次
训练条件：健身房
每周训练：4次
饮食情况：正常，会节食
特殊：产后2年，腹直肌分离（轻度）
```

**处理**：
```
【问题识别】
⚠️ 腹直肌分离需谨慎进行核心训练
⚠️ 有节食史可能影响训练效果
⚠️ 需要盆底肌训练

【建议调整】
1. 前4周以康复训练为主
2. 避免卷腹、仰卧起坐等压迫腹直肌的动作
3. 改用腹式呼吸和鸟狗式等康复动作
4. 节食需停止，保证营养充足

【训练计划（前4周）】
- 训练重点：呼吸练习、盆底肌、核心稳定
- 动作：腹式呼吸、鸟狗式、死虫式、臀桥
- 强度：低，每天可练习

【训练计划（4周后）】
- 逐步加入臀部训练
- 进阶核心训练
- 有氧训练

【饮食建议】
⚠️ 停止节食！
- 每日热量不低于1500大卡
- 蛋白质不低于80g
- 碳水不能太低（影响激素水平）
```

### 失败案例及修正

**原始计划**（有问题）：
```
每天跑步1小时
不吃主食
一个月瘦15斤
```

**问题识别**：
- 每天跑步会过度消耗肌肉
- 不吃主食会导致激素紊乱、暴食
- 一个月15斤太快，不健康且易反弹

**修正后**：
```
【运动计划】
- 每周4次训练（3次力量+1次有氧）
- 力量训练：每次4-5个动作
- 有氧：每周1次30分钟慢跑

【饮食计划】
- 绝对不能不吃主食
- 蛋白质要吃够（1.6g/kg）
- 制造适度热量缺口（300-500大卡）

【目标】
- 每月减重2-4斤
- 减脂为主，保留肌肉
- 建立可长期坚持的习惯
```

## 8. FAQ（常见问题）

**Q1：减脂和增肌可以同时进行吗？**
A：新手可以有一定程度的"同时"，但大多数情况下是分阶段的。先减脂后增肌效率更高。

**Q2：力量训练和有氧哪个先做？**
A：取决于目标。以增肌为主先力量，以减脂为主可以先力量后有氧，或分开不同天进行。

**Q3：训练后肌肉酸痛怎么办？**
A：这是DOMS（延迟性肌肉酸痛），正常现象。处理方法：多喝水、轻度活动、拉伸、充足睡眠。3-5天会自行恢复。

**Q4：多久换一次训练计划？**
A：一般4-8周换一次。如果感觉身体完全适应了，可以提前换。

**Q5：女性练力量会变得很壮吗？**
A：不会。女性雄激素水平低，很难练出大肌肉。力量训练只会让女性身材更紧致、线条更好。

**Q6：训练前要不要吃东西？**
A：可以吃，但不要吃太饱。香蕉、面包、全麦饼干等碳水化合物为主，空腹训练超过1小时可以吃点。
