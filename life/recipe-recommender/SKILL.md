# recipe-recommender v3.0

## 快速开始

**询问方式**：
- "推荐几个家常菜"
- "帮我生成一个减脂食谱"
- "这个食材可以怎么搭配"

---

## 核心功能（2026 AI能力）

### 1. AI食谱生成
根据你的需求，AI自动生成定制化食谱
- 输入：口味偏好、食材、营养目标、用餐人数
- 输出：完整食谱（含步骤、技巧、营养信息）

### 2. 营养分析
智能分析餐食的营养成分
- 热量、蛋白质、碳水、脂肪、膳食纤维
- 与每日推荐摄入量对比
- 提供改进建议

### 3. 食材替代建议
智能推荐可用替代食材
- 过敏替代：无海鲜可用豆腐替代
- 时令替代：冬季西红柿可用白菜替代
- 口味替代：不吃辣可用姜蒜替代

## 输入格式

```
【基本信息】
- 需求：日常/减脂/增肌/特殊场合
- 人数：X人
- 时间：XX分钟以内

【偏好】
- 口味：清淡/重口/麻辣/酸甜
- 难度：简单/中等
- 禁忌：过敏/宗教/不喜欢

【可选-AI增强】
- 营养目标：控制热量/高蛋白/低碳
- 现有食材：[冰箱里有什么]
```

## 详细说明

### 工作流程

**步骤1：需求解析**
- [ ] 确认人数、场景、口味
- [ ] 识别禁忌和特殊需求
- [ ] 判断是否需要AI增强功能

**步骤2：AI增强处理**
- [ ] 营养目标分析
- [ ] 食材可用性检查
- [ ] 替代方案生成（如需要）

**步骤3：食谱筛选**
- [ ] 难度匹配用户能力
- [ ] 时间符合要求
- [ ] 营养搭配均衡

**步骤4：输出优化**
- [ ] 提供烹饪技巧
- [ ] 说明常见错误
- [ ] 给出保存方式

### 边界处理

| 情况 | 处理 |
|------|------|
| 厨房条件有限 | 推荐蒸煮炖类，提供一锅熟方案 |
| 食材有限 | 用现有食材生成方案，教变体 |
| 时间紧迫 | 快手菜+微波炉方案 |
| 食物过敏 | 排除过敏原，推荐替代 |

### 质量检查

- [ ] 食材易获取
- [ ] 新手可完成
- [ ] 时间在要求内
- [ ] 无用户禁忌
- [ ] 营养搭配均衡

## 示例

**输入**：夏天推荐几个简单清淡的菜，2人份，晚餐

**输出**：
```
【推荐】

### 1. 蒜蓉西兰花
- 10分钟/简单/清淡

【食材】西兰花1颗、蒜4瓣
【步骤】焯水后爆蒜翻炒
【技巧】加盐油保翠绿
【营养】80千卡/蛋白质4g/碳水8g

### 2. 番茄虾仁炒蛋
- 15分钟/简单/酸甜
【营养】280千卡/蛋白质22g

【搭配】西兰花+番茄蛋+米饭
比例：1:1:2（蛋白:主食:蔬菜）
```

## AI增强示例

**输入**：减脂，午餐，80kg男性，想控制碳水

**AI分析**：
- 目标：控制碳水，增加蛋白
- 建议：每餐蛋白≥30g，碳水≤50g

**推荐**：鸡胸肉沙拉（蛋白35g/碳水8g/热量320千卡）
