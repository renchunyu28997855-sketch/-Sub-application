# gift-advisor

## 功能说明
帮助用户选择合适的礼品，根据不同场景、对象和预算。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户有送礼需求，想找到合适的礼品
- 用户了解收礼人的基本信息和偏好
- 用户有明确的预算范围

### 适用范围
- 适合：日常送礼、节日礼物、生日礼物、结婚礼物、商务礼品
- 不适合：定制品（需要专业定制流程）、价值过高的礼品（可能有合规问题）

### 已知限制
- 本技能不推荐违法物品（如管制刀具等）
- 不推荐有明显文化冲突的礼品
- 不保证礼品价格的实时准确性
- 不涉及代购或购买渠道

## 3. 详细工作流程（Detailed Workflow）

### 步骤1：场景分析
**检查点**：
- [ ] 确认送礼场景和目的
- [ ] 了解与收礼人的关系
- [ ] 确定预算范围
- [ ] 了解收礼人的基本信息

### 步骤2：需求挖掘
**检查点**：
- [ ] 收礼人的兴趣爱好
- [ ] 收礼人的当前需求
- [ ] 是否有特殊禁忌（如宗教、健康）
- [ ] 是否有隐含偏好

### 步骤3：礼品推荐
**检查点**：
- [ ] 推荐3-5款礼品
- [ ] 价格覆盖预算范围
- [ ] 类型多样化
- [ ] 说明推荐理由

### 步骤4：筛选建议
**检查点**：
- [ ] 根据收礼人特点给出建议
- [ ] 说明优缺点
- [ ] 提供购买建议
- [ ] 提醒注意事项

## 4. 边界情况（Edge Cases）

### 情况1：完全不了解收礼人
**场景**：要送礼但只知道是同事，不知道爱好
**处理方式**：
1. 询问更多信息（年龄、性别、有没有共同朋友了解）
2. 提供通用型礼品（水果、糕点、办公用品）
3. 建议送有文化感的礼物（书籍、文化产品）
4. 避免个性化太强的礼品

### 情况2：预算极低（100元以内）
**场景**：给领导送礼，预算只有100元
**处理方式**：
1. 建议考虑"有心意"的礼品（手写卡片+水果）
2. 不建议送太廉价或太大众的东西
3. 可以考虑DIY礼物
4. 强调礼物的心意而非价格

### 情况3：关系微妙（暧昧期）
**场景**：刚认识不久，想送点东西表达好感
**处理方式**：
1. 礼物不能太贵重（会有压力）
2. 不能太便宜（显得没诚意）
3. 推荐：鲜花、小众香水、精致糕点
4. 避免：太私人的物品（如贴身衣物）

### 情况4：跨文化送礼
**场景**：送给外国朋友中国特色礼物
**处理方式**：
1. 了解对方文化禁忌
2. 优先选择有中国文化特色但无文化禁忌的
3. 茶叶、丝绸、中国结是安全选择
4. 避免：中药、食品添加剂等可能引起误解的

### 情况5：收礼人什么都不缺
**场景**：对方什么都不缺，不知道送什么
**处理方式**：
1. 考虑"体验型"礼物（演出票、旅游券）
2. 送"消耗品"（高品质食品、酒）
3. 送"有心意"的礼物（手工、定制）
4. 直接送钱或礼品卡（有时是最实用的）

## 5. 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理策略 |
|----------|----------|----------|
| 礼品与场景不匹配 | 给病人送保健品过敏 | 提前了解禁忌 |
| 价格超出预算 | 推荐了超预算的礼品 | 重新筛选或说明原因 |
| 关系与礼品不匹配 | 普通朋友送太亲密的礼物 | 按关系调整推荐 |
| 文化冲突 | 送钟给老人（谐音"送终"） | 提供文化避讳清单 |

## 6. 质量验证清单（Quality Checklist）

### 输出前检查
- [ ] 礼品与场景匹配
- [ ] 价格在预算范围内
- [ ] 适合收礼人的身份和关系
- [ ] 没有明显的文化禁忌
- [ ] 有清晰的推荐理由
- [ ] 包含购买建议

### 验证问题
1. 是否有文化禁忌需要避免？
2. 礼品是否太私人化导致尴尬？
3. 是否考虑了收礼人的实际需求？

## 7. 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
场景：朋友乔迁新居
对象：朋友（关系较好）
年龄：28岁
预算：300-500元
兴趣：喜欢做饭、有猫、文艺青年
```

**输出**：
```
【场景分析】
- 送礼场景：乔迁之喜
- 关系亲疏：好友
- 预算定位：中等偏上

【推荐礼品】
| 礼品 | 价格区间 | 特点 | 适合原因 |
|------|----------|------|----------|
| 猫爬架 | 200-400 | 实用、有心意 | 朋友有猫，新家给猫一个礼物 |
| 装饰画/挂画 | 150-300 | 文艺、有格调 | 符合文艺青年品味，装点新家 |
| 香薰蜡烛套装 | 150-250 | 高颜值、有仪式感 | 提升生活品质 |
| 红酒/清酒 | 150-300 | 经典之选 | 搬家小酌庆祝 |
| 绿植盆栽 | 80-200 | 寓意好、实用 | 净化空气，竹报平安 |

【高性价比推荐】
1. **大型绿植**（如龟背竹、琴叶榕）
   - 价格：150-250元
   - 推荐理由：寓意好（竹报平安），净化空气，有猫的话要注意绿植毒性，龟背竹对猫有毒，建议选择无毒绿植
   - 购买渠道：花市或大型超市
   - 注意：选择对猫安全的植物（猫草、吊兰）

2. **香薰套装**
   - 价格：150-300元
   - 推荐理由：高颜值，有仪式感，营造家的氛围
   - 推荐品牌：Diptyque（贵）、观夏、野兽派
   - 适合送给有生活品质追求的朋友

【避坑建议】
1. 避免送尖锐物品（刀、剪刀——有"切断联系"的寓意）
2. 避免送钟（"送终"的谐音）
3. 避免送杯子（"一辈子"的谐音——除非对方结婚）
4. 避免送伞（"散"的谐音）
5. 送单数不要送双数（部分地区习俗）

【特别建议】
如果朋友喜欢做饭，可以送：
- 高品质调料礼盒（如日本酱油、意大利醋）
- 特色食材（如松露盐、藏红花）
- 精致餐具（如果知道对方风格）
```

### 边界/异常案例

**输入**：
```
场景：给导师送教师节礼物
对象：导师（大学教授，50岁）
关系：师生（比较熟）
预算：200元以内
偏好：不知道具体喜好
```

**处理**：
```
【问题识别】
⚠️ 教师节礼物讲究"尊师重道"
⚠️ 预算有限但要有心意
⚠️ 不了解导师具体喜好

【推荐礼品】
| 礼品 | 价格 | 特点 | 适合原因 |
|------|------|------|----------|
| 钢笔 | 100-200 | 实用、有品质 | 教授常用，有文化感 |
| 茶叶礼盒 | 100-200 | 通用、养生 | 中老年教授普遍爱喝茶 |
| 书籍 | 50-150 | 有心意 | 如果知道研究方向可以送专业书 |
| 按摩仪 | 150-200 | 健康、实用 | 老师经常伏案工作 |

【心意型推荐】
1. **手写贺卡 + 鲜花**
   - 贺卡亲自写感谢的话，比礼物本身更打动人心
   - 康乃馨或向日葵（代表尊敬）

2. **定制笔记本**
   - 扉页可以写上祝福语
   - 老师可以用很多年

【避坑】
- 不要送太贵重的（会让老师有压力）
- 不要送太私人的（如皮带、领带）
- 不要送太大众的（如营养品，除非确定需要）
```

---

### 失败案例及修正

**原始推荐**（有问题）：
```
推荐：按摩椅（家用）
理由：实用，能帮助放松
```

**问题识别**：
- 按摩椅价格昂贵（5000+），超出一般送礼预算
- 体积太大，新搬家可能没地方放
- 收礼人可能不喜欢这种占空间的礼物

**修正后**：
```
推荐：小型颈椎按摩仪
- 价格：200-400元
- 体积小，不占地方
- 实用，每天可用
- 有心意但不贵重

或者：高品质足浴盆
- 价格：200-500元
- 实用养生
- 有助于缓解疲劳
```

## 8. FAQ（常见问题）

**Q1：不知道送什么怎么办？**
A：1. 选择通用型礼品（鲜花、水果、食品礼盒）；2. 考虑"消耗品"（红酒、茶叶、香水）；3. 送"体验"（演出票、礼品卡）。

**Q2：朋友说"不要送礼"还要送吗？**
A：如果关系很好，可以送有心意的小礼物，不必贵重。一束花、一盒巧克力都能表达心意。

**Q3：给外国人送什么中国特色礼物好？**
A：茶叶、丝绸、中国结、折扇、书法作品、脸谱艺术品。注意了解对方文化是否有禁忌。

**Q4：结婚礼物送什么好？**
A：1. 红包（最实用）；2. 家电礼盒；3. 床品套件；4. 餐具套装。避免送单个的杯子（除非是套装）。

**Q5：生日礼物和节日礼物有区别吗？**
A：有。生日礼物可以更个性化，节日礼物通常更传统一些。比如端午节送粽子，中秋节送月饼。

**Q6：包装重要吗？**
A：非常重要。好看的包装能提升礼物的仪式感。如果预算有限，可以买普通礼物+精美包装。
