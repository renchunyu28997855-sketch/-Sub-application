# gift-advisor v3.0

礼品推荐建议技能。2026年集成AI礼品搭配、个性化定制建议、场景化推荐能力。

## 功能说明

帮助用户选择合适的礼品，根据不同场景、对象和预算。

---

## 假设声明（Assumptions）

- 用户有送礼需求，想找到合适的礼品
- 用户了解收礼人的基本信息和偏好
- 用户有明确的预算范围

---

## 输入输出（I/O Format）

### 标准输入格式
```
【基本信息】
- 场景：[生日/节日/纪念日/乔迁/婚礼/商务/探病]
- 对象：[父母/伴侣/朋友/同事/领导/老师/孩子]
- 关系：[至亲/一般/暧昧/商务]
- 预算：[XXX元以内/XXX-XXX元]

【收礼人信息】
- 年龄：XX岁
- 兴趣：[如：阅读/运动/数码/美食]
```

### 标准输出格式
```
【场景分析】
- 送礼场景：...
- 关系亲疏：...
- 预算定位：...

【推荐礼品】
| 礼品 | 价格区间 | 特点 | 适合原因 |

【避坑建议】
1. 避免送...
2. 避免送...
```

## 2026年新功能

| 功能 | 说明 |
|------|------|
| **AI礼品搭配** | 根据收礼人画像生成个性化方案 |
| **场景化推荐** | 特定节日/场合的精选礼品清单 |

---

## 工作流程（Workflow）

1. **场景分析** - 确认送礼场景和目的、关系亲疏、预算范围
2. **需求挖掘** - 收礼人兴趣爱好、当前需求、特殊禁忌
3. **礼品推荐** - 推荐3-5款礼品，价格覆盖预算，类型多样化
4. **筛选建议** - 根据收礼人特点给出建议，说明优缺点

---

## 质量验证清单（Quality Checklist）

- [ ] 礼品与场景匹配
- [ ] 价格在预算范围内
- [ ] 适合收礼人的身份和关系
- [ ] 没有明显的文化禁忌
- [ ] 有清晰的推荐理由
