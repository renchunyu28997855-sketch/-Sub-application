# movie-recommender

## 功能说明
根据用户喜好推荐电影，提供观影指南和简要影评。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户有观影需求，想找到合适的电影
- 用户有基本的观影偏好描述能力
- 用户能够访问常见的观影平台

### 适用范围
- 适合：主流电影类型（剧情、动作、喜剧、科幻等）
- 不适合：非常小众的独立电影（资料有限）、专业影评分析

### 已知限制
- 本技能基于公开信息和常见影评进行推荐
- 不保证实时票房和平台可观看状态
- 不提供盗版资源
- 不包含可能被认为不适当的内容（会提示观影注意）

## 3. 详细工作流程（Detailed Workflow）

### 步骤1：偏好分析
**检查点**：
- [ ] 了解用户喜欢的电影类型
- [ ] 识别用户的观影场景
- [ ] 了解是否有特殊要求
- [ ] 询问是否有看过的经典作为参考

### 步骤2：匹配推荐
**检查点**：
- [ ] 从喜欢的类型中筛选高分电影
- [ ] 考虑用户提到的"喜欢的电影"作为参考
- [ ] 确保推荐多样性
- [ ] 优先推荐经典和口碑之作

### 步骤3：详细介绍
**检查点**：
- [ ] 提供无剧透的剧情简介
- [ ] 说明推荐理由
- [ ] 指出适合的人群和场景
- [ ] 提醒可能的争议内容

### 步骤4：场景匹配
**检查点**：
- [ ] 根据观影人数推荐
- [ ] 根据场合推荐
- [ ] 根据时间段推荐
- [ ] 提供备选方案

## 4. 边界情况（Edge Cases）

### 情况1：完全没看过几部电影
**场景**：刚入门电影，想开始看电影
**处理方式**：
1. 推荐各类型的入门经典
2. 不要一次推荐太多
3. 说明每部电影的类型特点
4. 建议按类型逐步探索

### 情况2：要求很模糊
**场景**：只想看电影，不知道想要什么类型
**处理方式**：
1. 询问观影场景和人数
2. 推荐几部不同类型的经典
3. 让用户选择感兴趣的
4. 提供当下热门推荐

### 情况3：只看某类特定电影
**场景**：只看恐怖片，其他都不看
**处理方式**：
1. 尊重用户偏好
2. 提供恐怖片细分推荐（血腥/心理/恐怖喜剧等）
3. 指出恐怖片佳作
4. 可建议尝试恐怖片与其他类型结合（如恐怖喜剧）

### 情况4：带孩子看电影
**场景**：周末带6岁孩子看电影
**处理方式**：
1. 排除R级（成人）内容
2. 推荐动画或家庭友好电影
3. 指出可能不适合的部分
4. 建议观看前的准备

### 情况5：特定导演或演员偏好
**场景**：只喜欢诺兰导演的电影
**处理方式**：
1. 推荐该导演的其他作品
2. 指出导演的风格特点
3. 推荐相似风格的导演作品
4. 可以提供导演作品观看顺序

---

## 5. 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理策略 |
|----------|----------|----------|
| 需求不明确 | 不知道想看什么 | 引导询问观影场景和偏好 |
| 口味太狭窄 | 只看某种类型 | 尊重但可适度推荐拓展 |
| 推荐不符合口味 | 用户说推荐的不喜欢 | 了解更多不喜欢的原因 |
| 信息不准确 | 电影信息有误 | 承认错误并修正 |

## 6. 质量验证清单（Quality Checklist）

### 输出前检查
- [ ] 电影评分和基本信息准确
- [ ] 推荐理由充分
- [ ] 无重要剧透信息
- [ ] 适合人群描述准确
- [ ] 场景匹配建议合理
- [ ] 包含观看注意事项

### 验证问题
1. 是否有年龄不适合的内容需要提醒？
2. 是否有观影门槛（如需要了解前作）？
3. 推荐是否考虑了用户的口味偏好？

## 8. FAQ（常见问题）

**Q1：哪里可以看这些电影？**
A：推荐使用正规平台：爱奇艺、优酷、腾讯视频、哔哩哔哩（部分免费）。院线电影可等下映后在流媒体平台观看。

**Q2：老电影值得看吗？**
A：当然！经典老电影往往比新电影更有深度。如《肖申克的救赎》《阿甘正传》等，拍摄技术虽然不如现在，但故事和演技绝对经典。

**Q3：如何避免踩雷烂片？**
A：1. 看评分（豆瓣7分以上相对可靠）；2. 看评价数量（太少可能是刷分）；3. 避开流量明星+烂剧情的组合；4. 尊重个人口味。

**Q4：电影太长没时间看怎么办？**
A：1. 选择短片（90分钟内）；2. 分两次看；3. 了解剧情后看解说视频；4. 选择注意力集中的时候看。

**Q5：看完一部好电影后很失落怎么办？**
A：这是"看完好电影后的空虚感"（Post-Movie Depression），正常的。解决方法：1. 立即再看一部相关电影；2. 写观后感；3. 看幕后花絮；4. 讨论区看其他人的解读。

**Q6：可以推荐一个人看完很感动的电影吗？**
A：当然可以。根据你想要的感觉推荐：
- 亲情感动：《你好，李焕英》《7号房的礼物》
- 爱情感动：《泰坦尼克号》《情书》
- 人生感悟：《阿甘正传》《当幸福来敲门》
