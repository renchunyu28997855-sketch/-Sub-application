# movie-recommender v3.0

电影推荐系统技能。2026年集成AI观影口味分析、个性化影单生成、剧情概要无剧透能力。

## 功能说明

根据用户喜好推荐电影，提供观影指南和简要影评。

---

## 假设声明（Assumptions）

- 用户有观影需求，想找到合适的电影
- 用户有基本的观影偏好描述能力
- 用户能够访问常见的观影平台

---

## 输入输出（I/O Format）

### 标准输入格式
```
【基本信息】
- 观影人数：X人
- 观影场景：[独自/情侣/家庭/朋友聚会]

【偏好设置】
- 喜欢的类型：[动作/爱情/科幻/悬疑/喜剧/恐怖/动画]
- 偏好的风格：[文艺/商业/独立/大片]
- 年代偏好：[经典/近期/都可以]

【特殊要求】
- 是否烧脑：[喜欢/不喜欢/都可以]
- 结局偏好：[喜欢HE/接受BE/无所谓]
```

### 标准输出格式
```
【推荐电影】
| 电影 | 年份 | 评分 | 时长 | 类型 | 推荐指数 |

【强推荐】：电影名
**一句话推荐**：...

【观影场景匹配】
- 独自观影：推荐...
- 情侣约会：推荐...
```

## 2026年新功能

| 功能 | 说明 |
|------|------|
| **AI观影口味分析** | 分析用户偏好，预测喜欢程度 |
| **无剧透剧情概要** | AI生成不影响观感的剧情介绍 |
| **观影情绪匹配** | 根据想哭/想笑/想刺激等情绪推荐 |
| **续集关联提醒** | 自动提醒是否需要先看前作 |

---

## 工作流程（Workflow）

1. **偏好分析** - 了解用户喜欢的电影类型、观影场景
2. **匹配推荐** - 从喜欢的类型中筛选高分电影，确保多样性
3. **详细介绍** - 提供无剧透的剧情简介，推荐理由
4. **场景匹配** - 根据观影人数和场合推荐

---

- [ ] 无重要剧透信息
- [ ] 适合人群描述准确
- [ ] 场景匹配建议合理
