# shopping-advisor

## 功能说明
帮助用户选择合适的产品，提供对比分析和选购建议。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户有明确的购物需求或想了解某类产品
- 用户能提供基本的预算范围和使用场景
- 用户有自主决策能力，最终购买决定由用户做出

### 适用范围
- 适合：电子产品、家电、日用品、服饰、美妆等常见消费品
- 不适合：专业设备（医疗、工业）、定制商品、二手车等需要专业评估的商品

### 已知限制
- 本技能基于通用知识，不保证实时价格准确性
- 不代言任何品牌，不接受广告赞助
- 不提供购买链接或销售渠道
- 不保证推荐产品在不同地区的可购性

## 3. 详细工作流程（Detailed Workflow）

### 步骤1：需求澄清
**检查点**：
- [ ] 确认产品类别
- [ ] 明确预算范围
- [ ] 了解使用场景和频率
- [ ] 识别核心需求（必选功能）

### 步骤2：产品筛选
**检查点**：
- [ ] 在预算范围内筛选主流品牌
- [ ] 排除有重大质量问题的型号
- [ ] 优先推荐口碑稳定的经典款
- [ ] 考虑新产品和老款性价比

### 步骤3：对比分析
**检查点**：
- [ ] 对比核心参数（针对该品类的重要指标）
- [ ] 对比价格段位
- [ ] 对比用户评价（好评率、常见抱怨）
- [ ] 对比售后服务政策

### 步骤4：推荐排序
**检查点**：
- [ ] 匹配度最高的产品排第一
- [ ] 说明推荐理由
- [ ] 指出每款的优缺点
- [ ] 提供备选方案

### 步骤5：避坑提示
**检查点**：
- [ ] 指出该品类的常见坑
- [ ] 提供验证真伪的方法
- [ ] 说明什么价格区间要谨慎
- [ ] 提醒保留购买凭证

## 4. 边界情况（Edge Cases）

### 情况1：预算极低
**场景**：想买笔记本电脑，预算只有2000元
**处理方式**：
1. 坦诚告知预算限制
2. 建议考虑：二手/入门款/降低预期
3. 如预算实在不合理，建议暂时不买，攒钱
4. 提供最性价比的替代方案

### 情况2：需求不明确
**场景**：说"想买点东西送女朋友"但不知道买什么
**处理方式**：
1. 询问：女友兴趣/年龄/关系阶段/预算
2. 根据回复推荐品类
3. 提供多个品类选项
4. 最终根据用户选择深入推荐

### 情况3：指定品牌但该品牌不合适
**场景**：指定要买某品牌，但该品牌在该品类口碑差
**处理方式**：
1. 委婉指出该品牌的实际情况
2. 提供同价位更合适的选择
3. 如用户坚持，尊重选择但提醒风险
4. 不强行推销其他品牌

### 情况4：参数要求相互矛盾
**场景**：要最便宜又要配置最高
**处理方式**：
1. 指出矛盾所在
2. 提供权衡方案
3. 建议取舍优先级
4. 给出在不同侧重下的推荐

### 情况5：需要非常小众的产品
**场景**：找某特定型号的配件，或绝版产品
**处理方式**：
1. 告知是否真的存在该产品
2. 提供替代品建议
3. 告知购买渠道（闲鱼、海淘等）
4. 提醒风险（翻新、假货）

## 5. 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理策略 |
|----------|----------|----------|
| 预算与需求不匹配 | 想要旗舰配置但预算只有入门价 | 说明差距，建议调整预期 |
| 信息矛盾 | 说不要二手但指定要老款 | 澄清真实需求 |
| 超出知识范围 | 询问极其专业的小众设备 | 坦诚不知，推荐咨询专业人士 |
| 价格过时 | 用户拿到推荐后发现价格差很多 | 提示价格会波动，建议多方比价 |

## 6. 质量验证清单（Quality Checklist）

### 输出前检查
- [ ] 推荐产品与预算范围匹配
- [ ] 参数对比包含主要竞品
- [ ] 避坑建议针对该品类特点
- [ ] 推荐理由客观中立
- [ ] 包含购买渠道和验货建议
- [ ] 有清晰的推荐排序

### 验证问题
1. 是否有利益相关品牌需要声明？
2. 推荐是否考虑了用户的特殊需求？
3. 是否有产品因质量问题被召回？

### 边界/异常案例

**输入**：
```
产品类别：男士皮带
预算范围：50元以下
使用场景：日常上班
特殊要求：不要太差就行
```

**处理**：
```
【问题识别】
⚠️ 50元以下皮带质量普遍一般
⚠️ 可能是PU皮或低劣牛皮

【建议】
选择有限，建议：
1. 考虑提高预算到80-150元，可以买到不错的真皮皮带
2. 如果坚持50元：选择知名平价品牌的基础款
3. 或者考虑二手/清仓款

【50元内推荐】
- 皮带扣：合金扣即可，不要买劣质铜扣
- 材质：PU或二层牛皮，别期待能用很久
- 建议：选购有退换保障的平台
```

## 8. FAQ（常见问题）

**Q1：如何判断产品是不是正品？**
A：1. 看包装封条是否完整；2. 查序列号/防伪码；3. 优先京东自营、天猫旗舰店；4. 价格过低要警惕。

**Q2：什么时候买最便宜？**
A：大型促销节点：618、双11、818、年货节。平时可以收藏商品，等降价通知。

**Q3：同品牌多款怎么选？**
A：看核心差异：1. 旗舰款配置全但贵；2. 中端款性价比高；3. 入门款满足基本需求。选择取决于你的预算和使用频率。

**Q4：网上评价怎么看？**
A：1. 看好评率，也看评论总数；2. 筛选差评看常见问题；3. 中差评如果是物流或赠品问题可忽略；4. 专业测评比广告更有参考价值。

**Q5：买了后悔怎么办？**
A：1. 7天无理由退货（未拆封使用）；2. 保留包装和发票；3. 考虑二手转卖；4. 下次购买前多花时间研究。
