# home-reno

## 功能说明
提供家居装修和设计的建议，包括风格选择、空间规划、软装搭配等。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户有装修或改造家居的需求
- 用户对装修有基本的了解和预算
- 用户有一定的决策权限

### 适用范围
- 适合：住宅装修、旧房改造、软装搭配、空间优化
- 不适合：商业空间装修（需要专业设计公司）、专业建筑改造

### 已知限制
- 本技能提供通用装修建议，不针对特定户型
- 不提供专业的结构改造建议（涉及安全）
- 不提供具体的产品购买链接
- 不保证装修效果（涉及施工质量）

### 重要声明
涉及房屋结构改动（如拆墙、改水电）的工程，务必咨询专业装修公司和物业。

## 4. 边界情况（Edge Cases）

### 情况1：预算非常低
**场景**：想装修50平米的房子，预算只有3万
**处理方式**：
1. 明确告知3万预算非常紧张
2. 提供最低成本的装修方案
3. 建议优先级：功能性>美观性
4. 建议分期装修（先硬装，后慢慢添置家具）

### 情况2：老房子需要翻新
**场景**：90年代老房，想翻新但担心结构问题
**处理方式**：
1. 强调需要专业评估
2. 重点关注水电老化问题
3. 建议检查管道、电线状况
4. 提供老房翻新的重点注意事项

### 情况3：户型有缺陷
**场景**：买的二手房有个暗卫，没有窗户
**处理方式**：
1. 提供改善采光/通风的方案
2. 建议安装通风扇、换气扇
3. 使用浅色瓷砖和灯光改善
4. 考虑是否能够小范围改造（需物业批准）

### 情况4：对风格完全没概念
**场景**：第一次装修，完全不知道想要什么风格
**处理方式**：
1. 了解用户喜欢的颜色、材质
2. 提供不同风格的图片示例
3. 引导确定方向
4. 建议现代简约或北欧作为入门选择

### 情况5：有小孩/老人/宠物
**场景**：一家三口，有2岁孩子和老人同住
**处理方式**：
1. 强调安全第一（插座保护、圆角家具等）
2. 收纳要充足且方便取用
3. 地面考虑防滑
4. 预留足够的活动空间
5. 儿童房可以先简单，以后慢慢升级

## 5. 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理策略 |
|----------|----------|----------|
| 预算与需求不匹配 | 想要豪华装修但预算很低 | 说明差距，建议调整预期 |
| 涉及结构改动 | 想拆承重墙 | 明确告知不可行，建议找专业评估 |
| 风格混乱 | 想同时做北欧和中式 | 提供协调建议或建议专业设计 |
| 忽略实用性 | 只关注好看不顾实用 | 强调功能第一的原则 |

## 7. 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
房屋类型：新房
户型：两室一厅
面积：80平米
楼层：高层
装修范围：全屋
预算范围：15-20万
预算偏好：舒适型
风格偏好：简约风格
居住成员：年轻夫妻，无小孩
```

**输出**：
```
【房屋概况】
- 面积：80㎡
- 户型：两室一厅
- 需求：简约风格，空间显大，收纳充足

【装修风格建议】
简约风格要点：
- 色彩：以白、灰、木色为主，明亮干净
- 材质：天然木材、金属、玻璃
- 家具：简洁线条，多功能设计
- 收纳：隐形收纳，保持空间整洁
- 灯光：柔和暖光，可分区控制

【空间布局优化】
| 空间 | 现状问题 | 建议方案 |
|------|----------|----------|
| 客厅 | 面积中等 | 打通阳台延伸空间，选用细腿家具 |
| 餐厅 | 空间小 | 吧台代替餐桌，或折叠餐桌 |
| 主卧 | 收纳不足 | 定制到顶衣柜，内部优化分区 |
| 次卧 | 暂时空置 | 改为多功能房（书房+客房） |
| 厨房 | 封闭式 | 开放式或半开放式，空间更通透 |
| 卫生间 | 干湿未分离 | 玻璃隔断做干湿分离 |

【收纳方案】
1. **玄关**：鞋柜+换鞋凳+挂钩（进出门物品）
2. **客厅**：电视柜+隐形收纳柜+墙面搁板
3. **卧室**：到顶衣柜+床下收纳+床头柜
4. **厨房**：抽屉式橱柜+拉篮+墙面挂架
5. **卫生间**：镜柜+浴室柜+壁挂收纳

【预算分配建议】
| 项目 | 占比 | 金额（按18万总预算） |
|------|------|----------------------|
| 硬装 | 40% | 7.2万 |
| 家具 | 25% | 4.5万 |
| 家电 | 20% | 3.6万 |
| 软装 | 10% | 1.8万 |
| 备用 | 5% | 0.9万 |

【避坑建议】
1. **找靠谱施工队**：看工地再决定，不要只看报价
2. **水电改造**：提前规划，预留足够插座和水口
3. **防水**：卫生间、厨房、阳台防水一定要做好
4. **预算留余**：留10%备用金，应对超支
5. **环保材料**：甲醛问题重视，选择环保材料

【好物推荐】（简约风格）
家具：
- 宜家/北欧表情/梵几
- 价格参考：沙发1-2万，餐桌3-5千

灯具：
- 吊灯：简约设计
- 落地灯：补充照明
- 无主灯设计可以考虑

装饰：
- 绿植（龟背竹、琴叶榕）
- 装饰画（简约风格）
- 地毯（提升温馨感）

【装修时间规划】
| 阶段 | 时长 | 注意事项 |
|------|------|----------|
| 设计 | 2-4周 | 找设计师或自己设计 |
| 拆改 | 1周 | 非承重墙可动 |
| 水电 | 1-2周 | 很重要，多预留 |
| 泥瓦 | 2-4周 | 防水重点 |
| 木工 | 2-3周 | 定制柜体 |
| 油漆 | 1-2周 | 通风散味 |
| 安装 | 1-2周 | 灯具、洁具、家具 |
| 通风 | 2-4周 | 至少一个月 |

总计约3-4个月
```

### 边界/异常案例

**输入**：
```
房屋类型：旧房改造
户型：老公房
面积：45平米
居住成员：独居老人
风格偏好：无特别要求
特殊需求：需要无障碍设施
```

**处理**：
```
【问题识别】
⚠️ 老公房可能有管线老化问题
⚠️ 独居老人需要安全和便利
⚠️ 需要无障碍设计

【装修重点】
1. **安全第一**
   - 防滑地面（卫生间尤其重要）
   - 圆角家具（减少磕碰）
   - 充足照明（多处开关）
   - 紧急呼叫按钮

2. **便利性**
   - 减少弯腰和踮脚
   - 常用物品放在易取高度
   - 卫生间安装扶手
   - 门槛尽量消除

3. **水电检查**
   - 重点检查老化的电线
   - 更换老旧水管
   - 增加插座数量
   - 考虑智能家居（方便老人）

【建议方案】
- 风格：简约温馨，避免过多装饰
- 色彩：温暖色调，避免太冷或太暗
- 家具：稳固、舒适、高度合适
- 收纳：简化，减少家务负担

【特别注意】
- 老公房结构老旧，改造前务必请专业人士评估
- 物业和邻居沟通，避免纠纷
- 老人参与决策，尊重其生活习惯
```

### 失败案例及修正

**原始建议**（有问题）：
```
推荐：全屋做地暖
理由：地暖舒适度高
```

**问题识别**：
- 地暖成本较高，可能超出预算
- 层高可能不够
- 已经装修好的房子很难加地暖

**修正后**：
```
【供暖建议】
如果预算充足且层高允许：可以选择地暖
如果预算有限：
- 卫生间可安装浴霸（灯暖/风暖）
- 卧室使用暖气片
- 客厅/卧室可考虑空调+电暖器

如果是旧房改造：
- 地暖需要地面抬高，不适合层高矮的房
- 建议选择其他供暖方式
```

## 8. FAQ（常见问题）

**Q1：装修一般要多久？**
A：100平米左右的房子，全屋装修约3-4个月。其中设计1个月，施工2-3个月。

**Q2：找装修公司还是自己装？**
A：各有优缺点。装修公司省心但贵；自己装省钱但累且需要懂行。建议：不懂行的人找口碑好的装修公司半包。

**Q3：如何控制预算不超支？**
A：1. 前期做好详细预算表；2. 留10-15%备用金；3. 重大超支前先思考是否必要；4. 先做必须的（硬装），后做想要的（软装）。

**Q4：北欧风和日式风有什么区别？**
A：北欧风更偏冷色调、线条简洁、注重功能性；日式风更温暖（木色为主）、注重收纳、禅意氛围。

**Q5：旧房翻新要注意什么？**
A：1. 重点检查水电老化；2. 墙面地面是否需要重做；3. 门窗是否更换；4. 防水是否重新做。

**Q6：如何判断承重墙？**
A：肉眼判断不准确。一般外墙和超过15cm厚的墙可能是承重墙。务必咨询物业或请专业人士判断。
