# travel-planner

## 功能说明
帮助用户规划旅行，提供路线规划、住宿推荐、预算控制等。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户有明确的旅行意向（目的地、时间、人数、预算）
- 用户具备基本的旅行常识和自理能力
- 用户能够自行办理必要的旅行证件（如护照、签证）

### 适用范围
- 适合：国内游、出国游、自驾游、跟团游的各种场景
- 不适合：专业探险（需要专业资质）、极限旅行（超出常规安全范围）

### 已知限制
- 本技能提供通用建议，不针对特定供应商或服务
- 不提供实时的票务/酒店价格查询
- 不包含签证办理的具体指导（流程可能因国籍、地区而异）
- 不保证目的地的实时状况（如天气、突发事件）

## 3. 详细工作流程（Detailed Workflow）

### 步骤1：需求确认
**检查点**：
- [ ] 确认目的地和日期范围
- [ ] 确认出行人数和相互关系
- [ ] 确认总预算和预算偏好
- [ ] 了解特殊需求（如有）

### 步骤2：目的地研究
**检查点**：
- [ ] 确认目的地的最佳旅行季节
- [ ] 了解当地的气候和温度
- [ ] 调查当地的消费水平
- [ ] 确认是否需要签证或特殊证件

### 步骤3：路线规划
**检查点**：
- [ ] 串联可游览的景点，避免走回头路
- [ ] 合理分配每天的行程强度
- [ ] 预留足够的交通时间
- [ ] 考虑季节和天气因素

### 步骤4：住宿与交通
**检查点**：
- [ ] 住宿位置靠近主要景点或交通枢纽
- [ ] 交通方式符合预算和时间安排
- [ ] 预订可取消的住宿（应对变化）
- [ ] 重要景点门票提前预约

### 步骤5：预算分配
**检查点**：
- [ ] 总预算合理分配到各项目
- [ ] 预留10-15%的应急备用金
- [ ] 考虑淡旺季价格差异
- [ ] 列出购物预算上限

### 步骤6：清单整理
**检查点**：
- [ ] 按类别整理必备物品
- [ ] 提醒重要证件和文件
- [ ] 提供当地紧急联系方式
- [ ] 提示当地文化禁忌

## 4. 边界情况（Edge Cases）

### 情况1：预算极低
**场景**：想去西藏10天，预算只有3000元
**处理方式**：
1. 建议改为周边短途旅行
2. 如坚持，告知需搭车、沙发客等省钱方式的风险
3. 提供最经济的路线建议，但明确说明条件艰苦

### 情况2：时间很短（1-2天）
**场景**：周末想去旅行，只有2天
**处理方式**：
1. 建议周边城市，避免路上耽误太多时间
2. 提供紧凑行程，突出核心景点
3. 建议早出晚归，最大化游玩时间

### 情况3：多人意见不一致
**场景**：一家5口三代人，偏好完全不同
**处理方式**：
1. 识别共同兴趣点
2. 错开安排不同人的偏好活动
3. 留出自由活动时间
4. 投票决定部分安排

### 情况4：目的地不明确
**场景**：只知道想出去走走，不知道去哪
**处理方式**：
1. 询问偏好（自然风光/城市/海岛/山景）
2. 根据季节推荐合适目的地
3. 提供2-3个选项供选择

### 情况5：携带特殊人群
**场景**：带60岁老人和3岁小孩出行
**处理方式**：
1. 降低每日行程强度
2. 选择交通便利的地点
3. 预留充足休息时间
4. 准备应急药品和预案

---

## 5. 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理策略 |
|----------|----------|----------|
| 行程过于紧凑 | 每天超过3个景点且交通超过2小时 | 删减景点或延长天数 |
| 预算超支 | 估算费用超过总预算 | 调整住宿/交通档次 |
| 季节不合适 | 雨季去海岛/冬季去东北 | 建议更换目的地或时间 |
| 证件缺失 | 出国游但无护照 | 提示提前办理并预留时间 |
| 交通断档 | 景点之间交通不便 | 调整行程顺序或更换景点 |

## 6. 质量验证清单（Quality Checklist）

### 输出前检查
- [ ] 行程天数与出发/返程日期匹配
- [ ] 每日行程强度合理（不超过12小时）
- [ ] 交通方式衔接合理，有足够缓冲时间
- [ ] 住宿位置便于游玩
- [ ] 预算分配合理，总和等于总预算
- [ ] 包含必备物品清单
- [ ] 有安全提示和文化注意事项

### 验证问题
1. 是否有景点因季节/天气不适宜？
2. 是否有交通需要提前预订？
3. 是否有门票需要提前预约？
4. 是否有饮食禁忌需要考虑？

### 边界/异常案例

**输入**：
```
目的地：日本
出发地：北京
日期：元旦假期，7天6晚
人数：1人（独行）
总预算：15000元
预算偏好：经济型
旅行风格：深度游
特殊需求：第一次出国
```

**处理**：
```
【问题识别】
⚠️ 第一次出国日本，需要签证（约7-10个工作日）
⚠️ 元旦是日本旅游旺季，机票酒店价格较高
⚠️ 7天时间适合东京+京都/大阪联游

【建议调整】
1. 签证：立即办理，单次旅游签证约200元
2. 预算调整：元旦旺季，建议提高到18000元
3. 行程优化：7天建议只玩2个城市，避免走马观花

【推荐行程（东京+大阪）】
D1：抵达东京
D2-D4：东京深度游（浅草寺、秋叶原、新宿、涩谷、富士山一日游）
D5：新干线前往大阪
D6-D7：大阪+京都当日往返
D8：返程

【经济型建议】
- 住宿：商务酒店或胶囊旅馆，300-500元/晚
- 交通：购买JR Pass七日通票，约1400元
- 餐饮：便利店+回转寿司+超市，避免顿顿餐厅
```

### 失败案例及修正

**原始规划**（有问题）：
```
D1：抵达东京
D2：浅草寺→秋叶原→新宿→涩谷（一天走完所有热门景点）
D3：富士山一日游
D4：大阪城→道顿堀→心斋桥
D5：环球影城
D6：京都岚山→金阁寺→清水寺
D7：返程
```

**问题识别**：
- D2行程过于紧凑，东京一天不可能玩完所有景点
- 富士山一日游需要早起，前一天不应太累
- 京都不可能一天玩完所有景点

**修正后**：
```
D1：抵达东京，休整
D2：浅草寺+秋叶原+上野公园（轻松路线）
D3：涩谷+新宿+代代木公园（潮流路线）
D4：富士山一日游（需早起，住在富士山附近）
D5：新干线前往大阪，下午大阪城+道顿堀
D6：环球影城全天
D7：京都一日游（只选2-3个景点）
D8：返程
```

## 8. FAQ（常见问题）

**Q1：如何选择目的地？**
A：根据季节、预算、兴趣选择。夏季可选海边/高原，冬季可选冰雪/海岛。首次出行建议选择国内或交通便利的周边国家。

**Q2：签证怎么办理？**
A：确定目的地后，查该国使领馆官网的签证要求。一般需要：护照、照片、签证申请表、行程单、银行流水、在职证明等。建议提前1-2个月办理。

**Q3：如何避免旅行中的消费陷阱？**
A：1. 提前查好景点门票价格；2. 购物选择正规店铺；3. 餐饮先看菜单再点餐；4. 打车用正规出租车或打车软件；5. 纪念品理性购买。

**Q4：行李应该怎么打包？**
A：遵循"能带就不买"原则。必备：证件、药品、手机、充电宝、换洗衣物。根据目的地气候调整：海边带防晒，山里带雨具。

**Q5：遇到突发情况怎么办？**
A：1. 自然灾害：关注当地天气预报，及时调整行程；2. 生病受伤：就近就医，保留单据；3. 行李丢失：向航空公司申报；4. 护照丢失：联系当地使领馆。
