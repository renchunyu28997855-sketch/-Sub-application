# travel-planner v3.0

旅行行程规划技能。2026年集成AI智能路线规划、实时天气集成、文化差异提醒能力。

## 功能说明

帮助用户规划旅行，提供路线规划、住宿推荐、预算控制等。

---

## 假设声明（Assumptions）

- 用户有明确的旅行意向（目的地、时间、人数、预算）
- 用户具备基本的旅行常识和自理能力
- 用户能够自行办理必要的旅行证件

---

## 输入输出（I/O Format）

### 标准输入格式
```
【基本信息】
- 目的地：[具体地点/省份/国家]
- 日期：[出发日期 ~ 返回日期]
- 人数：X人（关系：情侣/家庭/朋友/独行）
- 总预算：XXXX元

【偏好设置】
- 旅行风格：[休闲/打卡/深度/冒险/亲子]
- 交通方式：[飞机/高铁/自驾]
- 住宿偏好：[酒店/民宿/青旅]
- 特殊需求：[有老人/儿童/宠物]
```

### 标准输出格式
```
【目的地概览】
- 最佳旅行季节、建议天数、当地特色

【行程安排】
| 天 | 日期 | 城市 | 行程 | 住宿 | 预算 |

【预算分配】
| 项目 | 占比 | 金额 |

【必备清单】
- 证件类、衣物类、电子设备、药品类

【注意事项】
- 天气提醒、文化禁忌、安全提示
```

## 工作流程（Workflow）

### 步骤1：需求确认
- [ ] 确认目的地和日期范围
- [ ] 确认出行人数和关系
- [ ] 确认总预算和偏好

### 步骤2：路线规划
- [ ] 串联景点避免走回头路
- [ ] 合理分配每日强度
- [ ] 预留交通缓冲时间

### 步骤3：资源匹配
- [ ] 住宿靠近交通枢纽
- [ ] 重要景点提前预约
- [ ] 预订可取消的住宿

### 步骤4：预算分配
- [ ] 各项目占比合理
- [ ] 预留10-15%应急金
- [ ] 列出购物预算上限

---

## 质量验证清单（Quality Checklist）

- [ ] 行程天数与日期匹配
- [ ] 每日强度合理（不超过12小时）
- [ ] 交通衔接有缓冲时间
- [ ] 预算分配总和等于总预算
- [ ] 包含必备物品清单
- [ ] 有安全提示和文化注意事项
