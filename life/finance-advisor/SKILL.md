## 快速开始

### 触发方式
- 直接说需求：`"帮我理理财"`、`"月入2万怎么存钱"`
- 描述财务状况：收入、支出、存款、债务
- 问具体问题：`"基金定投怎么选"`、`"要不要提前还房贷"`

### 核心原则
```
1. 先保障后投资（应急金 + 保险）
2. 高息债务优先还
3. 长期投资定期定额
4. 分散配置不跟风
```

---

## 输入格式

```
【基本信息】年龄、职业、婚姻状况

【财务状况】
- 月收入：XXXX元
- 月支出：XXXX元（或"不知道"）
- 存款/资产：XXXX元
- 负债：[房贷/消费贷/无]

【风险偏好】保守/稳健/积极

【理财目标】短期/中期/长期

【当前配置】存款方式、保险配置
```

---

## 输出格式

```
【财务诊断】收入、支出、储蓄率分析

【健康指标】
- 应急金：达标/不足
- 债务：正常/需注意/危险
- 保险：完善/基础/不足

【分配建议】标准普尔四账户

【分阶段行动】
| 阶段 | 目标 | 措施 |
|------|------|------|
| 第一步 | 建立应急金 | 3-6月支出 |
| 第二步 | 完善保险 | 医疗+意外 |
| 第三步 | 投资规划 | 指数基金定投 |

【AI增强】（可选）
- 智能分析：支出浪费点识别
- 风险评估：财务健康度评分
- 最优策略：债务还款/投资配置
```

## 2026 AI增强能力

### AI预算分析
```
输入：月度收支明细
输出：
- 支出结构饼图（饮食/居住/交通/娱乐占比）
- 浪费识别："外卖支出偏高，建议每周减少2次"
- 优化建议："调整为自带午餐，月省800元"
```

### AI投资建议
```
输入：风险偏好 + 投资经验 + 财务目标
输出：
- 个性化配置：股票/债券/现金比例
- 基金推荐：低费率宽基指数基金
- 定投计划：金额、频率、品种
- 再平衡提醒：年度调仓提示
```

### AI风险评估
```
输入：财务全貌
输出：
- 财务健康度评分（0-100）
- 风险预警：债务利率过高/应急金不足/保障缺失
- 改善建议：优先级排序的行动计划
```

## 边界情况

| 情况 | 处理 |
|------|------|
| 月光族 | 先记账+强制储蓄，从10%开始 |
| 高息债务 | 雪球法还债，停止产生新债务 |
| 无风险承受能力 | 保守为主，绝不推荐高风险投资 |
| 有疾病症状 | 先配置保险，再规划投资 |

---

## 质量检查

输出前确认：
- [ ] 应急金建议符合标准（3-6个月支出）
- [ ] 保险建议是基础保障（医疗+意外）
- [ ] 投资建议与风险偏好匹配
- [ ] 包含免责声明
- [ ] 行动计划具体可执行
