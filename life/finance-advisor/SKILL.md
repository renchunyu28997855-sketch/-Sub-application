# finance-advisor

## 功能说明
提供个人理财建议，包括预算规划、储蓄策略、投资配置等。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户有稳定的收入来源
- 用户具备基本的民事行为能力
- 用户寻求的是个人/家庭理财建议，不是商业投资建议

### 适用范围
- 适合：刚工作的年轻人、家庭理财、中产阶层理财
- 不适合：企业财务、高净值人群（需要专业机构）、复杂金融产品

### 已知限制
- 本技能不提供具体的投资荐股、基金代码等
- 不保证投资回报
- 不提供税务法律意见
- 不涉及复杂的跨境资产配置

### 重要声明
**投资有风险，入市需谨慎**。所有投资建议仅供参考，用户需自行承担投资结果。本技能不提供任何形式的投资收益保证。

## 3. 详细工作流程（Detailed Workflow）

### 步骤1：财务信息收集
**检查点**：
- [ ] 确认收入和支出情况
- [ ] 了解资产负债状况
- [ ] 识别已有的债务（利率、期限）
- [ ] 了解现有投资和保险配置

### 步骤2：财务健康诊断
**检查点**：
- [ ] 计算应急金是否充足（3-6个月支出）
- [ ] 评估债务负担（负债率<50%）
- [ ] 检查保险保障是否完善
- [ ] 分析储蓄率是否健康（>20%）

### 步骤3：目标设定
**检查点**：
- [ ] 短期目标是否现实
- [ ] 中期目标与财务能力匹配
- [ ] 长期目标（养老）是否有规划
- [ ] 目标按优先级排序

### 步骤4：规划建议
**检查点**：
- [ ] 应急金优先建立
- [ ] 高息债务优先还清
- [ ] 保险保障先完善
- [ ] 再进行投资规划

### 步骤5：资产配置建议
**检查点**：
- [ ] 配置与风险偏好匹配
- [ ] 投资分散化
- [ ] 长期投资选择低成本指数基金
- [ ] 保持足够的流动性

## 4. 边界情况（Edge Cases）

### 情况1：月光族，没有存款
**场景**：月收入8000，每月花光
**处理方式**：
1. 先帮助建立记账习惯
2. 使用"先存后花"方法（工资到账先存10%）
3. 削减非必要开支（奶茶、外卖、娱乐）
4. 设定小目标（第一个1万）
5. 建议使用自动化储蓄

### 情况2：有大额债务
**场景**：信用卡欠款10万，月息1.5%
**处理方式**：
1. 明确告知债务成本（年化18%）
2. 建议使用"雪球法"或"雪崩法"还债
3. 停止产生新债务
4. 如有房产可考虑低息贷款置换
5. 绝不建议以贷养贷

### 情况3：高收入但无规划
**场景**：月收入5万，但花得也厉害，没存款
**处理方式**：
1. 肯定收入优势
2. 分析支出结构，找出浪费
3. 强制储蓄（收入高更应存钱）
4. 建议咨询专业理财师
5. 关注长期目标（养老、税务规划）

### 情况4：即将退休
**场景**：55岁，积蓄50万，退休后养老金不多
**处理方式**：
1. 保守为主，不建议高风险投资
2. 确保有足够的流动性
3. 考虑商业养老保险补充
4. 规划医疗费用
5. 建议咨询专业的退休规划

### 情况5：刚毕业/低收入
**场景**：应届生，月收入5000
**处理方式**：
1. 先养活自己，不必强求储蓄
2. 优先建立应急金（哪怕每月500）
3. 投资自己（技能提升）是最重要的投资
4. 避免债务陷阱（信用卡分期）
5. 养成良好理财习惯

## 5. 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理策略 |
|----------|----------|----------|
| 期望不现实 | 想一年翻倍 | 解释复利原理，降低预期 |
| 忽视保险 | 有钱投资但无保险 | 先保障后投资，强调保险重要性 |
| 追涨杀跌 | 提到近期热门股票 | 强调长期投资，避免投机 |
| 过度保守 | 钱只存银行定期 | 说明通胀影响，适当配置权益资产 |

### 输出前检查

- [ ] 投资建议与风险偏好匹配
- [ ] 债务建议优先偿还高息债务
- [ ] 有清晰的行动计划

### 失败案例及修正

**原始建议**（有问题）：
```
立刻开始炒股票
追求高收益
分散投资20只股票
```

**问题识别**：
- 新手直接炒股风险极高
- 没有应急金就投资是危险的
- 20只股票太多无法管理

**修正后**：
```
【投资建议】
- 先确保有应急金
- 投资从低风险的基金开始
- 建议配置：
  - 货币基金（应急金）
  - 债券基金（稳健）
  - 指数基金定投（长期）

【具体建议】
- 每月定投指数基金（如沪深300）
- 设定定投计划后坚持执行
- 不管市场涨跌，坚持定投
- 至少持有3-5年以上

【明确不建议】
- 不建议新手直接炒股
- 不建议杠杆投资
- 不建议追热点
```

## 8. FAQ（常见问题）

**Q1：年轻人应该先投资还是先还债？**
A：如果有高息债务（如信用卡），先还债。如果是低息债务（如房贷），可以慢慢还，同时开始投资。

**Q2：应该如何分配收入？**
A：参考标准普尔家庭配置：10%应急金、20%保险、30%投资、40%储蓄。但比例可根据个人情况调整。

**Q3：什么是好的应急金标准？**
A：通常为3-6个月的生活支出。年轻人可以先以3个月为目标，逐步增加到6个月。

**Q4：定投基金怎么选？**
A：选择费率低、规模中等的指数基金，如沪深300、中证500等宽基指数。不推荐主动基金（费率 高、长期跑不赢指数）。

**Q5：要不要提前还房贷？**
A：取决于房贷利率和个人投资能力。如果房贷利率高于投资收益，可以考虑提前还；如果公积金贷款利率很低，不着急。

**Q6：保险怎么配置？**
A：基础配置：医疗险+意外险。有家庭责任的建议加定期寿险。教育金、年金险等不建议优先配置。
