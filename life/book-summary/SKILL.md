# book-summary

## 功能说明
帮助总结书籍的核心内容，提炼精华观点，提供行动指南。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户有明确的阅读目标或指定书籍
- 用户需要提取书籍的精华内容
- 用户愿意将知识转化为行动

### 适用范围
- 适合：非虚构类书籍（自我提升、商业、心理学、历史等）
- 不适合：小说、诗歌等虚构文学作品（更适合直接阅读体验）
- 不适合：专业学术著作（需要系统性学习）

### 已知限制
- 本技能基于书籍的已知信息进行总结
- 不提供书籍的完整内容
- 不保证对书籍内容的100%完全解读
- 对于不熟悉的书籍，会基于公开信息总结

## 3. 详细工作流程（Detailed Workflow）

### 步骤1：信息确认
**检查点**：
- [ ] 确认书籍信息（书名、作者）
- [ ] 确认总结深度需求
- [ ] 确认使用场景
- [ ] 确认是否需要行动清单

### 步骤2：内容提取
**检查点**：
- [ ] 识别书籍的核心主题
- [ ] 提炼3-5个核心观点
- [ ] 筛选最具价值的案例
- [ ] 提取方法论（如果有）

### 步骤3：结构化输出
**检查点**：
- [ ] 按逻辑顺序组织内容
- [ ] 使用清晰的标题和格式
- [ ] 保持简洁，避免冗余
- [ ] 确保可操作性

### 步骤4：行动转化
**检查点**：
- [ ] 将观点转化为具体行动
- [ ] 行动要可量化、可执行
- [ ] 提供优先级建议
- [ ] 设定合理的期望

### 步骤5：关联推荐
**检查点**：
- [ ] 推荐主题相关的其他书籍
- [ ] 说明推荐理由
- [ ] 提供阅读顺序建议

## 4. 边界情况（Edge Cases）

### 情况1：用户提供的书名不完全准确
**场景**："我想总结《高效能人士的7个习惯》，作者是..."
**处理方式**：
1. 确认正确的书名和作者
2. 如果信息有误，提供正确版本
3. 如果是多本书，选择用户可能想读的那本

### 情况2：用户不指定书籍，想找某主题的书
**场景**：想了解时间管理，有什么书推荐
**处理方式**：
1. 推荐2-3本该主题的经典书籍
2. 说明每本书的特点和适用人群
3. 建议阅读顺序
4. 可以帮助总结推荐书籍中的内容

### 情况3：用户想要某一章的详细笔记
**场景**：只想要《原子习惯》第二章的笔记
**处理方式**：
1. 确认是否为该章节
2. 提供该章节的详细总结
3. 说明该章节在全书中的位置和作用
4. 指出与其他章节的联系

### 情况4：书籍信息非常少
**场景**：只知道书名，不知道作者和内容
**处理方式**：
1. 尝试根据书名识别
2. 如无法确认，请求更多信息
3. 如果确认书籍存在但不了解内容，坦诚告知

### 情况5：用户要求总结整本书的每一个细节
**场景**：要求把《原则》500页内容全部总结
**处理方式**：
1. 解释总结的目的是提炼精华，非完整复述
2. 提供精华摘要和核心框架
3. 如需要详细某一方面，指出重点章节
4. 建议用户直接阅读原书获取完整内容

## 5. 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理策略 |
|----------|----------|----------|
| 书籍无法识别 | 书名不存在或信息太少 | 请求更多信息或提供相似推荐 |
| 内容理解偏差 | 用户指出总结有误 | 承认并修正，提供不同视角 |
| 书籍类型不适合总结 | 小说、诗集 | 建议直接阅读原著，分享阅读体验 |
| 版权问题 | 要求提供整本书内容 | 拒绝，只提供总结和引用 |

## 7. 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
书籍：《原子习惯》
作者：James Clear
需求类型：深度总结
关注重点：全部
使用场景：个人学习
```

**输出**：
```
【书籍信息】
- 书名：原子习惯（Atomic Habits）
- 作者：James Clear
- 出版年：2018
- 评分：4.8/5.0（豆瓣约9.0）
- 适合读者：想养成好习惯、戒除坏习惯的任何人

【核心观点】

**观点1：微小的改变可以带来巨大的成果**
每天进步1%，一年后将进步37倍。成功不是一次大的改变，而是每天微小习惯的累积。

**观点2：习惯的四个步骤：提示→渴望→反应→奖励**
- 提示：让好习惯显而易见，让坏习惯隐而不见
- 渴望：让习惯有吸引力
- 反应：让习惯简单易行
- 奖励：让习惯令人满足

**观点3：环境比意志力更重要**
改变环境，让好习惯更容易执行，让坏习惯更难执行。

**观点4：身份认同是习惯的核心**
不要只关注结果，要成为那样的人。相信并定义自己是哪种人。

**观点5：习惯叠加和两分钟法则**
把新习惯与已有习惯绑定；开始时只做2分钟，逐渐成为习惯。

【精华摘要】
1. 习惯的复利效应：小进步累积大成就
2. 习惯的四个环节缺一不可
3. 改变环境比改变自己更容易
4. 身份认同比目标设定更重要
5. "从不"到"总是"需要循序渐进
6. 追踪习惯但不要中断习惯

【方法论提炼】
| 步骤 | 环境设计 | 实践方法 |
|------|----------|----------|
| 1 | 让好习惯显而易见 | 习惯堆叠、愿景板 |
| 2 | 让好习惯有吸引力 | 绑定喜欢的事、社交强化 |
| 3 | 让好习惯简单易行 | 减少阻力、两分钟规则 |
| 4 | 让好习惯令人满足 | 即时奖励、追踪进度 |

【行动建议】
- [ ] 设定一个"两分钟"的习惯开始（如睡前阅读2分钟）
- [ ] 设计你的环境，让好习惯更容易（把书放在枕头边）
- [ ] 建立习惯追踪表，每天打勾
- [ ] 找到一个"问责伙伴"
- [ ] 设计你的身份认同（"我是一个阅读者"而非"我要读书"）

【金句摘录】
> "你每习惯一次，你就离你想成为的人更近一步。"
> "习惯是自我提高的复利。"
> "你不需要完美，只需要坚持。"

【相关推荐】
- 《深度工作》（Deep Work）- 专注力提升
- 《掌控习惯》（与本书同一作者） - 习惯养成系统
- 《思考快与慢》- 行为心理学基础
- 《有限理性的陷阱》- 决策心理学

【延展阅读】
- 想深入理解习惯的神经机制：推荐《贪婪的多巴胺》
- 想实践：可以建立自己的"习惯追踪系统"
```

### 失败案例及修正

**原始总结**（有问题）：
```
总结：这本书讲的是要努力坚持，才能成功
```

**问题识别**：
- 过于笼统，没有传达书籍的核心洞见
- 忽略了书中具体的方法论
- 没有提供可操作的内容

**修正后**：
```
总结：
《原子习惯》不是讲"努力坚持"，而是讲如何让好习惯更容易执行。

核心洞见：
1. 习惯的养成不在于目标设定，而在于系统建立
2. 要改变习惯，不是靠意志力，而是改变环境
3. 关键是让好习惯"显而易见、有吸引力、简单易行、令人满足"
4. 从"做两分钟"开始，而不是追求完美

可操作的方法：
1. 习惯堆叠：做完[现有习惯]后，我会[新习惯]
2. 两分钟规则：开始时只做2分钟
3. 环境设计：让好习惯的提示更明显
```

## 8. FAQ（常见问题）

**Q1：总结的书籍内容准确吗？**
A：我们基于书籍的公开信息和常见解读进行总结，力求准确。但不同人对同一本书可能有不同理解，建议参考后亲自阅读原书验证。

**Q2：可以总结任何书吗？**
A：主要适合非虚构类书籍（自我提升、商业、心理学、历史等）。虚构类（小说、诗歌）更适合直接阅读体验。

**Q3：总结和亲自阅读有什么区别？**
A：总结帮你快速了解核心观点，决定是否值得阅读。但无法替代亲自阅读的深度理解和个人感悟。

**Q4：可以只总结某一章吗？**
A：可以。请明确指出章节名称或内容范围。

**Q5：可以推荐书籍吗？**
A：可以。请告诉我们你想了解的主题、你的阅读水平和偏好，我们推荐合适的书籍。

**Q6：总结里提到的行动建议都能直接执行吗？**
A：行动建议是基于书中内容提炼的，可以直接尝试。每个人的情况不同，建议根据自己的实际情况调整。
