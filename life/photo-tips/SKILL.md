# photo-tips

## 功能说明
提供摄影技巧指导，包括构图、用光、参数设置等。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户有相机或手机摄影设备
- 用户想提升摄影技能
- 用户愿意学习和实践

### 适用范围
- 适合：摄影爱好者、想拍好照片的普通用户、想了解摄影基础的任何人
- 不适合：专业摄影师（需要进阶课程）、商业摄影（需要专业培训）

### 已知限制
- 本技能提供通用摄影技巧，不针对特定机型详细说明
- 手机摄影和相机摄影有所区别，会分别说明
- 不提供器材购买的具体建议
- 不涉及后期修图软件的详细教学

## 4. 边界情况（Edge Cases）

### 情况1：完全零基础
**场景**：第一次接触摄影，连光圈快门都不懂
**处理方式**：
1. 从最基础的概念讲起
2. 建议先学习手机摄影
3. 用简单易懂的语言解释
4. 提供大量示例帮助理解

### 情况2：只用手机拍照
**场景**：没有相机，只用手机想拍好照片
**处理方式**：
1. 讲解手机相机的特点
2. 介绍手机摄影的技巧
3. 建议好用的手机摄影App
4. 强调构图和光线的重要性

### 情况3：想拍特定题材
**场景**：想拍美食/宠物/宝宝，不知道怎么拍
**处理方式**：
1. 针对该题材的具体指导
2. 提供场景设置建议
3. 说明常用角度和构图
4. 提供光线处理技巧

### 情况4：遇到具体问题
**场景**：拍出来的照片总是糊的
**处理方式**：
1. 分析可能的原因（快门太慢/对焦不准/手抖）
2. 提供针对性的解决方法
3. 给出具体的参数建议
4. 建议练习方式

### 情况5：有一定基础想进阶
**场景**：会基本操作，想拍出专业感的照片
**处理方式**：
1. 讲解更高级的技巧
2. 介绍创意构图方法
3. 提供用光进阶技巧
4. 建议学习后期处理

## 7. 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
设备类型：手机+微单（索尼A6000）
摄影类型：人像摄影
当前水平：零基础
具体问题：拍人像不知道用什么设置
```

**输出**：
```
【人像摄影基础】

## 1. 相机/手机设置

### 微单（索尼A6000）
| 参数 | 建议值 | 说明 |
|------|--------|------|
| 模式 | 光圈优先（A档） | 控制虚化 |
| 光圈 | f/1.8-f/2.8 | 背景虚化，突出人物 |
| 快门 | >1/200s | 防止抖动凝固动作 |
| ISO | 100-800 | 光线好时用低ISO |
| 对焦 | 单点对焦 | 对眼睛对焦 |
| 白平衡 | 自动或日光 | 保持肤色自然 |

### 手机人像模式
- 使用人像模式（虚化背景）
- 确保光线充足
- 与拍摄对象保持距离（2-3米）
- 避免强烈背光

## 2. 构图技巧
- **三分法**：将人物眼睛放在交叉点上
- **留白**：画面不要太满，视线前方留空间
- **对角线**：身体倾斜，活泼生动
- **框架**：利用门窗等框架构图

## 3. 用光技巧
- **最佳时间**：日出后1小时、日落前2小时（黄金时间）
- **阴天**：柔光箱效果，皮肤柔和
- **窗边**：自然光是最好的人像光源
- **避免**：正午强烈阳光（形成难看阴影）

## 4. 拍摄角度
- **平视**：最自然
- **俯视**：显瘦，适合拍全身
- **仰视**：显高，但可能变形
- **特写**：重点突出表情

## 5. 拍摄姿势建议
- 让人放松，聊天抓拍
- 身体微微侧身
- 手不要藏起来
- 走动、转身抓拍
- 利用道具互动

## 6. 常见错误
- ❌ 只拍大头照（缺乏环境）
- ❌ 背景太乱（干扰主体）
- ❌ 背景虚化过度（看不清背景）
- ❌ 脸正对光源（光线刺眼）

## 7. 练习建议
1. 每天拍3张人像（不同角度）
2. 同一场景用不同参数拍
3. 模仿你喜欢的摄影师
4. 定期回顾，挑选进步的照片

【进阶技巧】
- 学会观察光线
- 尝试不同的构图
- 了解基本后期调色
- 学会引导模特
```

```

⚠️ 夜景拍糊：快门太慢，手抖
⚠️ 噪点太多：ISO太高

【手机夜景摄影技巧】

## 1. 解决模糊问题
方法：
- 靠墙或使用手机支架
- 用双手握持手机
- 寻找支撑点
- 使用倒计时拍摄避免触碰

## 2. 解决噪点问题
方法：
- 使用专业模式降低ISO（100-400）
- 适当延长快门时间
- 使用夜景模式（多帧合成）
- 保持手机稳定

## 3. 手机专业模式设置
| 参数 | 建议值 | 说明 |
|------|--------|------|
| ISO | 100-400 | 越低噪点越少 |
| 快门 | 1/15s-1s | 根据场景调整 |
| 对焦 | 无限远或对焦到建筑 |
| 白平衡 | 根据场景 |

## 4. 夜景拍摄技巧
- 蓝调时刻（天刚黑）最适合拍建筑
- 寻找人工光源（路灯、霓虹灯）
- 利用车灯形成光轨
- 拍摄人物时需要补光

## 5. 推荐App
- Lightroom Mobile（专业调色）
- VSCO（滤镜）
- Snapseed（简单调整）
```

## 8. FAQ（常见问题）

**Q1：买相机还是用手机？**
A：如果只是记录生活，手机足够；如果想学习摄影技术、创作更专业的作品，可以考虑相机。相机画质更好，可以换镜头，学习曲线更陡。

**Q2：什么是光圈、快门、ISO？**
A：简单理解：
- 光圈：控制背景虚化程度（数字越小虚化越强）
- 快门：控制画面是清晰还是模糊（速度快凝固画面，慢速拍出动感）
- ISO：控制画面亮度（越高越亮但噪点多）

**Q3：为什么拍出来的照片模糊？**
A：常见原因：1. 快门太慢（手抖）；2. 对焦没对准；3. 拍摄对象移动太快。解决方法：提高快门速度、对焦准确、使用防抖功能。

**Q4：黄金时间是什么？**
A：日出后1小时和日落前2小时，光线柔和、色温温暖，是拍照的最佳时间。正午阳光强烈，容易形成难看的阴影。

**Q5：如何让照片有故事感？**
A：1. 包含人物活动；2. 注意背景环境；3. 捕捉自然的表情和动作；4. 运用引导线；5. 留白让观者想象。

**Q6：手机能拍出虚化效果吗？**
A：可以。现在的手机有人像模式，通过算法模拟虚化。尽量选择简单的背景效果更好。
