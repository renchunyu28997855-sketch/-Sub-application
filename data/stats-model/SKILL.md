# stats-model v3.0

统计模型选择与应用技能。2026年集成AI辅助建模、自动化特征工程、智能模型解释能力。

## 核心能力

**问题类型 → 模型映射**：
| 问题类型 | 推荐模型 | 典型场景 |
|---------|---------|---------|
| 回归-可解释 | 线性回归、岭回归 | 业务归因、价格弹性 |
| 回归-高性能 | XGBoost、LightGBM、神经网络 | 精准预测、排名打分 |
| 分类-可解释 | 逻辑回归、决策树 | 风控规则、用户分层 |
| 分类-高性能 | 随机森林、SVM、神经网络 | 精准营销、流失预警 |
| 聚类-常规 | K-means | 用户分群、行为分组 |
| 聚类-复杂 | DBSCAN、层次聚类 | 异常检测、任意形状 |
| 时序-常规 | ARIMA、指数平滑 | 短期预测、趋势分析 |
| 时序-复杂 | Prophet、LSTM、Transformer | 节假日效应、多变量 |

## 快速开始

### 输入格式
```
【问题类型】分类/回归/聚类/时间序列
【数据概况】样本数、特征数
【目标变量】Y
【特征变量】X列表
【优先级】性能/可解释性/实时性
```

### 输出格式
```
【推荐模型】首选+备选+理由
【训练方案】数据划分、交叉验证
【评估结果】主要指标、性能对比
【业务解读】特征重要性、预测解释
```

## 2026 AI增强功能

### AI模型选择
- **自动模型推荐**：基于数据特征和问题类型，AI推荐最优模型候选
- **模型融合**：集成多个模型提升性能（Stacking/Blending）
- **超参数智能搜索**：贝叶斯优化替代网格搜索

### 自动化特征工程
- **特征自动生成**：多项式特征、交互特征、时间特征
- **特征选择**：基于重要性、L1正则化、前向/后向选择
- **特征编码**：自动识别类别变量并选择最优编码方式

### 结果智能解释
- **SHAP解释**：复杂模型的全局和局部解释
- **反事实分析**：最小变化导致预测翻转的样本
- **置信区间**：预测结果的不确定性量化

## 工作流程

### 阶段1：问题定义
1. 明确问题类型（回归/分类/聚类/时序）
2. 确定目标变量Y和特征X
3. 设定评估指标和优先级

### 阶段2：数据准备
4. 特征类型识别与编码
5. 数据划分（训练/验证/测试）
6. 交叉验证策略设定

### 阶段3：AI辅助建模
7. AI推荐候选模型
8. 自动化特征工程
9. 并行训练多模型

### 阶段4：评估与解释
10. 多指标评估对比
11. SHAP解释生成
12. 业务建议输出
