# ab-test v3.0

A/B测试实验设计技能。2026年集成AI辅助实验设计、智能化样本量计算、自动化结果解释能力。

## 核心能力

**问题 → 方案映射**：
| 实验类型 | 适用场景 | 推荐方法 |
|---------|---------|---------|
| 比例比较 | 转化率、点击率、留存率 | 卡方检验/Z检验 |
| 连续指标 | 收入、时长、评分 | t检验/Mann-Whitney |
| 多方案比较 | 多个设计方案选择 | ANOVA/多重比较校正 |
| 序贯检验 | 快速决策需求 | 置信区间法/贝叶斯 |
| 复杂效应 | 交互作用、细分分析 | 分层分析/CUPED |

---

## 快速开始

### 输入格式
```
【实验背景】产品/功能背景
【实验假设】要验证的假设
【核心指标】主要评估指标
【护栏指标】需监控的辅助指标
【实验组/对照组】方案描述
【预期效果】最小可检测效应（MDE）
```

### 输出格式
```
【实验设计】分组方案、样本量、周期
【数据分析】各组表现、显著性、置信区间
【实验结论】是否显著、建议、风险
```

## 2026 AI增强功能

### AI实验设计
- **智能假设生成**：基于业务目标AI生成可检验假设
- **多方案推荐**：AI推荐最优实验方案组合
- **流量预估**：AI预测实验周期和所需流量

### 智能化样本量计算
- **自适应计算**：基于历史数据自动估算方差和效应量
- **序贯监控**：AI辅助决定是否可提前停止或需延长期
- **分段样本量**：根据实验阶段动态调整样本需求

### 结果智能解释
- **自动归因**：AI分析影响结果的关键因素
- **细分洞察**：自动发现对实验效果敏感的细分用户群
- **置信区间可视化**：直观展示效果范围和不确定性

## 工作流程

### 阶段1：实验设计
1. 明确业务假设和成功标准
2. 选择核心指标（北极星）和护栏指标
3. 确定实验分组方案
4. AI辅助样本量计算
5. 预估实验周期

### 阶段2：实验执行
6. 流量分配（哈希分流确保随机）
7. 灰度发布和基础监控
8. 每日护栏指标检查

### 阶段3：数据分析
9. 样本量验证
10. 统计检验（t检验/卡方检验）
11. 效果量和置信区间计算
12. AI辅助结果解释

### 阶段4：结论输出
13. 判定显著性和建议
14. 风险提示和后续建议
