# report-generator v3.0

数据报告自动生成技能。2026年集成AI报告生成、自动可视化、动态报告能力。

## 功能说明

帮助自动生成各类数据报告，提升数据汇报效率。支持AI驱动的报告生成和自动可视化。

---

## 假设声明（Assumptions）

1. **数据已准备好**：报告所需数据已清洗完毕
2. **指标定义清晰**：每个指标的计算方式已确认
3. **模板已确定**：报告格式模板已建立（可使用标准模板）
4. **受众明确**：报告的读者角色已确定

---

## 输入输出（I/O Format）

### 标准输入格式

```
【报告类型】Dashboard/日报/周报/月报/专题报告/动态报告
【报告周期】时间范围
【数据来源】数据表/接口/文件
【指标清单】需要展示的指标
【受众】管理层/业务方/技术团队
```

### 标准输出格式

```
【报告文件】HTML/Excel/Word/PDF
【报告结构】摘要 → 核心指标 → 明细分析 → 趋势图表 → 结论建议
```

---

## 2026年新功能

### AI报告生成
- **自动内容生成**：AI分析数据并生成报告正文和洞察
- **智能摘要**：自动提炼关键发现和结论
- **自然语言查询**：用自然语言询问数据问题

### 自动可视化
- **图表自动选择**：AI根据数据特征推荐最佳图表类型
- **智能配色**：自动应用品牌配色方案
- **动态图表**：交互式图表支持数据下钻

### 动态报告
- **参数可配置**：用户选择指标、时间范围、对比维度
- **实时数据更新**：连接实时数据源
- **多格式导出**：支持HTML/Excel/PDF/PPT

## 工作流程（Workflow）

### 第一阶段：需求确认
1. 明确报告类型和周期
2. 确认核心指标清单
3. 确认受众和输出格式

### 第二阶段：数据准备
4. 数据提取（SQL/接口/文件）
5. 数据质量检查
6. 指标计算

### 第三阶段：报告生成
7. AI生成报告内容（如启用）
8. 图表生成和布局
9. 格式调整和美化

### 第四阶段：质量检查
10. 数据校验和审核
11. 发布或发送

---

## 质量验证清单（Quality Checklist）

- [ ] 报告周期和时间准确
- [ ] 核心指标数据已核实
- [ ] 对比数据（环比/同比）计算正确
- [ ] 图表与数据一致
- [ ] 结论有数据支撑
- [ ] 异常情况已标注说明
