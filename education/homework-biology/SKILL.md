# homework-biology v3.0

中小学生物题目解答与讲解技能。2026年集成AI智能答题、生命过程可视化、知识图谱关联。

## 功能说明

帮助用户解答中小学生物题目，提供详细的解答和知识点讲解。

---

## 假设声明（Assumptions）

- 用户具备基本的生物学术语理解能力
- 题目来源为正规教材或练习册
- 用户需要理解生物学概念和原理

### 适用范围
- 初中生物：细胞结构、植物生理、动物行为、微生物
- 高中生物：遗传学、分子生物学、生态学、实验技术

### 不适用范围
- 大学专业生物学（生物化学、细胞生物学等深层次内容）
- 实验操作类题目（需要显微镜观察等）
- 野外实践类题目

---

### 输入格式
```

题型：[可选，如"选择题"、"填空题"、"简答题"]

```

```
...

【举一反三】...
```

## 工作流程（Workflow）

### 步骤1：题目分析
- 识别题目类型（概念题/计算题/识图题/实验题）
- 判断知识点归属（细胞、遗传、生态等）
- 确定答题角度和要求

### 步骤2：知识点提取
- 回忆相关生物学原理
- 梳理知识网络
- 整理答题要点

### 步骤3：答案组织
- 按照题目要求组织答案
- 使用规范的生物学术语
- 注意答案的完整性和准确性

### 检查点
- [ ] 知识点回答准确
- [ ] 术语使用规范
- [ ] 答案逻辑清晰
- [ ] 要点完整
