# homework-biology

## 功能说明
帮助用户解答中小学生物题目，提供详细的解答和知识点讲解。

## 假设声明（Assumptions）

### 前提条件
- 用户具备基本的生物学术语理解能力
- 题目来源为正规教材或练习册
- 用户需要理解生物学概念和原理

### 适用范围
- 初中生物：细胞结构、植物生理、动物行为、微生物
- 高中生物：遗传学、分子生物学、生态学、实验技术

### 不适用范围
- 大学专业生物学（生物化学、细胞生物学、分子生物学等深层次内容）
- 实验操作类题目（需要显微镜观察等）
- 野外实践类题目

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
题目内容：细胞有丝分裂各时期的主要特点是什么？
年级/难度：高二
题型：简答题
```

**输出**：
```
【题目类型】细胞分裂
【难度等级】中等
【题目来源】高中生物必修一
【解题思路/答题要点】按分裂顺序依次描述各时期特点
【详细解答】
有丝分裂各时期主要特点：

1. 前期
- 染色质高度螺旋化形成染色体
- 细胞核解体，核膜消失
- 纺锤体形成（由纺锤丝构成）

2. 中期
- 染色体排列在赤道板上（赤道板不是实际结构）
- 染色体形态固定、数目清晰
- 是观察染色体的最佳时期

3. 后期
- 着丝点分裂，姐妹染色单体分开
- 染色体数目加倍
- 染色体在纺锤丝牵引下移向两极

4. 末期
- 染色体变成染色质
- 核膜、核仁重建
- 纺锤体消失
- 细胞质分裂，形成两个子细胞

【知识点】有丝分裂、染色体、染色质、纺锤体、赤道板
【易错提醒】1. 赤道板是虚拟平面，不实际存在；2. 后期着丝点分裂导致染色体数目加倍；3. 前期是染色质变染色体，末期相反
【举一反三】有丝分裂与减数分裂的主要区别是什么？（答案：有丝分裂染色体复制一次分裂一次，子细胞染色体数目不变；减数分裂复制一次分裂两次，子细胞染色体数目减半）
```

### 边界/异常案例

**输入**：
```
题目内容：孟德尔第一定律的内容是什么？
年级/难度：高二
```

**输出**：
```
【题目类型】遗传学基础
【难度等级】容易
【题目来源】高中生物必修二
【解题思路/答题要点】回忆孟德尔定律内容
【详细解答】
孟德尔第一定律（分离定律）：

在生物的体细胞中，控制同一性状的遗传因子成对存在，不相融合；在形成配子时，成对的遗传因子发生分离，分离后的遗传因子分别进入不同的配子中，配子中只含成对遗传因子中的一个。

通俗理解：
- 遗传因子（基因）在体细胞中成对存在（AA、Aa、aa）
- 配子中只含其中一个（显性或隐性）
- 后代表现型比例约为3:1（显性:隐性）

【知识点】孟德尔第一定律、分离定律、遗传因子、等位基因、显性、隐性
【易错提醒】1. 分离发生在配子形成时，不是在受精作用时；2. 分离的是成对的等位基因，不是所有基因
【举一反三】孟德尔第二定律（自由组合定律）的内容是什么？（答案：控制不同性状的遗传因子分离后，会自由组合）
```

### 失败案例及修正

**输入**：
```
题目内容：植物光合作用的场所是哪里？
```

**错误回答**（假设）：
```
细胞核
```

**问题**：光合作用场所是叶绿体，不是细胞核

**正确回答**：
```
【题目类型】细胞器功能
【难度等级】容易
【题目来源】初中生物/高中生物
【解题思路/答题要点】识记各细胞器功能
【详细解答】
植物光合作用的场所是叶绿体。

详细说明：
- 叶绿体存在于叶肉细胞中
- 叶绿体包括外膜、内膜、基粒（类囊体）、基质
- 光反应在基粒（类囊体膜）上进行
- 暗反应在基质中进行
- 总反应式：6CO2 + 6H2O → C6H12O6 + 6O2

【知识点】叶绿体、光合作用、叶肉细胞、基粒、基质
【易错提醒】光合作用场所是叶绿体，不是细胞核；呼吸作用场所是线粒体
【举一反三】呼吸作用的场所是什么？（答案：主要在线粒体）
```
