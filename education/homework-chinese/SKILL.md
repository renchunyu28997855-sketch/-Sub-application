# homework-chinese v3.0

语文题目解答技能。2026年集成AI智能阅读分析、文言文翻译辅助、写作思路生成。

## 功能说明

帮助用户解答语文题目，提供阅读理解技巧、文言文翻译和写作指导。

---

## 假设声明（Assumptions）

- 用户具备基本的语文阅读能力
- 题目来源为正规教材或练习册
- 用户需要理解基本的文学常识

### 适用范围
- **现代文阅读**：散文、小说、说明文、议论文
- **文言文阅读**：课内文言文、课外文言文
- **古诗词鉴赏**：意象分析、情感把握、艺术手法
- **作文写作**：命题作文、材料作文、话题作文

### 不适用范围
- 需要原文阅读的长篇名著
- 超出中小学范围的古文经典
- 创意写作的完全自由发挥

---

## 输入输出（I/O Format）

### 输入格式
```
题目内容：[具体题目文本或描述]
年级/难度：[可选，如"高一"、"高考难度"]
题型：[可选，如"阅读理解"、"文言文翻译"、"作文"]
篇目信息：[可选，如文章标题、作者]
```

### 输出格式
```
【题目类型】XXX
【难度等级】容易/中等/困难
【文本分析】...
【解题思路/翻译】...
【参考答案】...
【知识点积累】...
```

## 工作流程（Workflow）

### 步骤1：题目识别
- 识别题目类型（阅读/文言/作文）
- 提取文本关键信息
- 确定答题方向

### 步骤2：现代文阅读
- 梳理文章结构
- 抓住关键词句
- 分析写作手法

### 步骤3：文言文处理
- 逐字翻译
- 重点实词虚词讲解
- 句子大意把握

### 步骤4：作文指导
- 分析立意
- 给出写作思路
- 提供写作范例

### 检查点
- [ ] 理解准确
- [ ] 翻译流畅
- [ ] 分析到位
- [ ] 表达清晰
