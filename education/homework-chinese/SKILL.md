# homework-chinese

## 功能说明
帮助用户解答语文题目，提供阅读理解技巧、文言文翻译和写作指导。

## 假设声明（Assumptions）

### 前提条件
- 用户具备基本的语文阅读能力
- 题目来源为正规教材或练习册
- 用户需要理解基本的文学常识

### 适用范围
- **现代文阅读**：散文、小说、说明文、议论文
- **文言文阅读**：课内文言文、课外文言文
- **古诗词鉴赏**：意象分析、情感把握、艺术手法
- **作文写作**：命题作文、材料作文、话题作文

### 不适用范围
- 需要原文阅读的长篇名著
- 超出中小学范围的古文经典
- 创意写作的完全自由发挥

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
题目内容：翻译"落霞与孤鹜齐飞，秋水共长天一色"
年级/难度：高二
题型：文言文翻译
篇目信息：王勃《滕王阁序》
```

**输出**：
```
【题目类型】文言文翻译 - 古诗词
【难度等级】中等
【文本分析】
出自王勃《滕王阁序》，描写滕王阁秋日黄昏景色，是千古传诵的名句。
【翻译】
落日的晚霞与孤独的野鸭一起飞翔，秋天的江水和辽阔的天空连成一片，浑然一色。
【关键词解析】
- 落霞：落日的晚霞
- 孤鹜：孤独的野鸭（鹜：野鸭）
- 秋水：秋天的江水
- 长天：辽阔的天空
【艺术手法】
1. 对偶：落霞与孤鹜、秋水与长天
2. 动静结合：落霞"飞"、秋水"共"
3. 色彩映衬：晚霞、江水、天空
4. 意境开阔：画面浑然一体
【原文意境】
晚霞绚烂，孤鹜掠过水面；秋水澄澈，与天空连成一片。画面开阔高远，意境悠长，余韵无穷。
【知识点积累】
- 文言词汇：鹜（野鸭）、一色（浑然一体的颜色）
- 修辞手法：对偶、动静结合
- 名句积累：这是写景的名句，可用于写景作文
```

### 失败案例及修正

**输入**：
```
题目内容：赏析"春风又绿江南岸，明月何时照我还"的"绿"字
```

**错误回答**：
```
"绿"字用得很好
```

**问题**：赏析过于简单，没有具体分析

**正确回答**：
```
【题目类型】古诗词鉴赏 - 炼字
【难度等级】中等
【解题思路】从修辞、意境、情感多角度分析
【详细解答】
"绿"字的艺术效果：

1. 词性活用
   - 形容词用作动词
   - "使...变绿"
   - 形象生动

2. 画面感
   - 春风拂过，草木皆绿
   - 视觉冲击强烈
   - 春意盎然

3. 意境创造
   - 江南春景图
   - 生机勃勃
   - 色彩鲜明

4. 情感表达
   - 借景抒情
   - 思念家乡
   - "何时"表达归乡之盼

5. 炼字效果
   - 王安石《泊船瓜洲》
   - 数次修改才定"绿"
   - 体现了诗人推敲的精神

【知识点】炼字、词性活用、借景抒情、推敲
【易错提醒】赏析要从多角度进行，不能只说"用得好"
```

## FAQ（常见问题）

**Q1：文言文翻译有什么技巧？**
A：遵循"字字落实、句句通顺"原则，注意词类活用和句式调整。

**Q2：阅读理解总扣分怎么办？**
A：先抓文章主旨，再分析段落作用，关注标题和首尾段。

**Q3：作文不会立意怎么办？**
A：从材料中提取关键词，从不同角度思考，选择最熟悉的角度。
