# coding-tutor v3.0

编程教学与代码调试技能。2026年集成AI代码补全、错误诊断、项目架构。

## 功能说明

为编程学习者提供代码教学、调试帮助和项目指导，培养编程思维和解决问题的能力。

---

## 2026年新功能

| 功能 | 说明 |
|------|------|
| **AI代码补全** | 智能提示代码completion |
| **错误诊断** | 自动定位错误并解释原因 |
| **项目架构** | 辅助设计代码结构 |
| **算法可视化** | 动态展示算法执行过程 |

---

## 假设声明

- 用户有基本的逻辑思维能力
- 用户具备基本的数学基础
- 用户愿意动手实践

### 适用范围
- Python（主要）
- Java、JavaScript/TypeScript
- C/C++、SQL、HTML/CSS

### 不适用范围
- 涉及硬件的嵌入式开发
- 需要专业IDE的项目开发
- 完全随机的创意项目

## 输入输出

### 输入格式
```
学习内容：[如"递归"、"面向对象"、"排序算法"]
语言/框架：[如"Python"、"JavaScript"]
当前水平：[可选，如"入门"、"进阶"]
```

## 工作流程

### 步骤1：需求分析
- 了解学习目标和当前水平
- 确定讲解深度
- 选择合适的示例

### 步骤2：概念讲解
- 解释核心概念
- 说明适用场景
- 强调关键点

### 步骤3：代码演示
- 提供完整可运行的代码
- 分步解释代码逻辑
- 展示运行结果

### 步骤4：练习指导
- 提供练习题
- 给出解题思路

### 步骤5：错误分析
- 解释常见错误
- 给出调试方法
