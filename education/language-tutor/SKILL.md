# language-tutor

## 功能说明
为外语学习者提供全方位的学习辅导，包括发音、口语、写作、听力等技能训练。

## 假设声明（Assumptions）

### 前提条件
- 用户有明确的外语学习目标
- 用户愿意开口练习
- 用户有一定的基础（至少了解基本语法）

### 适用范围
- 英语（主要）
- 日语
- 法语
- 德语
- 西班牙语
- 其他语种可定制

### 不适用范围
- 完全零基础（需要先入门）
- 专业翻译
- 需要沉浸式环境的深层次学习

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
学习语言：英语
学习目标：口语提升
当前水平：中级（已通过CET-4）
具体需求：建立英语思维，减少翻译过程
```

**输出**：
```
【学习者水平】中级（已通过CET-4）
【学习目标】建立英语思维，提升口语流利度
【当前问题】
1. 口语表达时总是先想中文再翻译
2. 听力跟不上正常语速
3. 词汇量够但不会地道表达
4. 害怕开口，反应速度慢

【提升建议】

1. 建立英语思维（最重要）
   - 每天用英语描述自己的日程（不用说出口，心里想也行）
   - 用英语思考简单问题，如"今天天气怎么样"
   - 看图说话：看到什么就用英语描述什么
   - 推荐：从描述周围环境开始

2. 听力提升
   - 每天精听1段1分钟音频（TED/VOA慢速）
   - 影子跟读法：滞后2秒跟读，模仿语调
   - 泛听：洗漱、运动时放英语当背景音
   - 目标：3个月能听懂正常语速的简单对话

3. 地道表达积累
   - 掌握100个常用idioms（get on my nerves, hit the nail on the head）
   - 学习collocations（词语搭配）：do homework ≠ make homework
   - 避免中式英语（I'm very like ≠ I really like）
   - 推荐：《English Collocations in Use》

4. 口语练习
   - 用英语自言自语描述周围事物
   - AI对话软件练习（HelloTalk、Tandem）
   - 每周1次语言交换
   - 不追求完美，先说出来

【每日练习计划】
- 晨起：15分钟跟读（模仿语调）
- 午间：20分钟精听（1分钟材料反复听）
- 晚间：20分钟口语输出（描述今天发生的事）
- 睡前：10分钟复习今天学的表达

【学习资源推荐】
- App：TED（视频）、YouTube（频道）、HelloTalk（语伴）
- 书籍：《American Accent Training》、《English Collocations in Use》
- 播客：BBC Learning English、ESLPod
```

```
【提升建议】

- 用《TRY！日语能力考试N2》过一遍语法
- 每学一个语法，写一个例句
- 每天复习前一天学的语法
- 目标：建立N2语法框架

第3-4月：专项突破
- 语法：做专项练习题
- 阅读：每天1篇N2阅读
- 词汇：背N2词汇（推荐《N2词汇红宝书》）

第5-6月：模拟冲刺
- 每周一套完整真题
- 计时练习
- 错题分析
- 重点复习错题本

【每日学习内容】
- 语法：1个新语法 + 3个旧语法复习
- 词汇：20个新词 + 复习
- 阅读：1篇
- 听力：N3→N2过渡材料泛听

【推荐资源】
- 教材：《TRY！日语能力考试N2》、《新完全掌握日语能力考试》
- App：moji N2（词汇）、JLPT（日语考级）
- 网站：TBS podcasts for N2
- 真题：历年N2真题（网上可下载）
```

### 失败案例及修正

**输入**：
```
学习语言：英语
学习目标：托福100分
当前水平：六级450分
可用时间：2个月
```

**问题**：目标可能过高，两个月时间紧张

**修正**：
```
【分析】
- 目标：托福100分
- 现状：六级450分（约等于托福60-70分）
- 时间：2个月
- 差距：约30分

【现实评估】
两个月要提高30分非常挑战，但以下方案可以帮助最大化进步：

【策略调整】
1. 明确目标细分
   - 阅读：目标25+（最容易提分）
   - 听力：目标22+（需要大量练习）
   - 口语：目标22+（有模板可循）
   - 写作：目标25+（有套路）

2. 时间分配建议
   - 阅读：30%（最容易提分）
   - 听力：30%（需要时间积累）
   - 口语：20%（模板+练习）
   - 写作：20%（模板+练习）

3. 最低要求
   - 每天学习4小时
   - 单词：托福核心词汇（2000词）
   - 真题：近3年真题必须做完

【易错提醒】
- 不要贪多嚼不烂
- 核心词汇比大纲词汇更重要
- 真题是最好的练习材料
- 口语写作有模板，但要用活
```

## FAQ（常见问题）

**Q1：发音纠正怎么做？**
A：先听标准发音，模仿语调，注意连读和弱读，再录音对比。

**Q2：单词记不住怎么办？**
A：放在句子里记，使用间隔重复，推荐Anki等工具。

**Q3：听力听不懂怎么办？**
A：先从适合水平的材料开始，逐渐提高难度，精听+泛听结合。

**Q4：怎么克服"哑巴英语"？**
A：先从自言自语开始，找语伴练习，不要怕犯错。
