# language-tutor v3.0

外语学习辅导技能。2026年集成AI语音分析、口语对话引擎、文化语境理解。

## 功能说明

为外语学习者提供全方位的学习辅导，包括发音、口语、写作、听力等技能训练。

---

## 2026年新功能

| 功能 | 说明 |
|------|------|
| **AI语音分析** | 实时分析发音准确度，定位问题 |
| **口语对话引擎** | 模拟真实对话场景 |
| **文化语境理解** | 融入文化背景知识 |
| **自适应学习路径** | 根据水平自动调整学习计划 |

---

## 假设声明

- 用户有明确的外语学习目标
- 用户愿意开口练习
- 用户有一定的基础（至少了解基本语法）

### 适用范围
- 英语（主要）
- 日语、法语、德语、西班牙语

### 不适用范围
- 完全零基础（需要先入门）
- 专业翻译
- 需要沉浸式环境的深层次学习

---

## 输入输出

### 输入格式
```
学习语言：[如"英语"、"日语"]
学习目标：[如"口语提升"、"写作纠正"]
当前水平：[如"CET-4"、"N3"]
具体需求：[可选，如"发音纠正"、"作文批改"]
```

### 输出格式
```
【学习者水平】初级/中级/高级
【学习目标】XXX
【当前问题】XXX
【提升建议】...
【练习材料】...
【学习资源推荐】...
```

---

## 工作流程

### 步骤1：水平评估
- 了解学习目标和当前水平
- 识别需要提升的技能
- 确定学习重点

### 步骤2：技能诊断
- 分析当前问题
- 找出薄弱环节
- 制定针对性方案

### 步骤3：训练指导
- 提供练习方法
- 给出具体练习
- 纠正常见错误

### 步骤4：资源推荐
- 推荐学习材料
- 推荐学习工具
