# knowledge-explainer v3.0

复杂概念通俗讲解技能。2026年集成AI多模态解释、类比引擎、概念图谱。

## 功能说明

用通俗易懂的语言解释复杂的专业概念，配合生活实例和类比，帮助用户真正理解而非死记硬背。

---

## 2026年新功能

| 功能 | 说明 |
|------|------|
| **多模态解释** | 文字+图表+动画描述自适应切换 |
| **类比引擎** | 基于知识图谱自动匹配最佳类比 |
| **概念图谱** | 展示概念间的关联网络 |
| **理解度测试** | 讲解后自动生成理解度检测 |

---

## 假设声明

- 用户希望深入理解概念，而非仅知道定义
- 用户愿意通过类比和实例来学习
- 用户具备基本的常识和生活经验

### 适用范围
- 数学：微积分、线性代数、概率统计
- 物理：相对论、量子力学、电磁学
- 计算机：算法、数据结构、网络原理
- 经济：供需关系、通货膨胀、边际效应

### 不适用范围
- 需要大量前置知识的深奥理论
- 纯技术操作类
- 完全主观的价值判断

---

## 输入输出

### 输入格式
```
概念名称：[需要解释的概念]
当前理解程度：[可选，如"完全不了解"、"有一点了解"]
重点关注：[可选，如"实际应用"、"推导过程"]
```

### 输出格式
```
【概念名称】XXX
【专业定义】XXX
【通俗解释】...
【生活类比】...
【实例讲解】...
【相关概念】...
【记忆方法】...
```

---

## 工作流程

### 步骤1：概念识别
- 识别用户想了解的概念
- 评估用户的理解程度
- 确定讲解深度

### 步骤2：专业定义
- 给出准确的专业定义
- 解释术语的含义

### 步骤3：通俗转化
- 用生活类比解释
- 用简单语言重新表述

### 步骤4：实例讲解
- 给出具体实例
- 分析实例中的概念应用

### 步骤5：延伸关联
- 介绍相关概念
- 构建知识网络
