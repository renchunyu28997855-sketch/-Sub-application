# flashcard-generator v3.0

闪卡生成与记忆辅助技能。2026年集成AI智能出题、间隔重复引擎、记忆曲线预测。

## 功能说明

将知识点转化为问答形式的闪卡，支持间隔重复复习计划，帮助用户高效记忆。

---

## 2026年新功能

| 功能 | 说明 |
|------|------|
| **AI智能出题** | 自动识别关键信息，生成高质量问答对 |
| **间隔重复引擎** | 基于SM-2算法自动规划复习时间 |
| **记忆曲线预测** | 预测每个知识点的遗忘时间点 |
| **自适应难度** | 根据表现自动调整问题难度 |

---

## 假设声明

- 用户有明确的知识点需要记忆
- 用户愿意按照复习计划执行
- 用户理解闪卡学习的基本原理

### 适用范围
- 考前重点知识梳理
- 语言词汇记忆
- 概念定义背诵
- 公式定理记忆
- 历史时间线记忆

### 不适用范围
- 需要理解而非记忆的内容
- 技能操作类
- 主观开放性问题

---

## 输入输出

### 输入格式
```
知识点：[需要生成闪卡的内容]
科目/主题：[如"历史"、"数学"、"英语"]
数量要求：[可选，如"10张"、"全部"]
格式要求：[可选，如"问答对"、"只问不答"]
```

### 输出格式
```
【闪卡列表】
Q1: 问题 / A1: 答案
...

【复习计划】
- 即时：全部过一遍
- 1天后：回顾不熟悉的
- 3天后：再次检验
- 7天后：最终巩固

【记忆技巧】
...
```

## 工作流程

### 步骤1：内容分析
- 分析知识点结构
- 识别关键概念和事实
- 确定问答对数量

### 步骤2：问答对生成
- 设计问题（正面/反面/关联）
- 写出简洁答案
- 确保一问一答对应

### 步骤3：记忆策略
- 提供记忆技巧
- 设计记忆口诀
- 建议联想方法

### 步骤4：复习计划
- 制定间隔重复计划
- 确定检查点
