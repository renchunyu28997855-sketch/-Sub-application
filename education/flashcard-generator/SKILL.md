# flashcard-generator

## 功能说明
将知识点转化为问答形式的闪卡，支持间隔重复复习计划，帮助用户高效记忆。

## 假设声明（Assumptions）

### 前提条件
- 用户有明确的知识点需要记忆
- 用户愿意按照复习计划执行
- 用户理解闪卡学习的基本原理

### 适用范围
- 考前重点知识梳理
- 语言词汇记忆
- 概念定义背诵
- 公式定理记忆
- 历史时间线记忆

### 不适用范围
- 需要理解而非记忆的内容
- 技能操作类（不适合闪卡）
- 主观开放性问题

---

## 输入输出（I/O Format）

### 输入格式
```
知识点：[需要生成闪卡的内容]
科目/主题：[如"历史"、"数学"、"英语"]
数量要求：[可选，如"10张"、"全部"]
格式要求：[可选，如"问答对"、"只问不答"]
```

### 输入示例
```
知识点：第一次世界大战的知识点
科目/主题：历史
数量要求：8张
```

### 输出格式
```
【闪卡列表】
Q1: 问题
A1: 答案

Q2: 问题
A2: 答案
...

【复习计划】
- 即时：全部过一遍
- 1天后：回顾不熟悉的
- 3天后：再次检验
- 7天后：最终巩固

【记忆技巧】
...
```

## 详细工作流程（Detailed Workflow）

### 步骤1：内容分析
- 分析知识点结构
- 识别关键概念和事实
- 确定问答对数量

### 步骤2：问答对生成
- 设计问题（正面/反面/关联）
- 写出简洁答案
- 确保一问一答对应

### 步骤3：记忆策略
- 提供记忆技巧
- 设计记忆口诀
- 建议联想方法

### 步骤4：复习计划
- 制定间隔重复计划
- 确定检查点
- 提醒复习节奏

### 检查点
- [ ] 问题清晰
- [ ] 答案准确
- [ ] 一一对应
- [ ] 记忆方法有效

---

## 边界情况（Edge Cases）

### 情况1：知识点过于庞大
**处理方式**：分批生成，每次10-20张

### 情况2：需要图片辅助
**处理方式**：提供文字描述，用户可自行配图

### 情况3：复杂概念
**处理方式**：拆分为多个简单问题

### 情况4：用户已有闪卡
**处理方式**：帮助优化或补充

---

## 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理方式 |
|---------|---------|---------|
| 知识点太散 | 无法形成体系 | 帮用户梳理结构 |
| 问题太模糊 | 答案不明确 | 明确问题 |
| 数量太多 | 可能记不住 | 分批生成 |
| 内容错误 | 事实性错误 | 核实后修正 |

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
知识点：第一次世界大战的知识点
科目/主题：历史
数量要求：8张
```

**输出**：
```
【闪卡列表】
Q1: 第一次世界大战的导火索是什么？
A1: 萨拉热窝事件（1914年6月28日，奥匈帝国皇储斐迪南大公遇刺）

Q2: 一战主要分为哪几个战区？
A2: 西线（英法vs德）、东线（俄vs德奥）、南线（意大利vs奥匈）

Q3: 1917年美国为什么要参战？
A3: 德国无限制潜艇战 + 齐默尔曼电报事件

Q4: 一战结束时签署的条约叫什么？
A4: 《凡尔赛条约》（1919年）

Q5: 一战对世界格局的主要影响是什么？
A5: 欧洲衰弱、美日崛起、俄国十月革命、殖民地独立运动兴起

Q6: 西线战役中最著名的是什么？
A6: 凡尔登战役（1916年，"绞肉机"）

Q7: 俄国十月革命发生在哪一年？
A7: 1917年（列宁领导）

Q8: 第一次世界大战的起止时间？
A8: 1914年7月28日 - 1918年11月11日

【复习计划】
- 即时：全部过一遍，标记不熟悉的
- 1天后：只复习标记的
- 3天后：再次检验不熟悉的
- 7天后：最终巩固所有卡片
- 14天后：抽查，防止遗忘

【记忆技巧】
1. 时间线记忆：6月28萨拉热窝 → 1914一战爆发 → 1917美国参战 → 1918结束 → 1919凡尔赛
2. 口诀："萨拉热窝点导火索，凡尔赛来签合约"
3. 联想记忆：西线是"英法德"，想象铁轨（west）
4. 数字联想：1914=一起（一起打仗），1918=一起吧（结束吧）
```

### 边界/异常案例

**输入**：
```
知识点：英语动词短语
科目/主题：英语
数量要求：10张
```

**输出**：
```
【闪卡列表】
Q1: "look forward to" 是什么意思？
A1: 期待、盼望（to是介词，后接名词/动名词）

Q2: "break up" 有哪些意思？
A2: 1. 分手/离婚 2. 拆散/分解 3. 散会

Q3: "get along with" 是什么意思？
A3: 与...相处（得好）

Q4: "put off" 是什么意思？
A4: 推迟、延迟

Q5: "come across" 是什么意思？
A5: 偶然遇到、偶然发现

Q6: "turn down" 是什么意思？
A6: 1. 拒绝 2. 调低（音量等）

Q7: "look up to" 是什么意思？
A7: 尊敬、仰慕

Q8: "cut down on" 是什么意思？
A8: 减少、削减

Q9: "get away with" 是什么意思？
A9: 逃脱惩罚/做坏事未被发现

Q10: "run into" 是什么意思？
A10: 1. 撞上 2. 偶遇（人）

【复习计划】
- 即时：区分相近短语
- 1天后：检查是否记住每个短语的基本意思
- 3天后：能否正确使用（造句）
- 7天后：能否在语境中识别

【记忆技巧】
1. 介词联想：put off（off=离开，放到后面=推迟）
2. 方向记忆：look up（向上看=尊敬）/ look down（向下看=轻视）
3. 场景记忆：想象每个短语的使用场景
4. 造句记忆：每个短语自己造一个句子

【常见考点】
- look forward to + doing
- get along with + sb
- put off + doing
```

### 失败案例及修正

**输入**：
```
知识点：光合作用
科目/主题：生物
```

**问题**：用户要求生成，但知识点太宽泛

**修正**：
```
我需要您告诉我具体需要哪些方面的闪卡，比如：

1. 光合作用的基本概念（场所、原料、产物）
2. 光合作用的两个阶段
3. 光合作用的反应方程式
4. 光合作用的影响因素
5. 光合作用与呼吸作用的关系

或者您可以更具体地描述，比如：
- "生成关于光反应阶段的10张闪卡"
- "生成光合作用方程式的闪卡"

这样我生成的内容会更精准。

如果您不确定需要哪些，我可以为您生成一套全面的（约20张），覆盖以上所有方面。
```

## FAQ（常见问题）

**Q1：闪卡多久复习一次？**
A：遵循间隔重复原则，刚学的当天复习，之后逐渐拉长间隔。

**Q2：问题太多记不住怎么办？**
A：分批学习，每次10-15张，确保掌握后再学下一批。

**Q3：可以导出到Anki吗？**
A：可以告知我需要Anki格式，我会生成相应格式。

**Q4：闪卡适合所有内容吗？**
A：适合事实性记忆，如定义、公式、时间；不适合需要理解的过程。
