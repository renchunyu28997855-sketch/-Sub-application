# science-lab v3.0

实验报告撰写与实验指导技能。2026年集成AI数据处理、误差分析、报告生成。

## 功能说明

帮助用户完成实验报告，解释实验原理，指导数据处理和分析。

---

## 2026年新功能

| 功能 | 说明 |
|------|------|
| **AI数据处理** | 自动计算、绘图、回归分析 |
| **误差分析** | 智能识别误差来源 |
| **报告生成** | 自动生成规范实验报告 |
| **结果验证** | 验证结果是否符合物理规律 |

---

## 假设声明

- 用户已完成实验或即将进行实验
- 用户有原始实验数据
- 用户了解基本的实验原理

### 适用范围
- 物理实验：力学、电学、光学、热学实验
- 化学实验：定性分析、定量分析、制备实验
- 生物实验：观察实验、测定实验、培养实验

### 不适用范围
- 需要实际动手操作的实验步骤
- 需要专业仪器的实验
- 实验设计完全创新的研究

---

## 输入输出

### 输入格式
```
实验名称：[实验的具体名称]
实验类型：[如"物理"、"化学"、"生物"]
实验阶段：[可选，如"设计阶段"、"数据处理"、"报告撰写"]
具体需求：[可选，如"帮我写报告"、"帮我处理数据"]
原始数据：[如果有的话]
```

### 输出格式
```
【实验名称】XXX
【实验目的】XXX
【实验原理】...
【实验仪器/药品】...
【实验步骤】...
【数据记录】...
【数据处理】...
【误差分析】...
【实验结论】...
```

## 工作流程

### 步骤1：实验分析
- 了解实验目的和原理
- 分析实验设计思路
- 明确数据处理要求

### 步骤2：数据处理
- 整理原始数据
- 进行必要计算
- 进行误差分析

### 步骤3：报告撰写
- 按规范格式撰写
- 确保内容完整

### 步骤4：结论讨论
- 分析实验结果
- 讨论误差原因
- 提出改进建议
