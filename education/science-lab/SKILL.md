# science-lab

## 功能说明
帮助用户完成实验报告，解释实验原理，指导数据处理和分析。

## 假设声明（Assumptions）

### 前提条件
- 用户已完成实验或即将进行实验
- 用户有原始实验数据
- 用户了解基本的实验原理

### 适用范围
- 物理实验：力学、电学、光学、热学实验
- 化学实验：定性分析、定量分析、制备实验
- 生物实验：观察实验、测定实验、培养实验
- 综合实验：探究性实验、设计性实验

### 不适用范围
- 需要实际动手操作的实验步骤
- 需要专业仪器的实验
- 实验设计完全创新的研究

## 详细工作流程（Detailed Workflow）

### 步骤1：实验分析
- 了解实验目的和原理
- 分析实验设计思路
- 明确数据处理要求

### 步骤2：数据处理
- 整理原始数据
- 进行必要计算
- 进行误差分析

### 步骤3：报告撰写
- 按规范格式撰写
- 确保内容完整
- 注意格式规范

### 步骤4：结论讨论
- 分析实验结果
- 讨论误差原因
- 提出改进建议

### 检查点
- [ ] 原理阐述清楚
- [ ] 数据记录完整
- [ ] 计算正确
- [ ] 结论有据

## 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理方式 |
|---------|---------|---------|
| 数据明显错误 | 与理论值差距大 | 指出异常值，建议处理 |
| 公式使用错误 | 公式不当 | 指出正确公式 |
| 计算错误 | 数学运算错误 | 重新计算 |
| 结论不当 | 超出数据支持 | 调整结论范围 |

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
实验名称：用单摆测定重力加速度
实验类型：物理
实验阶段：报告撰写
具体需求：帮我写完整的实验报告
原始数据：摆长L和周期T的多次测量数据
```

**输出**：
```
【实验名称】用单摆测定重力加速度
【实验目的】
1. 学会用单摆测定当地的重力加速度
2. 验证周期与摆长的关系
3. 理解单摆的振动规律
【实验原理】
单摆在小角度摆动时近似为简谐运动，其周期公式为：
T = 2π√(L/g)

变形得：g = 4π²L/T²

通过测量不同摆长L和对应周期T，计算g值。
【实验仪器】
- 米尺（量程2m，分度值1mm）
- 游标卡尺（量程15cm，分度值0.02mm）
- 秒表（分度值0.1s）
- 单摆（带铁架台）
- 小钢球
【实验步骤】
1. 测量摆线长度（用米尺量10次取平均值）
2. 用游标卡尺测量摆球直径（量3次取平均值）
3. 摆长 L = 摆线长 + 摆球半径
4. 将摆球拉开约5°，释放
5. 记录30次全振动的时间t（秒表测量）
6. 计算周期 T = t/30
7. 改变摆长（5种不同摆长），重复步骤4-6
【数据记录】
| 次数 | 摆长L(cm) | 30次振动时间t(s) | 周期T(s) | g(m/s²) |
|------|-----------|-----------------|----------|---------|
| 1    | 100.0     | 60.2            | 2.007    | 9.81    |
| 2    | 90.0      | 57.0            | 1.900    | 9.87    |
| 3    | 80.0      | 53.8            | 1.793    | 9.82    |
| 4    | 70.0      | 50.3            | 1.677    | 9.85    |
| 5    | 60.0      | 46.6            | 1.553    | 9.84    |

【数据处理】
方法一：逐差法
g₁ = 4π²L₁/T₁² = 4×9.87×1.00/4.028 = 9.81 m/s²
g₂ = 4π²L₂/T₂² = 4×9.87×0.90/3.610 = 9.87 m/s²
...
g平均值 = (9.81+9.87+9.82+9.85+9.84)/5 = 9.84 m/s²

方法二：图像法
以T²为纵坐标，L为横坐标作图
斜率 k = 4π²/g
由图得斜率k = 4.03
g = 4π²/k = 39.48/4.03 = 9.80 m/s²

最终结果：g = 9.82 ± 0.05 m/s²
【误差分析】
1. 系统误差
   - 空气阻力：使周期偏大，结果偏小
   - 摆角过大：不再近似简谐运动
   - 支架不稳：影响测量精度

2. 偶然误差
   - 计时起停反应慢：读数偏大
   - 摆长测量误差
   - 摆球没有在同一平面内摆动

3. 修正建议
   - 摆角控制在5°以内
   - 使用光电门计时更精确
   - 多次测量取平均

【实验结论】
1. 测得当地重力加速度 g = 9.82 ± 0.05 m/s²
2. 与标准值9.80 m/s²相比，相对误差约0.2%
3. 验证了单摆周期公式 T = 2π√(L/g)
4. 测量结果在允许误差范围内

【思考题解答】
Q1：为什么摆角要小于5°？
A1：当摆角小于5°时，单摆的运动近似为简谐运动，周期公式 T = 2π√(L/g)才成立。摆角过大，误差会增加。

Q2：为什么取30-50次振动而不是1次？
A2：计时次数越少，起停误差占比越大。取30-50次可以减小计时误差，提高测量精度。

Q3：如何判断摆球在水平面内摆动？
A3：在摆球平衡位置正下方画一条线，观察摆球每次经过时是否经过该点。
```

### 边界/异常案例

**输入**：
```
实验名称：酸碱中和滴定
实验类型：化学
实验阶段：数据分析
具体需求：帮我计算未知碱的浓度
原始数据：已知浓度HCl，消耗体积，NaOH体积
```

**输出**：
```
【实验名称】酸碱中和滴定
【数据处理】
已知：
- HCl浓度：0.1000 mol/L
- HCl消耗体积：25.00 mL
- NaOH待测浓度

实验过程：
NaOH + HCl → NaCl + H₂O

计算：
n(HCl) = C × V = 0.1000 mol/L × 0.025 L = 0.0025 mol

由化学方程式知：n(NaOH) = n(HCl) = 0.0025 mol

c(NaOH) = n/V = 0.0025 / 0.02380 = 0.1050 mol/L

【重复实验】
| 实验次数 | HCl体积(mL) | NaOH体积(mL) | NaOH浓度(mol/L) |
|----------|-------------|--------------|----------------|
| 1        | 25.00       | 23.80        | 0.1050         |
| 2        | 25.00       | 23.75        | 0.1053         |
| 3        | 25.00       | 23.82        | 0.1049         |

最终结果：c(NaOH) = (0.1050 + 0.1053 + 0.1049) / 3 = 0.1051 mol/L

【误差分析】
- 滴定管读数误差：±0.02 mL
- 指示剂变色判断误差
- 温度影响（忽略）

最终报告：NaOH浓度 = 0.1051 ± 0.0005 mol/L
```

### 失败案例及修正

**输入**：
```
实验名称：测小灯泡的伏安特性曲线
实验阶段：报告撰写
问题：得出的曲线不是直线，觉得实验失败了
```

**问题**：对实验结果认识错误

**修正输出**：
```
【概念纠正】
小灯泡的伏安特性曲线本来就不是直线！

这是正常现象，不是实验失败：

1. 小灯泡是纯电阻，但温度影响电阻
   - 电流越大，灯丝温度越高
   - 温度升高，电阻增大
   - 所以电压和电流不是正比关系

2. 正确的曲线形状
   - I-U曲线是弯曲的（向上弯曲）
   - 电阻随电压升高而增大
   - 这是符合物理规律的

【实验结论】
1. 小灯泡的I-U曲线是非线性的
2. 原因：灯丝温度随电流增大而升高
3. 计算不同电压下的电阻值：
   | U(V) | I(A) | R(Ω) |
   |------|------|-------|
   | 0.5  | 0.10 | 5.0   |
   | 1.0  | 0.18 | 5.6   |
   | 1.5  | 0.24 | 6.25  |
   | 2.0  | 0.28 | 7.14  |

可以看到，电阻随电压升高而增大。

【实验启示】
这个实验告诉我们：
1. 金属的电阻随温度升高而增大
2. 非线性元件需要测多组数据才能描述特性
3. 不能用线性关系去拟合这个曲线
```

## FAQ（常见问题）

**Q1：实验数据处理有哪些方法？**
A：常用方法有平均值法、逐差法、图像法等，根据实验特点选择。

**Q2：误差怎么计算？**
A：绝对误差和相对误差，需要分析系统误差和偶然误差。

**Q3：实验结论怎么写？**
A：基于数据，得出具体结论，与理论值或预期比较。

**Q4：思考题怎么回答？**
A：结合实验原理和结果，分析原因，说明道理。
