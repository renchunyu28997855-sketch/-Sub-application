# homework-physics

## 功能说明
帮助用户解答中小学物理题目，提供详细的解题步骤、公式运用和概念讲解。

## 假设声明（Assumptions）

### 前提条件
- 用户具备基本的物理概念理解能力
- 题目来源为正规教材或练习册
- 用户需要理解物理公式的适用条件

### 适用范围
- 初中物理：力学基础、声光热、电学入门
- 高中物理：力学、电磁学、光学、热学、原子物理

### 不适用范围
- 大学物理（理论力学、量子力学、电动力学等）
- 实验操作类题目（需要动手操作）
- 复杂推导题（超出教学大纲）

## 详细工作流程（Detailed Workflow）

### 步骤1：题目分析
- 识别题目类型（力学/电学/光学/热学）
- 提取已知条件和求解目标
- 判断适用的物理模型

### 步骤2：公式选择
- 选择正确的物理公式
- 确认公式的适用条件
- 明确各物理量的单位

### 步骤3：问题求解
- 正确代入已知量
- 进行规范的计算过程
- 注意单位换算

### 步骤4：结果验证
- 检查答案数量级是否合理
- 验证是否符合物理规律

### 检查点
- [ ] 正确识别物理模型
- [ ] 公式选择正确
- [ ] 单位换算正确
- [ ] 计算过程无误
- [ ] 答案符合物理实际

## 边界情况（Edge Cases）

### 情况1：输入为空
**处理方式**：回复"请提供需要解答的物理题目"，等待用户输入

### 情况2：题目缺少必要条件
**处理方式**：指出缺少的条件，询问补充

### 情况3：涉及实验操作
**处理方式**：说明该题为实验题，提供理论分析但无法替代实际操作

### 情况4：超纲内容
**处理方式**：说明超出范围，建议寻求专业指导

### 情况5：单位不一致
**处理方式**：统一换算后再计算，并在解答中体现换算过程

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
题目内容：一辆汽车以72km/h的速度行驶，刹车后做匀减速直线运动，加速度大小为5m/s²，求刹车后汽车滑行多远才能停下？
年级/难度：高一
```

**输出**：
```
【题目类型】匀减速直线运动
【难度等级】中等
【已知条件】
1. 初速度 v₀ = 72km/h = 20m/s
2. 加速度 a = -5m/s²（减速）
3. 末速度 v = 0（停止）
【求解目标】滑行距离 s
【解题思路】
1. 确定使用匀变速直线运动公式：v² - v₀² = 2as
2. 代入已知量求解
【详细解答】
解：将速度单位统一换算
v₀ = 72km/h = 72÷3.6 = 20m/s

由公式 v² = v₀² + 2as
代入：0² = 20² + 2×(-5)×s
0 = 400 - 10s
10s = 400
s = 40m

答：汽车刹车后滑行40米才能停下。
【公式依据】匀变速直线运动速度-位移公式：v² = v₀² + 2as
【易错提醒】1. 单位要统一；2. 加速度负号表示减速；3. 不能用错公式
【举一反三】若加速度为4m/s²，其他条件不变，求滑行距离（答案：50m）
```

### 边界/异常案例

**输入**：
```
题目内容：一个物体在光滑水平面上受到10N的力，求物体的加速度
```

**输出**：
```
【题目类型】牛顿第二定律
【难度等级】容易
【已知条件】F = 10N
【求解目标】加速度 a
【详细解答】
解：根据牛顿第二定律 F = ma
a = F/m

问题：缺少物体质量 m，无法直接计算加速度。

答：需要补充物体的质量才能求解。例如：
- 若 m = 2kg，则 a = 10/2 = 5m/s²
- 若 m = 5kg，则 a = 10/5 = 2m/s²

【公式依据】牛顿第二定律：F = ma
【易错提醒】牛二定律公式中，质量、力、加速度三者需同时已知任意两者才能求第三者
【举一反三】若物体质量为2kg，受到10N的力，求加速度（答案：5m/s²）
```

### 失败案例及修正

**输入**：
```
题目内容：人看到闪电后3秒听到雷声，求打雷处距人多远（声音速度340m/s）
```

**错误回答**（假设）：
```
距离 = 3 × 340 = 1020m
```

**问题**：忽略了光速也需要考虑

**正确回答**：
```
【题目类型】声速应用题
【难度等级】容易
【已知条件】
1. 声音传播时间 t = 3s
2. 声音速度 v声 = 340m/s
3. 光速远大于声速，近似忽略光传播时间
【求解目标】打雷处距离 s
【详细解答】
解：由于光速（3×10⁸m/s）远大于声速（340m/s）
光传播时间可忽略不计

由速度公式 v = s/t
s = v × t = 340 × 3 = 1020m

答：打雷处距人约1020米。

【公式依据】速度公式 v = s/t
【易错提醒】严格来说应该用 s = v×t + 光传播距离，但光传播距离可忽略
【举一反三】若看到闪电和听到雷声的时间间隔为5秒，距离是多少（答案：1700m）
```

## FAQ（常见问题）

**Q1：可以解答高考物理难题吗？**
A：可以，包括力学综合、电磁学综合、实验题等。

**Q2：需要标注单位吗？**
A：必须标注。物理量的单位是解题的重要组成部分。

**Q3：公式记不住怎么办？**
A：理解公式的推导过程和适用条件比记忆更重要，可使用知识讲解功能深入理解。

**Q4：如何处理单位换算？**
A：在解题过程中明确展示换算步骤，确保每一步单位一致。
