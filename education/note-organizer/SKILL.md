# note-organizer v3.0

笔记整理与知识梳理技能。2026年集成AI知识图谱、自动分类、多模态呈现。

## 功能说明

帮助用户整理和优化学习笔记，支持多种笔记方法，构建清晰的知识体系。

---

## 2026年新功能

| 功能 | 说明 |
|------|------|
| **AI知识图谱** | 自动构建知识点关联网络 |
| **自动分类** | 智能识别内容类型并分类 |
| **多模态呈现** | 支持思维导图、大纲、表格等多种形式 |
| **复习建议** | 基于遗忘曲线推荐复习节奏 |

---

## 假设声明

- 用户有原始笔记或学习内容
- 用户希望整理和优化笔记
- 用户愿意使用建议的笔记方法

### 适用范围
- 课堂笔记整理
- 读书笔记梳理
- 考前知识点总结
- 知识体系构建

### 不适用范围
- 完全零散的信息
- 需要原创的内容
- 实时记录的速记

---

## 输入输出

### 输入格式
```
原始内容：[笔记内容、教材要点或学习材料]
当前状态：[可选，如"零散笔记"、"教材划线内容"]
笔记类型：[可选，如"康奈尔"、"思维导图"、"大纲"]
整理目的：[可选，如"复习迎考"、"构建体系"]
```

### 输出格式
```
【原始内容概要】XXX
【推荐笔记方法】XXX
【整理后的笔记】...
【复习建议】...
【知识关联】...
```

---

## 工作流程

### 步骤1：内容分析
- 分析原始笔记的结构
- 识别核心主题和分支
- 评估现有组织方式

### 步骤2：方法选择
- 根据内容特点推荐方法
- 康奈尔：适合需要总结的场景
- 思维导图：适合构建体系的场景
- 大纲：适合层级分明的场景

### 步骤3：内容重组
- 重新组织内容结构
- 补充遗漏的关联
- 删除冗余信息

### 步骤4：呈现优化
- 使用可视化格式
- 突出重点和关键
