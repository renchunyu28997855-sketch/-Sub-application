# note-organizer

## 功能说明
帮助用户整理和优化学习笔记，支持多种笔记方法，构建清晰的知识体系。

## 假设声明（Assumptions）

### 前提条件
- 用户有原始笔记或学习内容
- 用户希望整理和优化笔记
- 用户愿意使用建议的笔记方法

### 适用范围
- 课堂笔记整理
- 读书笔记梳理
- 考前知识点总结
- 知识体系构建

### 不适用范围
- 完全零散的信息
- 需要原创的内容
- 实时记录的速记

## 详细工作流程（Detailed Workflow）

### 步骤1：内容分析
- 分析原始笔记的结构
- 识别核心主题和分支
- 评估现有组织方式

### 步骤2：方法选择
- 根据内容特点推荐方法
- 康奈尔：适合需要总结的场景
- 思维导图：适合构建体系的场景
- 大纲：适合层级分明的场景
- 表格：适合对比性知识

### 步骤3：内容重组
- 重新组织内容结构
- 补充遗漏的关联
- 删除冗余信息

### 步骤4：呈现优化
- 使用可视化格式
- 突出重点和关键
- 提供记忆提示

### 步骤5：复习建议
- 给出复习方法
- 建议复习节奏
- 提供自测问题

### 检查点
- [ ] 结构清晰
- [ ] 重点突出
- [ ] 关联明确
- [ ] 便于复习

---

## 边界情况（Edge Cases）

### 情况1：内容零散无结构
**处理方式**：帮用户梳理框架，建立结构

### 情况2：内容过多
**处理方式**：分模块整理，每次聚焦一个部分

### 情况3：不清楚笔记方法
**处理方式**：介绍各方法特点，帮助选择

### 情况4：需要多方法结合
**处理方式**：组合使用，如康奈尔+思维导图

| 错误类型 | 识别方式 | 处理方式 |

```
# 光合作用知识网络

## 核心反应
CO₂ + H₂O + 光能 → C₆H₁₂O₆ + O₂
（记住：反应物在左边，生成物在右边）

## 场所：叶绿体
- 外膜、内膜
- 基粒（类囊体）：光反应
- 基质：暗反应

## 两个阶段对比

| | 光反应 | 暗反应 |
|---|---|---|
| 场所 | 基粒 | 基质 |
| 条件 | 需要光 | 不需要光 |
| 能量 | 光能→化学能 | 化学能储存 |
| 产物 | O₂、ATP、NADPH | C₆H₁₂O₆ |
| 原料 | H₂O | CO₂、C₃ |

## 光反应详细过程
1. 光解水：2H₂O → 4H⁺ + 4e⁻ + O₂
2. ATP合成
3. NADPH合成

## 暗反应详细过程（卡尔文循环）
1. CO₂固定：CO₂ + C₅ → 2C₃
2. C₃还原：C₃ + NADPH + ATP → C₆H₁₂O₆
3. C₅再生

## 影响因素
| 因素 | 影响 |
|------|------|
| 光照强度 | 正相关（达到饱和点前）|
| CO₂浓度 | 正相关 |
| 温度 | 有关（最适25-30℃）|
| 叶绿素 | 正相关 |

## 色素吸收光谱
- 叶绿素a：红光、蓝紫光（主要吸收）
- 叶绿素b：蓝紫光
- 类胡萝卜素：蓝光（保护作用）

## 与呼吸作用的关系
- 呼吸：有机物 + O₂ → CO₂ + H₂O + 能量
- 光合：CO₂ + H₂O + 光能 → 有机物 + O₂
- 关系：互为相反过程

【复习建议】
1. 先记两个阶段的场所和产物
2. 理解能量转换过程
3. 背诵总反应方程式
4. 理解影响因素的作用机理
5. 对比记忆光合vs呼吸作用

【知识关联】
- 细胞呼吸（相反过程）
- 化能合成作用
- 碳循环
- 能量流动

【自测问题】
1. 光反应和暗反应的区别？
2. 光合作用的影响因素有哪些？
3. 写出光合作用总反应式
```

```
# 英语语法体系

## 一、时态体系（16种）

### 一般时态
| 时态 | 结构 | 时间标志 |
|------|------|----------|
| 一般现在时 | do/does | always, usually |
| 一般过去时 | did | yesterday, last |
| 一般将来时 | will do | tomorrow, next |

### 进行时态
| 时态 | 结构 | 时间标志 |
|------|------|----------|
| 现在进行 | am/is/are doing | now, at present |
| 过去进行 | was/were doing | at that time |

### 完成时态
| 时态 | 结构 | 时间标志 |
|------|------|----------|
| 现在完成 | have/has done | since, for, already |
| 过去完成 | had done | by + 过去时间 |

### 完成进行时态
| 时态 | 结构 |
|------|------|
| 现在完成进行 | have/has been doing |
| 过去完成进行 | had been doing |

## 二、从句体系

### 名词性从句
- 主语从句：What he said is true
- 宾语从句：I know who she is
- 表语从句：The problem is how we can solve it
- 同位语从句：The news that he won made us happy

### 定语从句
- 关系词：who, whom, which, that, whose
- 区别：who/whom指人，which指物，that两者皆可

### 状语从句
- 时间：when, while, after, before, since
- 原因：because, since, as
- 条件：if, unless, as long as
- 结果：so...that, such...that
- 目的：so that, in order that

## 三、非谓语动词

| 形式 | 用法 |
|------|------|
| to do | 目的、打算 |
| doing | 进行、习惯 |
| done | 被动、完成 |

【复习建议】
```

## FAQ（常见问题）

**Q1：哪种笔记方法最好？**
A：没有最好的方法，只有最适合的方法。康奈尔适合总结，思维导图适合发散，大纲适合层级。

**Q2：笔记整理多久做一次？**
A：建议课后当天整理一次，期中期末再系统整理。

**Q3：电子笔记和手写笔记哪个好？**
A：电子笔记便于搜索和修改，手写笔记便于记忆，两者结合最好。

**Q4：整理后的笔记还需要再整理吗？**
A：定期回顾，发现问题及时修正。
