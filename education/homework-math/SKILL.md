# homework-math

## 功能说明
帮助用户解答中小学数学题目，提供详细的解题步骤和思路讲解。

## 假设声明（Assumptions）

### 前提条件
- 用户具备基本的数学符号理解能力
- 题目来源为正规教材或练习册，非竞赛高难度题
- 用户需要手写或电子文档格式的解答过程

### 适用范围
- 小学数学：算术运算、简单几何、基础应用题
- 初中数学：代数、平面几何、三角函数入门、概率统计
- 高中数学：函数、导数、立体几何、排列组合、解析几何

### 不适用范围
- 高等数学（微积分、线性代数、概率论等大学课程）
- 数学竞赛题目（如奥数）
- 需要作图软件辅助的几何题（如复杂尺规作图）

## 详细工作流程（Detailed Workflow）

### 步骤1：题目识别与分类
- 读取用户输入的题目内容
- 识别题目类型：代数/几何/概率/应用题/数论
- 判断知识点归属
- 评估难度等级（容易/中等/困难）

### 步骤2：解题方法选择
- 根据题目类型选择最适合的解题方法
- 检查是否有多种解法
- 确认解题方法的正确性

### 步骤3：详细解答生成
- 按步骤书写解答过程
- 每一步标注依据（如"移项"、"合并同类项"）
- 确保推导过程清晰连贯

### 步骤4：知识点与易错点标注
- 提炼题目涉及的知识点
- 指出常见错误和易错点
- 提供记忆技巧

### 步骤5：举一反三与延伸
- 提供同类型练习题
- 可适当扩展相关知识点

### 检查点
- [ ] 题目类型识别正确
- [ ] 解题方法选择合理
- [ ] 解答过程无逻辑跳跃
- [ ] 最终答案正确
- [ ] 易错提醒准确

## 边界情况（Edge Cases）

### 情况1：输入为空
**处理方式**：回复"请提供需要解答的数学题目"，等待用户输入

### 情况2：题目包含图片
**处理方式**：说明"暂不支持图片识别，请将题目以文字形式描述"

### 情况3：题目描述不完整
**处理方式**：
- 缺少条件：指出缺少的条件，询问补充
- 题意不明：基于合理假设给出解答，并说明假设前提

### 情况4：超纲题目
**处理方式**：说明该题超出技能范围（如大学数学、竞赛题），建议寻求专业帮助

### 情况5：题目有明显错误
**处理方式**：指出题目中的错误（如"除数不能为零"），并基于修正后的题目解答

### 情况6：用户要求只给答案
**处理方式**：可简化输出，但提醒"建议查看完整解题过程以更好理解"

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
题目内容：一元二次方程 x² - 5x + 6 = 0 如何求解？
年级/难度：初二
```

**输出**：
```
【题目类型】一元二次方程
【难度等级】容易
【解题思路】使用因式分解法或求根公式法
【详细解答】
方法一：因式分解
x² - 5x + 6 = (x-2)(x-3) = 0
根据"零乘积定理"，若 AB=0，则 A=0 或 B=0
所以 x-2=0 或 x-3=0
所以 x = 2 或 x = 3

方法二：求根公式
Δ = b² - 4ac = (-5)² - 4×1×6 = 25 - 24 = 1
x = (-b ± √Δ) / 2a = (5 ± 1) / 2
x = 3 或 x = 2
【知识点】一元二次方程、求根公式、因式分解、零乘积定理
【易错提醒】注意符号：方程是 x² - 5x + 6，要小心移项时的符号变化
【举一反三】尝试解 x² - 3x - 10 = 0（答案：x=5 或 x=-2）
```

### 边界/异常案例

**输入**：
```
题目内容：已知平行四边形 ABCD 的对角线 AC 和 BD 相交于点 O，AC=10cm，BD=6cm，求面积
年级/难度：初三
```

**输出**：
```
【题目类型】平行四边形面积（条件不足）
【难度等级】中等
【解题思路】分析已知条件，发现无法直接求出面积（需要边长和夹角）
【详细解答】
已知：平行四边形对角线长度 AC=10cm，BD=6cm
分析：
- 平行四边形面积公式：S = 底 × 高 或 S = ab·sinθ
- 仅知道对角线长度，无法确定面积
- 需要补充条件：如边长、夹角、或特定类型（矩形、菱形）

答：仅根据已知条件，无法确定平行四边形的面积。需要补充以下条件之一：
1. 相邻两边的长度及夹角
2. 若为矩形，需要一边长度
3. 若为菱形，需要一个内角的度数

【知识点】平行四边形面积公式、对角线性质
【易错提醒】不要误以为对角线长度可以决定面积，平行四边形形状可变
【举一反三】若已知 ABCD 是矩形，对角线 AC=10cm，一边 AB=6cm，求面积（答案：48cm²）
```

### 失败案例及修正

**输入**：
```
题目内容：解方程 2x + 3 = 7
```

**错误回答**（假设）：
```
x = 7 - 3 / 2 = 4 / 2 = 2
```

**问题**：运算顺序错误

**正确回答**：
```
【题目类型】一元一次方程
【难度等级】容易
【解题思路】移项后合并同类项
【详细解答】
2x + 3 = 7
第一步：移项，将常数3移到等式右边
2x = 7 - 3
2x = 4
第二步：两边同时除以2
x = 4 ÷ 2
x = 2
【知识点】一元一次方程、移项法则
【易错提醒】移项要变号（+3 变成 -3）；除法要注意运算顺序
【举一反三】解方程 3x - 5 = 16（答案：x=7）
```

## FAQ（常见问题）

**Q1：支持图片题目吗？**
A：目前仅支持文字输入。请将图片中的题目以文字形式描述。

**Q2：可以解答高中数学吗？**
A：可以，包括函数、导数、三角函数、解析几何、立体几何、概率统计等。

**Q3：为什么有些题给多种解法？**
A：不同解法有助于理解数学思想的多样性，考试时可选择最擅长的方法。

**Q4：题目不会做怎么办？**
A：提供"只给思路"选项，可以先看解题思路，如果还不懂再要求详细步骤。

**Q5：答案不确定怎么办？**
A：请在输入时注明"不确定答案"，我会提供详细过程供您验证。
