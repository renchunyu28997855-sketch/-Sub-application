# homework-math v3.0

中小学数学题目解答与讲解技能。2026年集成AI智能解题、多步推理展示、个性化学习路径推荐。

## 功能说明

帮助用户解答中小学数学题目，提供详细的解题步骤和思路讲解。

---

## 假设声明（Assumptions）

- 用户具备基本的数学符号理解能力
- 题目来源为正规教材或练习册
- 用户需要手写或电子文档格式的解答过程

### 适用范围
- 初中数学：代数基础、几何初步、函数入门
- 高中数学：函数、三角函数、数列、立体几何、解析几何、概率统计

### 不适用范围
- 大学数学（高等数学、线性代数、概率论等）
- 数学竞赛超纲内容
- 需要作图工具的复杂几何证明

---

## 输入输出（I/O Format）

### 输入格式
```
题目内容：[具体题目文本]
年级/难度：[可选，如"初三"、"高考难度"]
特殊要求：[可选，如"用配方法"、"只给思路"]
```

### 输出格式
```
【题目类型】XXX
【难度等级】容易/中等/困难
【解题思路】
1. 第一步...
2. 第二步...
【详细解答】
...
【知识点】...
【易错提醒】...
【举一反三】...
```

## 工作流程（Workflow）

### 步骤1：题目识别与分类
- 读取用户输入的题目内容
- 识别题目类型：代数/几何/概率/应用题/数论
- 判断知识点归属
- 评估难度等级

### 步骤2：解题方法选择
- 根据题目类型选择最适合的解题方法
- 检查是否有多种解法

### 步骤3：详细解答生成
- 按步骤书写解答过程
- 每一步标注依据

### 步骤4：知识点与易错点标注
- 提炼题目涉及的知识点
- 指出常见错误和易错点

### 步骤5：举一反三与延伸
- 提供同类型练习题

### 检查点
- [ ] 正确识别题目类型
- [ ] 选择正确解题方法
- [ ] 计算过程无误
- [ ] 答案正确
