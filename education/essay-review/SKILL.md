# essay-review

## 功能说明
帮助用户批改和优化作文，从立意、结构、语言、素材等多维度提供写作指导。

## 假设声明（Assumptions）

### 前提条件
- 用户有完整的作文内容
- 用户愿意接受修改意见
- 用户希望提升写作能力

### 适用范围
- 议论文：论点、论据、论证过程
- 记叙文：叙事、描写、情感表达
- 说明文：条理清晰、说明方法
- 读后感/书评：内容概括、观点表达
- 应用文：书信、演讲稿、申请书

### 不适用范围
- 需要深度文学评论的长篇名著
- 专业学术论文
- 完全创意写作没有特定标准

## 详细工作流程（Detailed Workflow）

### 步骤1：通读全文
- 阅读并理解原文
- 把握文章主旨
- 初步判断优缺点

### 步骤2：分项分析
- 分析立意深度与新颖度
- 评估结构完整性与逻辑性
- 评价语言表达与文采
- 检查素材运用与典型性
- 评估开头结尾的吸引力

### 步骤3：具体问题
- 指出具体问题和不足
- 说明问题原因
- 提出修改方向

### 步骤4：修改建议
- 给出具体可操作的修改方法
- 提供修改示范
- 给出提升练习建议

### 检查点
- [ ] 评价客观全面
- [ ] 问题指出到位
- [ ] 建议具体可操作
- [ ] 示范有帮助

## 边界情况（Edge Cases）

### 情况1：只有题目没有内容
**处理方式**：分析题目，提供写作思路和范例

### 情况2：文章问题较多
**处理方式**：按重要性排序，先改核心问题

### 情况3：风格与要求不符
**处理方式**：指出差距，提供调整建议

### 情况4：优秀作文
**处理方式**：分析亮点，指出可以提升的地方

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
作文内容：[一篇关于"坚持"的议论文，约600字]
文体类型：议论文
年级/要求：高三
批改重点：结构和论据
```

**输出**：
```
【文体类型】议论文
【总体评价】
文章观点明确，语言流畅，但论据不够充分，论证深度有待加强。
结构基本完整，但部分段落逻辑稍显松散。

【分项评价】
- 立意：★★★★☆（观点明确，有一定深度）
- 结构：★★★☆☆（略显松散，过渡不够自然）
- 语言：★★★★☆（表达流畅，有一定文采）
- 素材：★★☆☆☆（事例陈旧，缺乏说服力）
- 创新：★★☆☆☆（缺乏独特视角）

【具体问题】
1. 【论据不足】第二段只用一个事例，论证力度不够
   问题：一个事例不能充分证明观点
   建议：至少需要2-3个不同角度的事例

2. 【深度不够】分析部分过于浅显，没有挖掘"为什么"
   问题：只说了"坚持很重要"，没说清"为什么重要"
   建议：多问几个"为什么"，深入分析

3. 【素材陈旧】使用的爱迪生事例已被用烂
   问题：考官看过无数次，缺乏新鲜感
   建议：使用更时新的事例

4. 【结构问题】分论点之间有交叉
   问题：第2段和第3段的观点有重复
   建议：重新梳理分论点，确保不交叉

5. 【语言问题】结尾过于口号化
   问题："让我们一起坚持吧"显得空洞
   建议：结尾要回扣论点，升华主题

【修改建议】
1. 每个分论点至少加2个论据（事实+道理论证）
2. 增加"为什么"的追问，深入分析原因
3. 替换为更时新的事例（身边的普通人、逆袭故事）
4. 重新梳理分论点，确保逻辑清晰
5. 结尾要有深度，回扣题目

【修改示范】
原文：创新很重要，爱迪生就是例子。
修改：创新是发展的第一动力。以屠呦呦为例，她在研究青蒿素时经历了190次失败，正是坚持创新，才最终拯救了数百万疟疾患者的生命。这启示我们：面对困难时，唯有创新才能突破。

【提升练习】
1. 每天积累一个新颖素材（关注时事新闻、人物传记）
2. 练习"五问分析法"：是什么-为什么-怎么做-有何例-怎么用
3. 背诵10个万能金句
4. 每周写一篇作文并修改
```

### 边界/异常案例

**输入**：
```
作文内容：[一篇写景的记叙文]
文体类型：记叙文
年级/要求：初二
批改重点：语言描写
```

**输出**：
```
【文体类型】记叙文（写景）
【总体评价】
文章情感真挚，善于观察，但景物描写手法单一，语言不够生动。
观察细致是优点，但表达方式需要丰富。

【分项评价】
- 立意：★★★★☆（借景抒情，情感真挚）
- 结构：★★★★☆（时间顺序清晰）
- 语言：★★★☆☆（描写手法单一）
- 情感：★★★★★（情感真挚动人）
- 观察：★★★★☆（观察细致入微）

【具体问题】
1. 【描写手法单一】通篇只有视觉描写
   问题：没有调动其他感官
   建议：加入听觉、嗅觉、触觉等描写

2. 【比喻较少】景物比喻不够丰富
   问题：比喻使用较少，形象性不足
   建议：增加比喻、拟人等修辞

3. 【缺乏层次】景物描写没有远近、动静之分
   问题：所有景物平铺直叙
   建议：按照一定顺序（由远及近、由高到低等）

【修改建议】
1. 从多个感官角度描写：视觉、听觉、嗅觉、触觉
2. 增加修辞手法：比喻、拟人、排比、夸张
3. 注意描写的层次和顺序
4. 动静结合：既有静态描写，也有动态描写

【修改示范】
原文：春天来了，花开了，树叶绿了。

修改：春天迈着轻盈的步伐走来了。花儿竞相开放，红的像火，粉的像霞，白的像雪，远远望去，就像一幅绚丽的画卷。蜜蜂在花间嗡嗡地闹着，蝴蝶翩翩起舞。微风拂过，带来阵阵花香，沁人心脾。嫩绿的小草从土里探出头来，好奇地打量着这个世界。
```

### 失败案例及修正

**输入**：
```
作文题目："责任"
文体类型：议论文
问题：完全不知道该怎么写
```

**帮助输出**：
```
【题目分析】"责任"
【审题】
"责任"是一个常见的议论文题目，可以从以下几个角度切入：

1. 什么是责任？
   - 责任是分内应做的事
   - 责任是一种担当
   - 责任与角色的关系

2. 为什么要有责任？
   - 个人层面：成长进步的前提
   - 社会层面：和谐运转的基础
   - 国家层面：民族复兴的需要

3. 怎样承担责任？
   - 从小事做起
   - 从现在做起
   - 勇于担当

【写作思路】
思路一：并列式
- 分论点1：责任是成就自我的基石
- 分论点2：责任是维系社会的纽带
- 分论点3：责任是推动民族前进的力量

思路二：递进式
- 第一层：什么是责任
- 第二层：为什么要承担责任
- 第三层：怎样承担责任

思路三：对比式
- 有责任感的 vs 没有责任感的（对比）
- 承担责任vs逃避责任（选择）

【参考素材】
正面：
- 钟南山：84岁仍奋战在抗疫一线
- 张桂梅：创办华坪女子高中，帮助贫困女孩读书
- 袁隆平：一生致力于杂交水稻研究

反面：
- 逃避责任的危害
- "甩锅"行为的代价

【开头结尾示范】
开头：责任是什么？它是诸葛亮"鞠躬尽瘁，死而后已"的忠诚，是范仲淹"先天下之忧而忧"的胸怀，是无数普通人默默奉献的坚守。

结尾：责任如山，重在担当。让我们扛起肩上的责任，在时代的洪流中砥砺前行。
```

## FAQ（常见问题）

**Q1：作文改多少遍才能提高？**
A：每一篇认真修改都比不改强，建议同一篇修改2-3遍。

**Q2：素材记不住怎么办？**
A：不需要死记硬背，理解素材的核心精神，考试时灵活运用。

**Q3：语言怎么才能有文采？**
A：多读优秀作文，背诵好词好句，模仿运用。

**Q4：考试作文有什么技巧？**
A：结构清晰、开头结尾精彩、字迹工整、紧扣题目。
