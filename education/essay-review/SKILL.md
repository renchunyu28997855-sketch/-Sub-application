# essay-review v3.0

作文批改与润色技能。2026年集成AI多维度评分、结构分析、语言润色。

## 功能说明

帮助用户批改和优化作文，从立意、结构、语言、素材等多维度提供写作指导。

---

## 2026年新功能

| 功能 | 说明 |
|------|------|
| **AI多维度评分** | 立意、结构、语言、素材多维评分 |
| **结构分析** | 自动分析文章逻辑结构 |
| **语言润色** | 智能推荐表达优化 |
| **素材推荐** | 基于主题推荐相关素材 |

---

## 假设声明

- 用户有完整的作文内容
- 用户愿意接受修改意见
- 用户希望提升写作能力

### 适用范围
- 议论文：论点、论据、论证过程
- 记叙文：叙事、描写、情感表达
- 说明文：条理清晰、说明方法
- 读后感/书评：内容概括、观点表达

### 不适用范围
- 需要深度文学评论的长篇名著
- 专业学术论文
- 完全创意写作没有特定标准

## 输入输出

### 输入格式
```
作文内容：[完整的作文文本]
文体类型：[如"议论文"、"记叙文"]
年级/要求：[可选，如"高三"、"高考水平"]
批改重点：[可选，如"结构"、"语言"、"立意"]
```

### 输出格式
```

【总体评价】...
【分项评价】
- 立意：★★★☆☆

【具体问题】...
【修改建议】...
【修改示范】...
【提升练习】...
```

## 工作流程

### 步骤1：通读全文
- 阅读并理解原文
- 把握文章主旨
- 初步判断优缺点

### 步骤2：分项分析
- 分析立意深度与新颖度
- 评估结构完整性与逻辑性
- 评价语言表达与文采

### 步骤3：具体问题
- 指出具体问题和不足
- 说明问题原因
- 提出修改方向

### 步骤4：修改建议
- 给出具体可操作的修改方法
- 提供修改示范
