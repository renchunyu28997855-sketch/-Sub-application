# study-plan v3.0

个性化学习计划制定技能。2026年集成AI智能规划、适应性调整、效果追踪。

## 功能说明

帮助用户制定科学合理的个性化学习计划，包括学科规划、时间管理和进度追踪。

---

## 假设声明（Assumptions）

- 用户有明确的学习目标
- 用户有可用的学习时间
- 用户愿意执行计划并定期反馈

### 适用范围
- 学期学习计划
- 考前复习计划
- 技能学习计划
- 自学成长计划

### 不适用范围
- 需要系统性培训的复杂技能
- 需要导师长期指导的专业学习
- 完全零基础的陌生领域（需要先入门）

---

## 输入输出（I/O Format）

### 输入格式
```
学习目标：[具体目标，如"Python编程"、"英语口语"]
总时长：[如"3个月"、"一学期"]
每周可用时间：[如"10小时"、"每天2小时"]
当前水平：[如"零基础"、"中级"]
学习风格：[可选，如"视觉型"、"听觉型"、"动手型"]
```

### 输出格式
```
【学习目标】XXX
【总时长】XX天/周
【学习风格】XXX
【总体方案】...
【周计划】...
【每日学习表】...
【进度检查点】...
【调整策略】...
```

## 工作流程（Workflow）

### 步骤1：目标分析
- 了解学习目标和动机
- 明确学习内容的范围
- 确定优先级

### 步骤2：水平评估
- 评估现有水平
- 识别基础和薄弱环节
- 确定学习起点

### 步骤3：计划制定
- 确定学习重点和难点
- 制定阶段性目标
- 安排每周学习任务

### 步骤4：时间规划
- 安排每日学习时段
- 平衡各科时间分配
- 预留休息和调整时间

### 步骤5：反馈机制
- 设置进度检查点
- 建立调整策略
- 提供激励建议

### 检查点
- [ ] 目标明确具体
- [ ] 计划切实可行
- [ ] 时间分配合理
- [ ] 有检查和调整机制
