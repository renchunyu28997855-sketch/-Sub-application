# homework-history

## 功能说明
帮助用户分析历史题目，提供详细的历史背景分析、事件关联和知识点讲解。

## 假设声明（Assumptions）

### 前提条件
- 用户具备基本的历史时间概念
- 题目来源为正规教材或练习册
- 用户需要理解历史事件的因果关系

### 适用范围
- 中国古代史、近代史、现代史
- 世界古代史、近代史、现代史
- 专题史（政治史、经济史、文化史）

### 不适用范围
- 史料分析题（需要原文阅读）
- 历史研究方法类题目
- 超出中小学大纲的历史内容

## 详细工作流程（Detailed Workflow）

### 步骤1：题目定位
- 识别历史事件或人物
- 确定历史时期
- 判断题目类型（事实题/分析题/评价题）

### 步骤2：背景梳理
- 明确历史背景
- 梳理相关事件关联
- 准备答题要点

### 步骤3：答案组织
- 按题目要求组织答案
- 使用规范的历史术语
- 注意时间线准确

### 检查点
- [ ] 时间地点准确
- [ ] 历史事实正确
- [ ] 分析逻辑清晰
- [ ] 评价客观合理

---

## 边界情况（Edge Cases）

### 情况1：时间记忆模糊
**处理方式**：提供准确时间，并说明记忆方法

### 情况2：事件关联复杂
**处理方式**：画时间线或思维导图帮助理解

### 情况3：评价类题目
**处理方式**：提供多角度分析方法

### 情况4：超纲内容
**处理方式**：说明超出范围，尽量提供已知信息

---

## 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理方式 |
|---------|---------|---------|
| 时间错误 | 与史实不符 | 纠正并说明正确时间 |
| 张冠李戴 | 事件人物混淆 | 指出并区分 |
| 分析片面 | 缺少角度 | 补充分析角度 |
| 超纲 | 大学内容 | 说明范围限制 |

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
题目内容：简述鸦片战争对中国社会的影响
年级/难度：高一
题型：简答题
```

**输出**：
```
【题目类型】历史影响分析
【难度等级】中等
【时代背景】19世纪中期，西方工业革命发展，中国闭关锁国
【解题思路/答题要点】从政治、经济、思想文化、社会性质多角度分析
【详细解答】
鸦片战争（1840-1842年）对中国社会的影响：

一、政治方面
- 中国主权遭到破坏，关税自主权被剥夺
- 五口通商，外国势力侵入
- 开始沦为半殖民地半封建社会

二、经济方面
- 自然经济逐渐解体
- 逐渐被卷入世界资本主义市场
- 白银外流加重人民负担

三、思想文化方面
- "睁眼看世界"思想兴起（林则徐、魏源）
- 开始向西方学习的新思潮

四、社会性质变化
- 社会性质：封建社会 → 半殖民地半封建社会
- 主要矛盾：农民阶级与地主阶级 → 帝国主义与中华民族、封建主义与人民大众

五、阶级结构变化
- 产生买办阶级
- 农民阶级受压迫加重

【知识点】鸦片战争、半殖民地半封建社会、南京条约、五口通商
【易错提醒】1. 鸦片战争是中国近代史的开端；2. 影响是多方面的，要全面分析；3. 社会性质改变是根本性变化
【举一反三】《南京条约》的主要内容是什么？（答案：割让香港岛、赔款2100万银元、五口通商、协定关税）
```

### 边界/异常案例

**输入**：
```
题目内容：为什么工业革命首先在英国发生？
年级/难度：高三
题型：论述题
```

**输出**：
```
【题目类型】历史原因分析
【难度等级】困难
【时代背景】18世纪中叶英国
【解题思路/答题要点】从政治、经济、技术、市场多因素分析
【详细解答】
工业革命首先在英国发生的原因：

一、政治前提
- 资产阶级代议制确立（1688年光荣革命）
- 政治稳定，政策支持
- 重视科技创新的传统

二、经济基础
- 资本原始积累完成（圈地运动、海外贸易、殖民掠夺）
- 手工业发达，技术积累
- 资本主义经济快速发展

三、技术条件
- 发明家辈出（瓦特、哈格里夫斯等）
- 工厂制度确立
- 自然科学进步

四、市场需求
- 国内外市场需求扩大
- 原材料需求增加
- 劳动力分工需要

五、地理优势
- 岛国环境，便于原料进口和产品出口
- 煤炭资源丰富（工业革命能源）

总结：工业革命是多种因素共同作用的结果，英国具备了所有必要条件。

【知识点】工业革命、圈地运动、瓦特、工厂制度、资本原始积累
【易错提醒】1. 工业革命是渐进过程，不是突变；2. 原因分析要全面，不能单一归因
【举一反三】工业革命对英国产生了什么影响？
```

### 失败案例及修正

**输入**：
```
题目内容：辛亥革命发生在哪一年？
```

**错误回答**（假设）：
```
1919年
```

**问题**：辛亥革命发生在1911年，不是1919年（五四运动）

**正确回答**：
```
【题目类型】历史时间记忆
【难度等级】容易
【解题思路/答题要点】准确记忆重要历史事件时间
【详细解答】
辛亥革命发生在1911年。

重要时间节点：
- 1911年10月10日：武昌起义（辛亥革命爆发）
- 1912年1月1日：中华民国临时政府成立
- 1912年2月12日：清朝灭亡

相关区分：
- 1919年：五四运动（新民主主义革命开端）
- 1921年：中国共产党成立

【知识点】辛亥革命、武昌起义、中华民国、孙中山
【易错提醒】注意区分相近历史事件的时间：辛亥革命1911年，五四运动1919年，中国共产党1921年
【举一反三】辛亥革命的历史意义是什么？
```

## FAQ（常见问题）

**Q1：历史时间记不住怎么办？**
A：使用时间线记忆法，把重要事件串联起来。

**Q2：历史分析题怎么答？**
A：按"原因-过程-结果-影响"的逻辑框架分析。

**Q3：如何区分相近的历史事件？**
A：从时间、地点、人物、性质、结果等方面对比记忆。
