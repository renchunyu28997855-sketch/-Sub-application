# homework-history v3.0

中小学历史题目分析技能。2026年集成AI智能分析、时间线可视化、历史事件关联。

## 功能说明

帮助用户分析历史题目，提供详细的历史背景分析、事件关联和知识点讲解。

---

## 假设声明（Assumptions）

- 用户具备基本的历史时间概念
- 题目来源为正规教材或练习册
- 用户需要理解历史事件的因果关系

### 适用范围
- 中国古代史、近代史、现代史
- 世界古代史、近代史、现代史
- 专题史（政治史、经济史、文化史）

### 不适用范围
- 史料分析题（需要原文阅读）
- 历史研究方法类题目
- 超出中小学大纲的历史内容

---

## 输入输出（I/O Format）

### 输入格式
```
题目内容：[具体题目文本]
年级/难度：[可选，如"初三"、"高考难度"]
题型：[可选，如"选择题"、"简答题"、"论述题"]
```

### 输出格式
```
【题目类型】XXX
【难度等级】容易/中等/困难
【时代背景】...
【解题思路/答题要点】
1. ...
2. ...
【详细解答】
...
【知识点】...
【易错提醒】...
【举一反三】...
```

## 工作流程（Workflow）

### 步骤1：题目定位
- 识别历史事件或人物
- 确定历史时期
- 判断题目类型（事实题/分析题/评价题）

### 步骤2：背景梳理
- 明确历史背景
- 梳理相关事件关联
- 准备答题要点

### 步骤3：答案组织
- 按题目要求组织答案
- 使用规范的历史术语
- 注意时间线准确

### 检查点
- [ ] 时间地点准确
- [ ] 历史事实正确
- [ ] 分析逻辑清晰
- [ ] 评价客观合理
