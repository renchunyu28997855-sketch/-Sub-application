# homework-politics

## 功能说明
帮助用户解答思想政治、道德与法治题目，提供时事分析、制度理解和价值判断的指导。

## 假设声明（Assumptions）

### 前提条件
- 用户具备基本的政治术语理解能力
- 题目来源为正规教材或练习册
- 用户需要理解政治经济学的分析框架

### 适用范围
- **经济生活**：商品货币、价值规律、社会主义市场经济
- **政治生活**：国家制度、政党制度、国际关系
- **哲学生活**：辩证唯物主义、历史唯物主义
- **道德与法治**：公民道德、法律知识、法治意识

### 不适用范围
- 大学政治经济学深层次理论
- 时事评论（仅提供客观分析）
- 超出中小学大纲的专业内容

## 边界情况（Edge Cases）

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
题目内容：为什么说社会主义市场经济能够实现共同富裕？
年级/难度：高三
题型：论述题
考察模块：经济生活
```

**输出**：
```
【题目类型】经济分析论述题
【难度等级】困难
【考察模块】经济生活 - 社会主义市场经济
【相关知识点】
- 社会主义的本质
- 按劳分配为主体
- 国家宏观调控
- 先富带后富
- 公有制为主体
【解题思路】
1. 理论依据：社会主义本质 + 市场经济特性
2. 制度保障：公有制为主体
3. 实现机制：按劳分配 + 宏观调控
4. 实践路径：先富带后富
【参考答案】
社会主义市场经济能够实现共同富裕，是因为：

1. 制度基础：公有制为主体
   - 国有经济控制国民经济命脉
   - 集体经济保障共同利益
   - 为实现共同富裕奠定制度基础

2. 分配保障：按劳分配为主体
   - 多劳多得，少劳少得
   - 效率与公平统一
   - 防止两极分化

3. 宏观调控：政府合理调节
   - 税收调节收入分配
   - 社会保障体系
   - 转移支付

4. 发展路径：先富带后富
   - 鼓励一部分人先富起来
   - 通过税收和扶贫帮助后富
   - 最终实现共同富裕

【时事关联】脱贫攻坚成果、乡村振兴战略
```

### 边界/异常案例

**输入**：
```
题目内容：简述规律的客观性和主观能动性的关系
年级/难度：高二
题型：简答题
考察模块：哲学
```

**输出**：
```
【题目类型】哲学概念辨析
【难度等级】中等
【考察模块】哲学 - 唯物论
【解题思路】从"是什么-关系-方法论"角度回答
【参考答案】
一、规律的客观性
- 规律是事物运动过程中固有的本质的必然的联系
- 规律是客观的，不以人的意志为转移
- 人不能创造、改变、消灭规律

二、主观能动性
- 人能够认识和利用规律
- 人能够能动地认识世界和改造世界
- 发挥主观能动性要以尊重客观规律为前提

三、二者的关系
1. 尊重规律是发挥主观能动性的前提
   - 只有认识规律、尊重规律，才能更好地发挥能动性
2. 发挥主观能动性是认识利用规律的必要条件
   - 人要积极主动地去认识和研究规律
3. 二者是辩证统一的关系
   - 既要尊重规律，又要发挥能动性

四、方法论
- 把尊重客观规律与发挥主观能动性结合起来
- 既要实事求是，又要勇于创新

【知识点】规律、客观性、主观能动性、实事求是
【易错提醒】规律是客观的，但人发挥主观能动性可以认识和利用规律
```

### 失败案例及修正

**输入**：
```
题目内容：我国为什么要坚持中国共产党的领导？
```

**错误回答**：
```
因为共产党是执政党
```

**问题**：回答过于简单，没有从多角度说明原因

**正确回答**：
```
【题目类型】政治制度分析
【难度等级】容易
【考察模块】政治生活 - 党的领导
【解题思路】从领导地位、性质、宗旨、执政方式多角度回答
【参考答案】
我国坚持中国共产党领导的原因：

1. 领导地位
   - 中国共产党是中国特色社会主义事业的领导核心
   - 是最高政治领导力量

2. 政党性质
   - 中国共产党是中国工人阶级的先锋队
   - 同时是中国人民和中华民族的先锋队

3. 根本宗旨
   - 坚持全心全意为人民服务
   - 代表最广大人民的根本利益

4. 执政理念
   - 立党为公、执政为民
   - 始终保持与人民群众的血肉联系

5. 实践验证
   - 历史和人民选择了中国共产党
   - 党的领导是中国革命、建设和改革成功的根本保证

【知识点】党的领导、执政党、先锋队、宗旨
```

## FAQ（常见问题）

**Q1：论述题怎么组织答案？**
A：按照"理论依据+材料分析+结论"的框架组织，要有明确的论点、论据和论证过程。

**Q2：如何区分相近的政治概念？**
A：从概念的内涵、外延、适用范围等方面对比理解。

**Q3：时事热点题目怎么答？**
A：结合教材理论分析具体事件，做到理论联系实际。
