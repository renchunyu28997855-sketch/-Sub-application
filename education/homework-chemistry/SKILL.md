# homework-chemistry

## 功能说明
帮助用户解答中小学化学题目，提供详细的解题步骤、反应原理和知识点讲解。

## 假设声明（Assumptions）

### 前提条件
- 用户具备基本的化学元素和化合物知识
- 题目来源为正规教材或练习册
- 用户需要理解化学方程式的含义

### 适用范围
- 初中化学：物质构成、常见化学反应、化学计算
- 高中化学：氧化还原反应、化学平衡、电化学、有机化学基础

### 不适用范围
- 大学化学（有机合成、分析化学、物理化学等）
- 实验操作类题目（需要动手操作）
- 复杂推断题（超出教学大纲）

## 详细工作流程（Detailed Workflow）

### 步骤1：题目分析
- 识别题目类型（反应类型、计算类型、推断类型）
- 提取已知反应物和生成物
- 判断反应类型和条件

### 步骤2：化学方程式处理
- 书写正确的化学式
- 配平化学反应方程式
- 标注反应条件（加热、催化剂等）

### 步骤3：计算与推导
- 根据化学方程式进行计算
- 运用化学定律（质量守恒、原子守恒）
- 进行摩尔计算

### 步骤4：结果验证
- 检查方程式是否配平
- 验证计算结果

### 检查点
- [ ] 化学式书写正确
- [ ] 方程式已配平
- [ ] 计算过程无误
- [ ] 答案符合化学计量

## 边界情况（Edge Cases）

### 情况1：化学式书写错误
**处理方式**：首先指出错误化学式，修正后继续解答

### 情况2：方程式未配平
**处理方式**：展示配平过程，不要直接假设已配平

### 情况3：缺少反应条件
**处理方式**：根据常识补充常见反应条件

### 情况4：有机化学复杂结构
**处理方式**：说明有机结构式需要图片或更详细描述

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
题目内容：配平化学反应方程式 Fe + O2 → Fe2O3
年级/难度：初三
```

**输出**：
```
【题目类型】化学方程式配平
【难度等级】容易
【解题思路】使用最小公倍数法配平
【详细解答】
步骤1：写出未配平的方程式
Fe + O2 → Fe2O3

步骤2：找出两边原子个数的最小公倍数
- Fe：左边1个，右边2个 → 最小公倍数2
- O：左边2个，右边3个 → 最小公倍数6

步骤3：配平氧原子（最小公倍数6）
O2前系数应为 6÷2 = 3
Fe2O3前系数应为 6÷3 = 2

步骤4：配平铁原子
Fe2O3前已确定系数为2，则Fe左边应为4

步骤5：验证
左边：4Fe + 3O2
右边：2Fe2O3
Fe：4 = 4 ✓
O：3×2 = 6，2×3 = 6 ✓

配平后方程式：
4Fe + 3O2 = 2Fe2O3（点燃条件下）
【化学方程式】4Fe + 3O2 = 2Fe2O3
【知识点】化学方程式配平、最小公倍数法、质量守恒定律
【易错提醒】配平后要检查所有原子都守恒，不能遗漏任何元素
【举一反三】配平 Al + O2 → Al2O3（答案：4Al + 3O2 = 2Al2O3）
```

### 边界/异常案例

**输入**：
```
题目内容：计算 98g 纯硫酸（H2SO4）中含有的氧原子个数
年级/难度：高一
```

**输出**：
```
【题目类型】摩尔计算
【难度等级】中等
【已知条件】
1. H2SO4质量 = 98g
2. H2SO4的摩尔质量 = 98g/mol
【求解目标】氧原子个数
【详细解答】
解：
步骤1：计算H2SO4的物质的量
n = m/M = 98g / 98g/mol = 1mol

步骤2：计算1个H2SO4分子中的氧原子数
1个H2SO4分子含有4个氧原子

步骤3：计算氧原子总个数
N = 1mol × 4 × NA = 1 × 4 × 6.02×10²³ = 2.408×10²⁴个

答：98g硫酸中含有 2.408×10²⁴ 个氧原子。
【化学方程式】—
【知识点】摩尔、摩尔质量、阿伏伽德罗常数、分子结构
【易错提醒】1个H2SO4分子有4个O原子，1mol H2SO4有4mol O原子
【举一反三】98g H2SO4中含有多少克氧元素？（答案：64g）
```

### 失败案例及修正

**输入**：
```
题目内容：写出氢气燃烧的化学方程式
```

**错误回答**（假设）：
```
H2 + O2 = H2O（未配平）
```

**问题**：方程式未配平

**正确回答**：
```
【题目类型】化学方程式书写
【难度等级】容易
【解题思路】先写出反应物和生成物，再配平
【详细解答】
步骤1：写出反应物和生成物
H2 + O2 → H2O

步骤2：配平
氢原子：左边2个，右边2个，已平衡
氧原子：左边2个，右边1个，需要×2

配平后：2H2 + O2 = 2H2O

步骤3：标注反应条件
2H2 + O2 = 2H2O（点燃）
【化学方程式】2H2 + O2 = 2H2O（点燃）
【知识点】化学反应方程式、配平、燃烧反应
【易错提醒】燃烧反应需要写"点燃"条件；配平后检查所有原子都守恒
【举一反三】写出甲烷（CH4）燃烧的化学方程式（答案：CH4 + 2O2 = CO2 + 2H2O）
```

## FAQ（常见问题）

**Q1：如何配平复杂的化学方程式？**
A：使用最小公倍数法、奇偶配平法或氧化还原配平法，根据题目选择合适方法。

**Q2：有机化学方程式需要配平吗？**
A：有机燃烧反应需要配平，一般的有机反应方程式也需要配平。

**Q3：摩尔计算总是出错怎么办？**
A：注意单位换算，理解n = m/M和N = n×NA两个核心公式。
