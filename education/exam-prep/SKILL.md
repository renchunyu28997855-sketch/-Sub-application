# exam-prep v3.0

考试备考策略与计划技能。2026年集成AI智能备考规划、个性化时间管理、应试技巧预测。

## 功能说明

为用户提供考试备考的全面指导，包括计划制定、知识点梳理、模拟训练和应试技巧。

---

## 假设声明（Assumptions）

- 用户有明确的考试目标
- 用户有可用的备考时间
- 用户愿意按照计划执行

### 适用范围
- **国内考试**：中考、高考、考研、公务员笔试
- **语言考试**：托福、雅思、GRE、GMAT
- **专业考试**：CPA、法考、医考、工程类考试

### 不适用范围
- 需要实地操作的考试
- 完全随机的选拔考试
- 时间极短（少于1周）的紧急备考

---

## 输入输出（I/O Format）

### 输入格式
```
考试类型：[具体考试名称]
目标分数：[可选，如"600分"、"7分"]
可用时间：[如"100天"、"3个月"]
当前水平：[可选，如"中等"、"已完成第一轮复习"]
薄弱科目：[可选，如"数学"、"听力"]
```

### 输出格式
```
【考试类型】XXX
【目标分数】XXX
【可用时间】XX天/周
【总体策略】...
【阶段规划】...
【每日时间表】...
【应试技巧】...
【资源推荐】...
```

## 工作流程（Workflow）

### 步骤1：信息收集
- 了解考试类型和目标分数
- 评估当前水平和可用时间
- 识别薄弱科目

### 步骤2：策略制定
- 确定总体备考策略
- 划分备考阶段
- 制定阶段目标

### 步骤3：计划细化
- 制定每周学习任务
- 安排每日学习时间
- 预留模拟考试时间

### 步骤4：技巧指导
- 传授各科应试技巧
- 提供时间分配建议
- 心理调适方法

### 检查点
- [ ] 计划切实可行
- [ ] 时间分配合理
- [ ] 重点突出
- [ ] 留有调整余地
