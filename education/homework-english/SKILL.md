# homework-english

## 功能说明
帮助用户解答各类英语题目，提供解题技巧和语言能力提升建议。

## 假设声明（Assumptions）

### 前提条件
- 用户具备基础的英语词汇和语法知识
- 题目来源为正规教材或练习册
- 用户需要理解基本的英语语法规则

### 适用范围
- **阅读理解**：细节题、主旨题、推断题、词义猜测题
- **完形填空**：词汇辨析、上下文推断、语法结构
- **写作**：应用文、图表作文、议论文
- **翻译**：中译英、英译中
- **语法**：时态语态、从句、非谓语

### 不适用范围
- 专业翻译（如法律、医学文献）
- 超出大纲的深奥语法
- 需要native speaker润色的高级写作

## 详细工作流程（Detailed Workflow）

### 步骤1：题目识别
- 识别题目类型和考查重点
- 分析题目要求
- 确定解题策略

### 步骤2：解题技巧应用
- 阅读理解：先看题干，再定位原文
- 完形填空：先通读全文，再逐题选择

### 步骤3：答案解析
- 分析正确答案的语言点
- 解释解题思路
- 指出易错点

### 步骤4：能力提升建议
- 提供举一反三的练习
- 给出相关知识点扩展

### 检查点
- [ ] 解题思路正确
- [ ] 答案准确
- [ ] 解析清楚
- [ ] 知识点覆盖

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
题目内容：阅读理解 - 关于环保的文章后，回答文章主旨是什么
年级/难度：高三
题型：阅读理解主旨题
```

**输出**：
```
【题目类型】阅读理解 - 主旨大意题
【难度等级】中等
【考查重点】文章主旨大意、作者意图
【解题技巧】
1. 看标题和首尾段
   - 标题是文章的核心
   - 首段往往点明主旨
   - 尾段往往是总结

2. 关注反复出现的关键词
   - 反复出现的词往往是主题词
   - 围绕一个话题展开

3. 关注转折词后的内容
   - but, however, yet 后往往是重点
   - 表达作者真正观点

4. 排除法
   - 排除过于具体的选项
   - 排除过于宽泛的选项
   - 选择最贴合文章的

【正确答案及解析】
文章主要讲述了气候变化的影响及应对措施

解析：文章开篇提出全球变暖问题（首段），中间论述其影响（冰川融化、海平面上升），结尾呼吁采取行动（总结段）。文章结构为"问题-影响-解决"，主旨明确。

【语言知识点】
- climate change（气候变化）
- global warming（全球变暖）
- sustainability（可持续性）
- carbon emission（碳排放）
- renewable energy（可再生能源）

【举一反三】
练习其他主旨题，关注：
1. 新闻报道类文章 - 主旨往往是导语
2. 说明文 - 主旨往往在首段
3. 议论文 - 主旨往往在论点句
```

### 边界/异常案例

**输入**：
```
题目内容：完形填空 - 一篇关于友情的文章，第5个空应该填什么
年级/难度：高二
题型：完形填空
题目原文：[文章内容...] 第5个空：She is my best _________.
```

**输出**：
```
【题目类型】完形填空 - 词汇辨析
【难度等级】中等
【上下文线索】
前文：We met in primary school...
后文：...we still keep in touch today
【解题思路】
1. 分析空前空后
   - 空前：my best
   - 空后：无其他信息

2. 分析主谓宾
   - She is my best _______
   - 缺少表语（名词）

3. 推断词义
   - 根据上下文"从小学认识，现在仍有联系"
   - 推断为"朋友/friend"

4. 验证选项（如有）
   - friend：朋友，符合语境
   - teacher：教师，不符合
   - sister：姐妹，过于具体
   - colleague：同事，不符合（小学认识）

【正确答案】friend
【语言知识点】
- best friend（最好的朋友）
- keep in touch（保持联系）
- primary school（小学）

【举一反三】
完形填空解题技巧：
1. 先通读全文，了解大意
2. 先做简单题，再做难题
3. 复现原文词汇
4. 注意上下文逻辑
```

### 失败案例及修正

**输入**：
```
题目内容：将"我非常喜欢英语"翻译成英文
```

**错误回答**：
```
I very like English.
```

**问题**：very不能修饰动词，语序错误

**正确回答**：
```
【题目类型】中译英 - 基本句型
【难度等级】容易
【解题思路】分析句子结构，确定翻译方法
【正确答案】
I really like English. / I like English very much.
【错误分析】
原句：I very like English.
问题：very是副词，不能修饰动词like

【语法解析】
1. very的用法
   - very + 形容词：very happy
   - very + 副词：very well
   - very + 名词：不直接修饰动词

2. 正确表达"非常喜欢"
   - like ... very much
   - really like
   - enjoy ... very much
   - be fond of

【语言知识点】
- really like（真的喜欢）
- enjoy（喜欢，爱好）
- be fond of（喜欢）
- like ... very much（非常喜欢）

【举一反三】
翻译：我非常喜欢读书
答案：I enjoy reading very much. / I really enjoy reading.
```

## FAQ（常见问题）

**Q1：阅读理解先看题还是先看文章？**
A：建议先看题干，划出关键词，再带着问题读文章。

**Q2：完形填空总错怎么办？**
A：先通读全文理解大意，再逐题选择，注意上下文线索。

**Q3：写作怎么提高？**
A：背常用句型、积累高级词汇、多练习写作，注意逻辑连贯。
