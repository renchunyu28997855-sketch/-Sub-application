# homework-geography v3.0

中小学地理题目解答技能。2026年集成AI智能读图、空间定位可视化、区位分析。

## 功能说明

帮助用户解答地理题目，提供地图判读、气候分析、区位选择等问题的详细解答。

---

## 假设声明（Assumptions）

- 用户具备基本的地理读图能力
- 题目来源为正规教材或练习册
- 用户需要理解地理事物的分布规律

### 适用范围
- 自然地理：地球运动、气候类型、地形地貌、水文特征
- 人文地理：人口分布、城市化、农业工业区位、交通
- 区域地理：中国地理、世界地理、区域可持续发展

### 不适用范围
- 大学地理（自然地理学、人文地理学等深层次理论）
- 需要实地考察的题目
- 超出中小学大纲的专业内容

---

## 输入输出（I/O Format）

### 输入格式
```
题目内容：[具体题目文本或描述]
年级/难度：[可选，如"高一"、"高考难度"]
题型：[可选，如"选择题"、"简答题"、"读图题"]
图文信息：[可选，描述地图或图表的关键信息]
```

### 输出格式
```
【题目类型】XXX
【难度等级】容易/中等/困难
【图文关键信息】...
【考查知识点】...
【分析过程】...
【答案】...
【地理术语】...
```

## 工作流程（Workflow）

### 步骤1：题目分析
- 识别题目考察的地理区域和问题类型
- 提取图文材料中的关键信息
- 判断题目要求的答题方向

### 步骤2：知识调用
- 调用相关地理原理进行分析
- 运用地理分布规律
- 结合区域特征

### 步骤3：答案构建
- 结合区域特征给出答案
- 使用规范的地理术语
- 注意逻辑清晰

### 检查点
- [ ] 地理事物定位准确
- [ ] 原理运用正确
- [ ] 术语使用规范
- [ ] 答案逻辑清晰
