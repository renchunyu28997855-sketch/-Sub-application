# homework-geography

## 功能说明
帮助用户解答地理题目，提供地图判读、气候分析、区位选择等问题的详细解答。

## 假设声明（Assumptions）

### 前提条件
- 用户具备基本的地理读图能力
- 题目来源为正规教材或练习册
- 用户需要理解地理事物的分布规律

### 适用范围
- 自然地理：地球运动、气候类型、地形地貌、水文特征
- 人文地理：人口分布、城市化、农业工业区位、交通
- 区域地理：中国地理、世界地理、区域可持续发展

### 不适用范围
- 大学地理（自然地理学、人文地理学等深层次理论）
- 需要实地考察的题目
- 超出中小学大纲的专业内容

## 详细工作流程（Detailed Workflow）

### 步骤1：题目分析
- 识别题目考察的地理区域和问题类型
- 提取图文材料中的关键信息
- 判断题目要求的答题方向

### 步骤2：知识调用
- 调用相关地理原理进行分析
- 运用地理分布规律
- 结合区域特征

### 步骤3：答案构建
- 结合区域特征给出答案
- 使用规范的地理术语
- 注意逻辑清晰

### 检查点
- [ ] 地理事物定位准确
- [ ] 原理运用正确
- [ ] 术语使用规范
- [ ] 答案逻辑清晰

## 边界情况（Edge Cases）

### 情况1：需要读图但无图
**处理方式**：请用户描述图中关键信息，或说明需要图片支持

### 情况2：综合分析题
**处理方式**：分角度逐步分析，确保全面

### 情况3：区位选择题
**处理方式**：从自然和社会经济多角度分析

### 情况4：超纲内容
**处理方式**：说明超出范围，提供已知相关信息

## 多个示例（Multiple Examples）

### 典型案例

**输入**：
```
题目内容：读图判断气候类型并说明特征
年级/难度：高一
题型：读图题
图文信息：最冷月气温约15℃，年降水量约2000mm，集中在夏季
```

**输出**：
```
【题目类型】气候类型判断
【难度等级】中等
【图文关键信息】
1. 最冷月气温：约15℃（位于0-15℃区间，为亚热带）
2. 年降水量：约2000mm（丰富）
3. 降水季节分配：夏季多雨
【考查知识点】气候类型判断依据、气候特征描述
【分析过程】
1. 根据最冷月气温判断热量带
   - 最冷月15℃ > 0℃ → 排除温带和寒带
   - 最冷月15℃在0-15℃区间 → 亚热带

2. 根据年降水量和季节分配判断具体气候类型
   - 年降水量2000mm：丰富
   - 夏季多雨型 → 符合季风气候特征

3. 综合判断：该地为亚热带季风气候

【答案】亚热带季风气候
特征：夏季高温多雨，冬季温和少雨；最冷月均温>0℃；年降水量>1000mm

【地理术语】亚热带季风气候、季风、气旋、反气旋、锋面雨
```

### 边界/异常案例

**输入**：
```
题目内容：分析上海发展工业的区位条件
年级/难度：高二
题型：综合分析题
```

**输出**：
```
【题目类型】工业区位分析
【难度等级】中等
【解题思路】从自然和社会经济多角度分析
【分析过程】
一、自然条件
- 地形：平坦长江三角洲平原，利于建设
- 气候：亚热带季风气候，温和湿润
- 水源：河网密布，水源充足

二、社会经济条件
- 交通：海陆空交通便利（上海港、虹桥枢纽）
- 市场：长三角经济圈，市场广阔
- 科技：高校、科研机构集中，技术力量强
- 劳动力：劳动力素质高、数量多
- 政策：国家政策支持

三、历史基础
- 近代工业发展早，基础好
- 工业门类齐全，协作条件好

【答案】上海发展工业的区位条件优越，主要得益于便利的交通、广阔的市场、强力的科技支撑和良好的政策环境。

【地理术语】区位条件、工业区位因素、交通条件、市场条件
【易错提醒】分析区位要全面，不能只考虑一两个因素
【举一反三】比较上海与郑州发展工业的区位差异
```

### 失败案例及修正

**输入**：
```
题目内容：说出世界人口分布的特点
```

**错误回答**：
```
人口分布在世界各地
```

**问题**：回答过于笼统，没有具体特点

**正确回答**：
```
【题目类型】人口分布
【难度等级】容易
【解题思路】从分布规律和影响因素角度回答
【详细解答】
世界人口分布特点：

一、分布规律
1. 北半球多于南半球（北半球陆地面积大）
2. 低纬度向高纬度递减（气候越寒冷人口越少）
3. 沿海多于内陆（沿海交通便利）
4. 平原多于山地（地形平坦、交通便利）

二、主要分布区
1. 亚洲东部和南部（农业发达地区）
2. 欧洲（工业革命发源地）
3. 北美东部（开发较早）

三、影响因素
- 气候（主要因素）
- 地形
- 水源
- 交通
- 历史开发程度

【知识点】人口分布规律、影响因素、世界人口分布区
【易错提醒】不能只说"分布不均"，要具体说明分布特点和主要分布区
```

## FAQ（常见问题）

**Q1：如何判断气候类型？**
A：先根据最冷月和最热月气温判断热量带，再根据降水量季节分配确定具体类型。

**Q2：区位分析题怎么答？**
A：从自然因素（地形、气候、水源、土壤、资源）和社会经济因素（市场、交通、劳动力、政策、技术）多角度分析。

**Q3：读图题没有图怎么办？**
A：请描述图中关键信息（如气温曲线、降水量柱状图的数据），或说明需要图片支持。
