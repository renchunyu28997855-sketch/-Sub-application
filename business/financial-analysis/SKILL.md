# financial-analysis v3.0

财务报表分析技能。2026年集成AI辅助财务分析、异常检测、自动报告生成能力。

## 核心能力

**分析维度 → 指标体系**：
| 分析维度 | 核心指标 | 用途 |
|---------|---------|------|
| 盈利能力 | 毛利率、净利率、ROE、ROA | 评估赚取利润能力 |
| 偿债能力 | 流动比率、速动比率、资产负债率 | 评估债务风险 |
| 运营能力 | 周转率、存货周转天数、应收账款周转天数 | 评估资产效率 |
| 现金流 | 经营现金流、自由现金流、现金储备 | 评估真实造血能力 |
| 成长能力 | 营收增长率、利润增长率 | 评估发展趋势 |

## 快速开始

### 输入格式
```
【分析对象】公司名称
【分析周期】年度/季度
【财务数据】三张表或关键指标
【分析目的】投资/诊断/对标/融资
```

### 输出格式
```
【核心指标】盈利能力/偿债/运营/现金流
【AI异常检测】发现的可疑点
【趋势分析】多年对比
【综合结论】优势/风险/建议
```

## 2026 AI增强功能

### AI财务分析
- **智能指标计算**：自动计算并解释财务指标含义
- **行业对标**：AI自动匹配可比公司并输出对比分析
- **自然语言报告**：财务数据自动生成可读分析报告

### 异常检测
- **财务舞弊信号**：应收账款虚增、存货异常、毛利率突变
- **关联关系验证**：三张表勾稽关系AI校验
- **时间序列异常**：同比/环比异常波动识别

### 自动报告生成
- **模板化报告**：输入数据自动输出完整财务分析报告
- **图表自动生成**：趋势图、对标雷达图自动制作
- **风险摘要**：AI自动提取关键风险点并分级

## 工作流程

### 阶段1：数据收集
1. 获取财务报表（资产负债表、利润表、现金流量表）
2. 验证数据完整性和勾稽关系
3. 识别异常数据点

### 阶段2：指标计算
4. 计算盈利能力指标（毛利率、净利率、ROE、ROA）
5. 计算偿债能力指标（流动比率、速动比率、资产负债率）
6. 计算运营能力指标（周转天数、周转率）
7. 计算现金流指标（经营现金流、自由现金流）

### 阶段3：AI辅助分析
8. AI异常检测（舞弊信号、波动异常）
9. 行业对标分析
10. 趋势分析

### 阶段4：报告输出
11. 生成综合分析报告
12. 输出风险提示和改进建议
