# retention-analysis v3.0

用户留存分析技能。2026年集成AI流失预测、行为序列分析、智能干预建议能力。

## 核心能力

**分析维度**：
| 分析阶段 | 核心指标 | 分析深度 |
|---------|---------|---------|
| 新用户激活 | 次日留存、3日留存 | 行为漏斗、引导优化 |
| 成长阶段 | 7日/14日留存 | Cohort对比、行为聚类 |
| 成熟阶段 | 30日/90日留存 | 生命周期价值、流失预警 |
| 流失阶段 | 流失率、流失周期 | AI预测、原因归因 |

## 快速开始

### 输入格式
```
【产品类型】App/网站/订阅服务/电商/游戏
【分析周期】时间段
【核心行为】定义什么是"留存"
【用户量级】DAU/MAU/总用户
【分析目标】诊断型/预测型/策略型
```

### 输出格式
```
【数据概览】核心留存指标
【流失诊断】AI识别的关键流失节点
【原因归因】流失原因优先级排序
【干预建议】AI生成的个性化策略
【效果预测】预期留存提升
```

## 2026 AI增强功能

### AI流失预测
- **早期预警**：基于行为序列预测7/30天流失概率
- **风险分层**：高/中/低流失风险用户自动识别
- **流失去向**：预测流失用户去向（竞品/沉默/自然流失）

### 行为序列分析
- **行为模式挖掘**：识别留存用户vs流失用户的行为差异
- **关键行为识别**：找到决定留存的"aha时刻"
- **路径分析**：用户从注册到流失的行为轨迹

### 智能干预建议
- **个性化策略推荐**：基于用户特征和流失原因自动生成策略
- **最优干预时机**：预测最佳触达时间窗口
- **策略效果预测**：预估各策略的留存提升效果

## 工作流程

### 阶段1：数据诊断
1. 核心指标计算（次留/7留/30留）
2. 留存曲线拟合与关键节点识别
3. Cohort多维度对比分析

### 阶段2：AI流失分析
4. 用户行为特征提取
5. AI流失风险预测模型
6. 流失原因归因分析

### 阶段3：策略生成
7. 流失原因与干预策略映射
8. ICE优先级排序
9. 个性化策略推荐

### 阶段4：效果追踪
10. 策略效果监控
11. A/B测试验证
12. 持续优化迭代
