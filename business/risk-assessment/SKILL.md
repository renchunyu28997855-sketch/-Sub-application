# risk-assessment v3.0

风险识别与评估技能。2026年集成AI风险识别、概率评估、缓解建议能力。

## 核心能力

**风险分类框架**：
| 类别 | 典型风险 | 评估重点 |
|------|---------|---------|
| 技术风险 | 技术选型、系统故障、安全漏洞 | 可靠性、扩展性、安全性 |
| 业务风险 | 需求变化、竞争加剧、供应链断供 | 市场适应、合作伙伴 |
| 运营风险 | 人员流失、流程失效、执行偏差 | 效率、质量、稳定性 |
| 合规风险 | 法规变化、合规违规、政策调整 | 合规性、许可证 |
| 财务风险 | 成本超支、收入下降、资金链断裂 | 流动性、盈利性 |

**风险值计算**：风险值 = 概率 × 影响（1-100）

| 等级 | 风险值 | 处理策略 |
|------|--------|---------|
| 🔴 高 | ≥60 | 立即处理，制定专项缓解方案 |
| 🟡 中 | 30-59 | 制定应对策略，纳入监控 |
| 🟢 低 | <30 | 持续监控，定期复盘 |

## 快速开始

### 输入格式
```
【评估对象】项目/业务/产品名称
【评估范围】涉及的领域
【评估目的】管理风险/决策参考/合规要求
```

### 输出格式
```
【评估范围】
【风险矩阵】TOP10风险
【高风险应对】策略+责任人
【监控机制】指标+告警阈值
```

## 2026 AI增强功能

### AI风险识别
- **多维度扫描**：基于行业数据和项目特征，AI自动识别潜在风险
- **风险关联分析**：识别风险传导路径和连锁反应
- **新兴风险预警**：识别行业趋势带来的新风险

### AI概率评估
- **历史数据学习**：基于行业风险数据库估算概率
- **专家判断融合**：结合主观专家意见和客观数据
- **情景模拟**：蒙特卡洛模拟概率分布

### AI缓解建议
- **策略匹配**：基于风险类型推荐最佳应对策略
- **成本效益分析**：评估策略实施成本和风险降低幅度
- **最佳实践推荐**：从行业案例库推荐可行方案

## 工作流程

### 阶段1：范围定义（步骤1-2）
1. 明确评估对象和边界
2. 识别干系人和关注点

### 阶段2：AI风险识别（步骤3-4）
3. AI多维度风险扫描
4. 风险梳理和归类

### 阶段3：AI风险评估（步骤5-6）
5. AI概率评估（含不确定性）
6. AI影响评估（多维度）

### 阶段4：风险分析（步骤7-8）
7. 计算风险值，确定优先级
8. 风险关联分析

### 阶段5：应对策略（步骤9-10）
9. AI推荐+人工确认应对策略
10. 分配责任和资源

### 阶段6：监控机制（步骤11）
11. 建立监控指标和告警阈值
