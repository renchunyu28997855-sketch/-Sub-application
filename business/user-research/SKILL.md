## 核心能力

1. **问卷设计** - 定量调研问卷，结构化问题设计
2. **访谈执行** - 深度访谈提纲与技巧
3. **用户画像** - 典型用户画像构建
4. **AI辅助分析** - 2026年AI能力集成


## 调研方法选择

| 场景 | 方法 | 样本量 | 特点 |
|------|------|--------|------|
| 验证假设 | 问卷 | n≥100 | 定量、统计显著性 |
| 探索原因 | 访谈 | 5-15人 | 定性、深度洞察 |
| 评估易用性 | 可用性测试 | 5-10人 | 行为观察 |
| 全面洞察 | 混合方法 | 先访后卷 | 定性+定量 |

## 工作流程（渐进式披露）

### 阶段一：规划

```
1. 明确调研目的 → 核心假设是什么？
2. 定义目标用户 → 谁是目标用户？
3. 选择方法 → 问卷/访谈/测试？
```

### 阶段二：设计

```
4. 设计问卷/访谈提纲
5. 优化问题（避免引导性）
6. 预测试（小范围）
```

### 阶段三：执行

```
7. 投放问卷/执行访谈
8. 数据收集与清洗
```

### 阶段四：分析

```
9. 定量/定性分析
10. AI辅助分析（画像生成、情绪分析）
11. 洞察提炼
```

## 常见问题

**Q: 样本量多少够？**
- 问卷：n≥100；访谈：5-15人（主题饱和即止）

**Q: 问卷和访谈怎么选？**
- 验证假设用问卷，探索原因用访谈

**Q: AI分析准确吗？**
- AI辅助提升效率，最终洞察需人工验证
