## 工作流程（渐进式披露）

### 阶段一：准备

```
1. 了解业务背景 → 上周期OKR/业务优先级
2. 确定目标方向 → 3-5个核心方向
3. 对齐公司/上级OKR
```

### 阶段二：OKR制定

```
4. 制定O → 鼓舞人心、简洁有力
5. 拆解KR → 每个O对应2-4个KR
6. 关联Initiatives → 关键行动对齐KR
```

### 阶段三：对齐沟通

```
7. 上下对齐 → 与上级OKR对齐
8. 横向对齐 → 与相关团队协调
9. 公开透明 → 全员可见
```

### 阶段四：执行追踪

```
10. 制定执行计划 → 月/周分解
11. 定期check-in → 周/双周更新
12. 风险识别 → 阻碍和应对
```

### 阶段五：复盘

```
13. 评分KR → 0-1分，0.7成功
14. 分析原因 → 达成/未达成
15. 经验总结 → 制定改进计划
```

## 质量检查清单

### Objective质量
- [ ] O简洁有力，1-2句话
- [ ] O鼓舞人心，有激励性
- [ ] O对齐上级/公司目标
- [ ] 每个周期3-5个O

### Key Results质量
- [ ] 每个O有2-4个KR
- [ ] KR是可量化的
- [ ] KR有挑战性（0.7完成是成功）
- [ ] KR是结果，不是任务

### 执行质量
- [ ] 有定期check-in
- [ ] KR进展有更新
- [ ] 风险和阻碍有记录

## 常见问题

**Q: OKR和KPI有什么区别？** OKR是目标管理（挑战性），KPI是绩效考核（现实可达）。

**Q: OKR完成率多少算好？** 0.6-0.7是"成功"，1.0完成说明目标不够挑战。

**Q: OKR周期内可以调整吗？** 原则上不调整，重大变化可审慎调整。
