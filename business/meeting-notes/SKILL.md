1. **会议信息整理** - 讨论要点系统梳理

## 工作流程（渐进式披露）

### 阶段一：信息收集

```
1. 收集会议信息 → 录音/笔记/议程
2. 确认基本信息 → 主题/时间/人员
3. 了解会议背景
```

### 阶段二：内容整理

```
4. 梳理讨论内容 → 按议题/时间顺序
5. 提取关键发言 → 必要时记录
6. 标注争议点和共识点
```

### 阶段三：决策与行动

```
7. 识别决策事项 → 明确结论和决策人
8. 识别行动项 → 明确内容/负责人/截止
9. 确认优先级和依赖
```

### 阶段四：纪要生成

```
10. 生成正式纪要 → 按模板格式
11. 审核和发送 → 相关人员确认
12. 建立追踪 → 纳入跟踪系统
```

## 常见问题

**Q: 会议纪要应该包含哪些内容？** 基本信息、讨论要点、决策事项、行动项、下次会议。
