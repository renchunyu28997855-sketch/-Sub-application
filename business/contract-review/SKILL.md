## 工作流程（渐进式披露）

### 阶段一：合同了解

```
1. 了解签约背景 → 目的、重要性、谈判地位
2. 获取合同文本 → 完整文本、版本、附件
3. 确认主体资格
```

### 阶段二：合同审查

```
4. 形式审查 → 结构完整性、签章、生效条件
5. 实质条款 → 标的/价格/履行/违约/终止/争议
6. 风险识别 → 高/中/低分级
```

### 阶段三：修改建议

```
7. 制定修改建议 → 针对高/中风险条款
8. 优先级排序 → P0/P1/P2
9. 提供参考文本 → 修改后条款参考
```

### 阶段四：报告输出

```
10. 生成审查报告 → 结论/风险/建议
11. 签约建议 → 法务审核/签章管理
12. 后续跟踪 → 重要风险跟踪
```

## 常见问题

**Q: 合同应该审查哪些方面？** 主体资格、标的、价格、履行、违约、终止、争议解决。

**Q: 哪些条款是必须有的？** 双方、标的、数量、价格、履行时间、违约责任、争议解决。

**Q: 发现合同风险怎么办？** 评估等级、提出修改、协商、必要时拒绝签约。
