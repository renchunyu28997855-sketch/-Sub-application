## 工作流程（渐进式披露）

### 阶段一：信息收集

```
1. 收集基础信息 → 公司/项目/团队/商业模式
2. 收集财务数据 → 营收/利润/现金流/用户
3. 收集融资历史 → 既往估值和条款
```

### 阶段二：行业分析

```
4. 行业分析 → 规模/增长/竞争/趋势
5. 可比公司分析 → 选择合适可比
6. 估值基准 → 行业平均倍数
```

### 阶段三：估值计算

```
7. 选择估值方法 → 早期用户/成长PS/成熟PE
8. 多方法估值 → ARR倍数/DC/用户
9. 综合估值 → 加权确定区间
```

### 阶段四：投资分析

```
10. 投资条款建议 → 估值/股份/保护条款
11. 风险分析 → 业务/财务/团队/市场
12. 退出路径 → IPO/并购/转让
```

## 常见问题

**Q: 早期项目怎么估值？** 团队因素、赛道市场规模、技术壁垒、类比法。

**Q: 估值和投资条款什么关系？** 估值决定股权比例，条款保护投资人权利。

**Q: 如何避免估值过高？** 合理假设、选择合适可比、考虑风险和稀释。
