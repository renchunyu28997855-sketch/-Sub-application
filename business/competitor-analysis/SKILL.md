## 工作流程（渐进式披露）

### 阶段一：范围界定

```
1. 明确分析边界 → 直接竞品/间接竞品/替代品
2. 信息来源 → 官方渠道/用户渠道/行业渠道
3. 分析目的 → 产品规划/融资/战略
```

### 阶段二：信息收集

```
4. 基础信息 → 产品功能/定价/目标用户
5. 深度挖掘 → 用户评价/功能体验/融资动态
6. AI采集 → 自动追踪竞品更新
```

### 阶段三：分析框架

```
7. 多维对比 → 产品/市场/商业/用户
8. SWOT分析 → 优势/劣势/机会/威胁
9. 差异化空间 → 市场空白点识别
```

### 阶段四：洞察输出

```
10. 策略建议 → 差异化切入点
11. 风险提示 → 竞品威胁/新进入者
12. 监控告警 → 设置竞品动态提醒
```

## 常见问题

**Q: 如何选择竞品分析深度？** 快速概览适合日常监控，深度分析适合产品规划，全面调研适合融资。

**Q: 竞品信息从哪获取？** 官网/应用商店/用户评价/行业报告/社交媒体/招聘广告等。

**Q: 如何保证分析客观性？** 多方信源交叉验证，避免主观偏好，用数据说话。
