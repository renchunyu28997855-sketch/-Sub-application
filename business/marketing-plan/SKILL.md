# marketing-plan

## 功能说明
帮助制定完整的营销方案，覆盖渠道选择、内容策略、预算分配和效果追踪。

---

## 1. 假设声明（Assumptions）

### 前提条件
- 用户有明确的产品或服务
- 目标市场已定义
- 有一定的营销预算（或预算待定）
- 产品已上市或即将上市

### 适用范围
- 新产品上市营销
- 品牌推广营销
- 活动营销（大促、新品发布）
- 效果营销（付费广告、增长）
- 私域运营
- 全域营销整合

### 不适用场景
- 纯品牌建设无转化目标（适合品牌策略）
- 极度受限的预算（需重新定义目标）
- 产品尚未准备好的预热（可能浪费资源）

### 市场假设
- 目标用户可触达（渠道覆盖）
- 竞争对手可研究
- 市场信息可获取

## 3. 详细工作流程（Detailed Workflow）

### 阶段一：目标设定（第1-2步）

```
步骤1：明确营销目标
├── 业务目标（营收、用户数、市场份额）
├── 营销目标（获客、激活、留存）
├── 品牌目标（认知度、美誉度）
├── 目标量化（SMART）
└── 检查点：目标是否与业务目标对齐

步骤2：定义目标用户
├── 人口统计特征
├── 行为特征
├── 心理特征
├── 决策旅程阶段
└── 检查点：用户画像是否清晰
```

### 阶段二：策略规划（第3-5步）

```
步骤3：设计营销漏斗
├── 认知（Awareness）：如何让用户知道
├── 兴趣（Interest）：如何让用户感兴趣
├── 欲望（Desire）：如何让用户想要
├── 行动（Action）：如何让用户购买
├── 留存（Retention）：如何让用户复购
└── 检查点：漏斗是否完整

步骤4：选择营销渠道
├── 付费渠道（SEM、信息流、KOL）
├── 内容渠道（短视频、图文、直播）
├── 私域渠道（微信、社群、邮件）
├── 口碑渠道（SEO、PR、用户推荐）
├── 渠道特性匹配用户旅程
└── 检查点：渠道是否覆盖目标用户

步骤5：制定内容策略
├── 内容类型（品牌内容、产品内容、用户内容）
├── 内容主题（围绕用户痛点和兴趣）
├── 内容形式（图文、短视频、直播）
├── 内容调性（符合品牌定位）
└── 检查点：内容是否有差异化
```

### 阶段三：预算分配（第6-7步）

```
步骤6：分配预算
├── 渠道预算分配
├── 内容制作预算
├── 活动预算
├── 机动预算（应急）
└── 检查点：预算是否合理

步骤7：设定效果指标
├── 漏斗各阶段指标
├── KPI（关键业绩指标）
├── ROI计算方式
└── 检查点：指标是否可衡量
```

### 阶段四：执行计划（第8-9步）

```
步骤8：制定执行计划
├── 时间表（甘特图）
├── 里程碑
├── 责任分工
├── 资源需求
└── 检查点：计划是否可执行

步骤9：建立监控机制
├── 数据看板
├── 监测频率
├── 复盘机制
└── 检查点：监控是否到位
```

## 4. 边界情况（Edge Cases）

### 输入为空的情况

| 场景 | 处理方式 |
|------|----------|
| 用户只说"帮我做营销方案" | 询问产品、目标、预算、时间 |
| 预算未定 | 提供行业基准，建议预算范围 |
| 目标模糊 | 提供常见营销目标选项 |
| 竞品未知 | 提供竞品研究框架 |

### 特殊情况

| 场景 | 处理方式 |
|------|----------|
| 预算极少 | 聚焦单一渠道，内容驱动 |
| 预算充足 | 多渠道组合，快速测试放大 |
| 市场竞争激烈 | 差异化定位，内容为王 |
| 新兴市场 | 教育市场，内容先行 |
| B2B企业营销 | 强调线索质量和转化周期 |

### 预算分配场景

| 场景 | 策略 |
|------|------|
| 预算<10万 | 聚焦1-2个渠道，内容营销为主 |
| 预算10-50万 | 2-3个渠道组合 |
| 预算50-100万 | 多渠道+活动 |
| 预算>100万 | 全渠道+持续优化 |

## 5. 错误处理（Error Handling）

### 常见错误

| 错误类型 | 识别特征 | 处理方式 |
|----------|----------|----------|
| 目标过多 | 同时追增长、品牌、线索 | 聚焦核心目标，排序优先级 |
| 渠道贪多 | 所有渠道都做 | 选择核心渠道，深耕 |
| 内容同质 | 和竞品内容相似 | 差异化定位 |
| 忽视私域 | 只关注拉新 | 建立用户沉淀机制 |
| ROI不跟踪 | 只管投不管效果 | 建立追踪机制 |

### 效果问题

| 问题 | 可能原因 | 解决方向 |
|------|----------|----------|
| 曝光低 | 渠道选择/内容问题 | 测试新渠道/优化内容 |
| 转化低 | 落地页/产品问题 | 优化转化路径 |
| ROI低 | 渠道效率低 | 优化渠道组合 |
| 线索质量差 | 用户定位问题 | 精准定向 |

### 失败案例：某营销方案问题

**问题特征**：
- 目标写着"提升品牌知名度"
- 渠道列了10个
- 预算平均分配
- 没有追踪ROI

**问题诊断**：
- 目标不可衡量
- 渠道贪多嚼不烂
- 资源分散
- 无法评估效果

**修正方向**：
- 目标量化（曝光量、转化量）
- 聚焦2-3个核心渠道
- 按效果分配预算
- 建立ROI追踪

## 8. FAQ（常见问题）

### Q1：营销目标和业务目标有什么区别？
**A**：
- 业务目标：营收、用户数、市场份额
- 营销目标：获客量、获客成本、转化率
- 关系：营销目标服务于业务目标

### Q2：如何选择营销渠道？
**A**：
1. 用户在哪？（目标用户调研）
2. 竞品在哪？（竞品渠道分析）
3. 我们擅长什么？（自身能力）
4. 成本和效果？（测试数据）

### Q3：内容营销多久见效？
**A**：
- SEO：3-6个月
- 社交媒体：持续积累
- KOL合作：短期效果
- PR公关：持续曝光
- 建议：内容营销是长期投资

### Q4：如何提高ROI？
**A**：
1. 精准定向（减少浪费）
2. 优化创意（提升点击）
3. 优化落地页（提升转化）
4. 用户分层（差异化策略）
5. 私域运营（降低复购成本）

### Q5：预算怎么分配？
**A**：
- 常见比例：付费60%+内容20%+活动20%
- 根据阶段调整：初创期内容为主，成长期付费+内容，稳定期私域为主
- 预留10-20%机动预算

### Q6：营销效果不好怎么办？
**A**：
1. 检查数据是否准确
2. 分析哪个环节出问题（曝光/点击/转化）
3. 调整策略或渠道
4. 小范围测试新方案
5. 数据驱动决策
