## 工作流程（渐进式披露）

### 阶段一：目标设定

```
1. 明确营销目标 → 业务目标/营销目标/品牌目标
2. 定义目标用户 → 用户画像/行为特征/决策旅程
3. 竞品分析 → 竞品策略/差异化机会
```

### 阶段二：策略规划

```
4. 设计营销漏斗 → 认知/兴趣/欲望/行动/留存
5. 选择营销渠道 → 付费/内容/私域/口碑渠道
6. 制定内容策略 → 内容类型/主题/形式/调性
```

### 阶段三：执行规划

```
7. 分配预算 → 渠道预算/内容预算/机动预算
8. 制定执行计划 → 时间表/里程碑/分工
9. 设定效果指标 → KPI/ROI/复盘机制
```

## 常见问题

**Q: 如何选择核心渠道？** 选用户聚集、竞品有效、自身擅长的渠道。小预算聚焦1-2个。

**Q: 预算如何分配？** 常见：付费60%+内容20%+活动20%。初创期以内容为主。

**Q: 多久见效？** SEO需3-6个月，社媒持续积累，付费短期见效。

**Q: ROI低怎么办？** 精准定向、优化创意/落地页、用户分层、私域运营。
