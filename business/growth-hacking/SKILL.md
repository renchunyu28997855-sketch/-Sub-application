## 工作流程（渐进式披露）

### 阶段一：诊断

```
1. 定义北极星指标 → 业务核心/用户价值/可衡量
2. 分析增长漏斗 → 获客/激活/留存/裂变/变现
3. 定位瓶颈 → 数据驱动找到最大损失点
```

### 阶段二：策略

```
4. 制定增长策略 → AARRR各阶段策略矩阵
5. 优先级排序 → ICE评分/Impact/Effort
6. 设计增长实验 → 假设/对照/样本/周期
```

### 阶段三：执行

```
7. 开发实现 → 功能/埋点/AB框架
8. 运行实验 → 数据监控/异常处理
9. 分析决策 → 统计显著性/效果评估
```

## 常见问题

**Q: 如何选择北极星指标？** 选择反映用户价值、可衡量、简单的核心指标。常见：DAU/MAU/付费用户数。

**Q: 增长实验多久跑一次？** 通常1-2周，需达到95%置信度。每周可跑2-4个实验。

**Q: 裂变活动怎么防作弊？** 设备指纹+行为检测+奖励延迟+人工审核。

**Q: 留存低怎么办？** 分析流失节点→调研流失原因→优化新用户引导→建立召回机制。
