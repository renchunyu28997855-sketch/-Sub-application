## 工作流程（渐进式披露）

### 阶段一：信息收集

```
1. 了解产品价值 → 核心价值、差异化
2. 分析成本结构 → 直接/间接/边际成本
3. 研究竞品定价 → 价格带、策略
```

### 阶段二：策略选择

```
4. 选择定价方法 → 成本/竞争/价值/动态
5. 确定定价策略 → 分层/免费增值/捆绑
6. 设计版本差异 → 功能/用户/存储差异
```

### 阶段三：方案设计

```
7. 设计价格结构 → 月付/年付/终身
8. 设定具体价格 → 价值锚定、折扣策略
9. 验证价格方案 → 盈亏平衡、敏感度
```

### 阶段四：执行与优化

```
10. 制定上线计划 → 过渡策略、沟通
11. 监控系统 → 转化率、ARPU、续费率
12. AI持续优化 → 基于数据的动态调整
```

## 常见问题

**Q: 如何知道用户愿意付多少钱？** 用户调研、竞品对比、试销测试、拍卖竞价。

**Q: 定价高了怎么调整？** 分析原因，增加价值或降低价格，改变结构。

**Q: 如何应对价格战？** 避免正面战，差异化，聚焦细分。
