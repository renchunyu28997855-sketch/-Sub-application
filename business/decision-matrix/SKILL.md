2. **利弊分析** - 优缺点系统梳理

## 工作流程（渐进式披露）

### 阶段一：问题定义

```
1. 明确决策问题 → 目标、范围、约束
2. 收集方案信息 → 列出所有可行方案
3. 确认方案完整性 → 排除不可行方案
```

### 阶段二：分析方法选择

```
4. 选择分析方法 → 加权评分/利弊/情景/决策树
5. 确定评估因素 → 识别关键维度
6. 设定因素权重 → 确定各因素重要性
```

### 阶段三：分析执行

```
7. 执行分析 → 打分、计算、梳理
8. 敏感性分析 → 测试假设变化
9. 风险评估 → 识别决策风险
```

### 阶段四：结论输出

```
10. 综合结论 → 汇总各方法结论
11. 提出建议 → 结合偏好给出建议
12. 制定监控 → 决策后监控计划
```

## 常见问题

**Q: 如何确定因素权重？** 专家讨论法、层次分析法(AHP)、参照历史。

**Q: 评分主观怎么办？** 明确评分标准、多方参与、敏感性分析。

**Q: 多个方法结论矛盾怎么办？** 检查假设、识别核心差异、尝试加权综合。
