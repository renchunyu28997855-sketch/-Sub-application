# blog-writer v3.0

博客文章创作技能。2026年集成Human-AI协作工作流、风格延续、多平台适配能力。

## 功能说明
创作各类博客文章，包括教程类、观点类、行业分析类，提供吸引读者的开头和SEO优化建议。

---

## 假设声明（Assumptions）

1. **主题明确**：用户有明确的文章主题和大致方向
2. **目标读者可知**：用户大致了解文章面向的读者群体
3. **内容真实性**：用户提供的事实、数据、案例是真实可靠的
4. **适用场景**：内容营销博客、技术博客、个人博客、资讯博客

## 工作流程（Workflow）

### 阶段一：主题分析与素材收集
1. **理解主题和角度** - 确认核心观点、差异化角度
2. **素材收集与整理** - 用户提供的资料/数据、补充背景信息

### 阶段二：结构设计
3. **制定文章大纲** - 设计开头（吸引读者）、主体结构、结尾
4. **SEO关键词布局** - 确定核心关键词，规划分布

### 阶段三：内容创作
5. **撰写各部分内容** - 开头痛点引入、主体论点+论据+案例、结尾总结+行动引导
6. **优化标题和SEO元素** - 主标题含关键词、Meta描述诱导点击

### 检查点
- [ ] 主题是否突出
- [ ] 结构是否清晰
- [ ] 是否有干货价值
- [ ] SEO元素是否完整

---

## 质量验证清单（Quality Checklist）

### 内容价值检查
- [ ] 文章有明确的核心观点或价值点
- [ ] 内容真实可信，无虚假信息
- [ ] 对目标读者有实际帮助

### 结构逻辑检查
- [ ] 开头能吸引读者继续阅读
- [ ] 逻辑层次清晰，段落分明
- [ ] 每个观点有充分论据支撑
- [ ] 结尾有总结或行动引导

### SEO检查
- [ ] 标题包含核心关键词
- [ ] Meta描述有吸引力且含关键词
- [ ] 关键词分布自然合理（密度2-3%）
