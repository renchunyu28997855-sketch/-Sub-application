# brand-story

## 功能说明
帮助品牌构建有感染力的叙事，包括品牌起源故事、创始人故事、品牌价值观等。

---

## 假设声明（Assumptions）

1. **品牌真实存在**：用户描述的是真实品牌，非虚构
2. **情感真诚**：品牌故事应基于真实经历，虚构故事会损害信任
3. **目标明确**：用户了解品牌故事的用途（官网/宣传册/社交媒体）
4. **适用场景**：品牌官网、企业宣传册、社交媒体、融资材料
5. **不支持**：虚假包装、欺骗性叙事

## 详细工作流程（Detailed Workflow）

### 阶段一：信息收集
1. **了解品牌背景**
   - 创始人和团队背景
   - 创立时间和地点
   - 创立契机和初心

2. **挖掘故事元素**
   - 真实的困难和挑战
   - 突破性时刻
   - 价值观形成故事
   - 用户与品牌的故事

### 阶段二：主题提炼
3. **确定叙事主线**
   - 核心主题是什么
   - 情感基调如何
   - 如何与目标受众共鸣

4. **设计故事结构**
   - 开头：如何引人入胜
   - 发展：核心故事线
   - 高潮：转折点或突破
   - 结尾：品牌使命/愿景

### 阶段三：内容创作
5. **撰写多个版本**
   - 短版：适合社交媒体/简介
   - 中版：适合官网/宣传册
   - 长版：适合深度阅读

6. **优化与调整**
   - 语言流畅有感染力
   - 细节真实可信
   - 情感真挚不煽情

### 检查点
- [ ] 故事真实可查证
- [ ] 情感真实不夸张
- [ ] 与品牌定位一致
- [ ] 多版本结构一致

## 边界情况（Edge Cases）

### 情况1：品牌故事平淡
**问题**：品牌经历没有明显戏剧性
**处理方式**：
- 挖掘日常中的不平凡
- 从用户角度讲故事
- 强调"坚持"本身的力量

### 情况2：创始人背景复杂
**问题**：创始人经历丰富不知如何取舍
**处理方式**：
- 选择与品牌最相关的经历
- 围绕品牌使命筛选素材
- 避免堆砌经历

### 情况3：多品牌/多产品线
**问题**：企业有多个品牌如何处理
**处理方式**：
- 区分企业品牌与产品品牌
- 以企业品牌故事为根基
- 各产品线可有独立小故事

## 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理方式 |
|----------|----------|----------|
| 虚假夸大 | 明显不实陈述 | 标注并建议修改 |
| 抄袭痕迹 | 与知名品牌故事高度相似 | 重新创作 |
| 情感过度 | 煽情过度适得其反 | 调整为更克制的表达 |
| 信息混乱 | 多版本内容矛盾 | 统一各版本核心信息 |

### 失败案例及修正

**原始问题**：
```
我们公司成立于2010年，是一家专注于做最好产品的公司。我们的产品非常好，品质第一，用户至上，欢迎大家购买。
```

**问题分析**：
1. 缺乏具体故事 - 只有空洞描述
2. 语言套话 - "品质第一、用户至上"谁都会说
3. 没有独特性 - 任何公司都这么说
4. 无法引发共鸣

**修正后**：
```
【叙事主线】从被供应商坑了50万开始，坚定做"透明"的企业

【故事节选】
2012年，我们被合作了3年的供应商坑了50万。那是我们全部的现金储备。

那一个月，我每天只睡3个小时。合伙人问我怎么办。我说：我们不追回那笔钱，但我们从此只找愿意公开成本的供应商。

客户问：你们凭什么比别人贵10%？
我说：因为我们敢把所有成本摊开给你看。

10年了，我们没欠过供应商一分钱。我们的毛利率比行业低5个百分点，但我们的供应商留存率是97%。

这不是什么商业秘诀。这就是我们笨拙的做人方式。
```

## FAQ（常见问题）

**Q1：品牌故事一般多长？**
A：视用途而定：
- 社交媒体简介：50-150字
- 官网"关于我们"：300-500字
- 品牌宣传册：500-1000字
- 融资材料：300-500字

**Q2：品牌故事和品牌定位有什么区别？**
A：
- 品牌定位：我是谁，我能做什么（理性层面）
- 品牌故事：我为什么存在，经历了什么（情感层面）
- 两者需一致，故事服务于定位

**Q3：初创公司需要品牌故事吗？**
A：需要，但可以从小处着手：
- 创始人故事是最容易入手的角度
- 不用刻意编造，真实经历即可
- 随着品牌发展，故事可以演进

**Q4：可以虚构品牌故事吗？**
A：强烈不建议。虚构故事一旦被揭穿，会严重损害品牌信任。

**Q5：品牌故事需要多久更新一次？**
A：建议：
- 重大里程碑时可更新（如融资、重大转型）
- 但核心叙事主线应保持稳定
- 避免频繁变化让消费者困惑

# 短版（100字）

植感生活，起源于一次皮肤"抗议"。

2018年，我用了市面上的沐浴露后全身过敏。医生说是我用的产品里的化学添加剂在作怪。

那一刻我开始想：有没有更纯粹的东西，可以干净地清洁身体，但不伤害皮肤？

于是我开始研究手工皂。找了云南的冷榨植物油，请教了老工匠，经历了无数次失败。

现在，每一块皂都是28天的等待。72岁的奶奶说，这才叫肥皂。

---

# 中版（300字）

植感生活，起源于一次皮肤"抗议"。

2018年的春天，我用了某知名品牌的沐浴露后全身过敏。皮肤科医生说是化学添加剂导致的接触性皮炎。那一刻我开始想：我们每天用的洗护产品，到底含有什么？

带着这个疑问，我开始研究成分表。SLS、SLES、合成香料、paraben……这些名字让我越看越不安。

于是我开始寻找更纯粹的清洁方式。古法手工皂进入了我的视野——用冷榨植物油和碱反应，不添加合成表面活性剂，温和清洁。

我找到了云南做冷榨植物油的老厂家，请教了有30年经验的手工皂匠人。配方调整了40多次，皂化反应失败过无数次，有时候一锅皂整整28天，最后得到的是一滩松糕。

但我没有放弃。因为我知道，真正的健康，是需要时间等待的。

现在，植感生活的每一块皂，从配料到成熟，需要整整28天。用的是云南的冷榨山茶油、希腊的初榨橄榄油、泰国的手工椰浆。

72岁的老匠人说：一块好皂，讲究的是时间。
