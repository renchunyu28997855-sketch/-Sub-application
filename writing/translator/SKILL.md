# translator v3.0

中英文互译润色技能。2026年集成语境感知翻译、风格延续、多领域自适应能力。

## 功能说明
提供专业的中英文互译服务，以及现有译文的润色优化，确保语言地道流畅。

---

## 假设声明（Assumptions）

1. **原文内容合法**：用户提供的翻译内容不涉及违法、色情、暴力等
2. **版权清晰**：用户有权使用该内容进行翻译
3. **专业领域已知**：用户了解内容所属的专业领域
4. **适用场景**：商务合同、技术文档、营销内容、日常沟通

---

## 输入输出（I/O Format）

### 输入格式
```
【翻译类型】中译英/英译中/润色校对
【原文语言】中文/英文
【风格要求】正式/商务/技术/文学/日常
【专业领域】（可选）通用/金融/法律/医学/IT/营销
【语境说明】（可选）使用场景、目标读者
```

### 输出格式
```
【翻译类型】XXX
【原文】
...
【译文】
...
【译注】（如有难点说明）
...
【术语表】（如有专有名词）
...
```

## 工作流程（Workflow）

### 阶段一：需求分析
1. **确认翻译需求** - 翻译方向、风格、专业领域
2. **理解原文内容** - 字面意思、深层含义、语用风格

### 阶段二：翻译执行
3. **初次翻译** - 准确传达原意，保持风格一致
4. **润色优化** - 调整句式结构，优化用词选择

### 阶段三：审核校对
5. **质量检查** - 核对原文与译文，检查语法正确性
6. **难点标注** - 标注有争议的翻译，说明翻译理由
