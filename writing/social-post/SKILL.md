# social-post v3.0

社交媒体内容创作技能。2026年集成AI内容生成、热点分析、多平台适配能力。

## 功能说明
为不同社交媒体平台创作内容，包括微博、小红书、知乎、抖音等，确保内容符合平台调性和用户喜好。

---

## 假设声明（Assumptions）

1. **平台明确**：用户明确指定目标平台
2. **账号定位已知**：用户的账号风格/粉丝群体大致了解
3. **内容方向清晰**：用户有大致的内容主题
4. **适用场景**：个人IP打造、品牌营销、种草分享、知识输出

---

## 输入输出（I/O Format）

### 输入格式
```
【目标平台】小红书/微博/知乎/抖音/公众号
【内容主题】XXX
【内容类型】种草/测评/教程/观点/日常分享
【核心信息】1. XXX 2. XXX
【风格偏好】（可选）接地气/高端/专业/搞笑
```

### 输出格式
```
【目标平台】XXX
【标题】XXX
【话题标签】#XXX #XXX
【正文】
...
【互动引导】
...
```

## 2026年新功能

### AI内容生成
- **智能改写**：一键优化文案，提升传播力
- **风格迁移**：适配不同平台的语言风格
- **批量生成**：同时产出多平台内容版本

### 热点分析
- **实时追踪**：监控平台热门话题和趋势
- **借势建议**：推荐适合追的热点的切入角度
- **竞品分析**：分析同类爆款内容的规律

### 多平台适配
| 平台 | 内容风格 | 长度 | 最佳时间 |
|------|----------|------|----------|
| 小红书 | 种草、真实分享 | 300-1000字 | 18:00-21:00 |
| 微博 | 短平快、话题感 | 100-300字 | 12:00-13:00 |
| 知乎 | 专业、深度 | 500-3000字 | 工作时间 |
| 抖音 | 视觉冲击、节奏感 | 脚本15-60秒 | 12:00-14:00 |

## 工作流程（Workflow）

### 阶段一：平台分析
1. 确认目标平台特点和用户群体
2. 分析平台内容形式和标签使用习惯

### 阶段二：内容策划
3. 确定内容角度和差异化
4. 匹配平台语言风格

### 阶段三：创作优化
5. 撰写符合平台调性的正文
6. 设计标签和互动引导

### 检查点
- [ ] 符合平台调性
- [ ] 内容真实可信
- [ ] 标签使用得当
- [ ] 有明确互动引导
