# localization v3.0

内容本地化翻译技能。2026年集成文化智能分析、多市场并行适配、法规自动检测能力。

## 功能说明
帮助内容进行本地化适配，包括文化调整、用语本地化、法规合规等，让内容在目标市场更接地气。

---

## 假设声明（Assumptions）

1. **目标市场明确**：用户有明确的目标国家/地区
2. **文化背景已知**：用户了解目标市场的基本文化特点
3. **内容可调整**：用户允许在翻译基础上进行文化适配调整
4. **适用场景**：App/网站出海、跨国品牌、跨境电商、海外营销

---

## 输入输出（I/O Format）

### 输入格式
```
【内容类型】App界面/营销文案/产品说明/网站内容
【源语言】中文/英文
【源内容】XXX
【目标市场】XXX（如：美国、日本、东南亚）
【目标语言】（如：英语、日语）
【文化背景】（可选）已知的文化特点
```

### 输出格式
```
【目标市场】XXX
【文化背景】XXX
【本地化要点】
1. 文化适配...
2. 用语调整...
3. 法规注意...
【原文】
...
【本地化后】
...
【文化适配说明】
...
```

## 工作流程（Workflow）

### 阶段一：市场调研
1. **了解目标市场** - 文化特点、语言习惯、社会价值观和禁忌
2. **分析内容类型** - 不同类型内容的本地化要求不同

### 阶段二：本地化策划
3. **制定本地化策略** - 文化元素处理方式、视觉元素调整、法规合规要求
4. **处理文化差异** - 俗语/比喻的本地化、颜色/数字的文化含义

### 阶段三：执行与审核
5. **进行本地化翻译** - 翻译基础内容、文化适配调整
6. **审核本地化质量** - 语法正确性、文化适当性、用户体验流畅

## 质量验证清单（Quality Checklist）

### 语言质量检查
- [ ] 语法正确
- [ ] 表达地道
- [ ] 术语统一
- [ ] 语气适当

### 文化适配检查
- [ ] 无文化冲突
- [ ] 无文化冒犯
- [ ] 符合当地习惯
- [ ] 节日/事件已本地化

### 合规性检查
- [ ] 符合当地法规
- [ ] 无虚假宣传
- [ ] 广告法合规
