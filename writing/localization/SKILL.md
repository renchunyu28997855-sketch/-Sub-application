# localization

## 功能说明
帮助内容进行本地化适配，包括文化调整、用语本地化、法规合规等，让内容在目标市场更接地气。

---

## 假设声明（Assumptions）

1. **目标市场明确**：用户有明确的目标国家/地区
2. **文化背景已知**：用户了解目标市场的基本文化特点
3. **内容可调整**：用户允许在翻译基础上进行文化适配调整
4. **适用场景**：App/网站出海、跨国品牌、跨境电商、海外营销
5. **不支持**：高度专业化内容（法律/医学，建议专业机构）

## 详细工作流程（Detailed Workflow）

### 阶段一：市场调研
1. **了解目标市场**
   - 目标国家/地区的文化特点
   - 语言习惯和表达方式
   - 社会价值观和禁忌

2. **分析内容类型**
   - 不同类型内容的本地化要求不同
   - 营销类需要高度文化适配
   - 技术类侧重准确翻译

### 阶段二：本地化策划
3. **制定本地化策略**
   - 文化元素处理方式
   - 视觉元素调整
   - 法规合规要求

4. **处理文化差异**
   - 俗语/比喻的本地化
   - 颜色/数字的文化含义
   - 节日/事件的本地化

### 阶段三：执行与审核
5. **进行本地化翻译**
   - 翻译基础内容
   - 文化适配调整
   - 视觉元素建议

6. **审核本地化质量**
   - 语法正确性
   - 文化适当性
   - 用户体验流畅

### 检查点
- [ ] 语言地道自然
- [ ] 文化适配恰当
- [ ] 无文化冒犯
- [ ] 法规合规

## 边界情况（Edge Cases）

### 情况1：文化元素冲突
**问题**：源文化中的概念在目标文化中不存在
**处理方式**：
- 直译+解释
- 用目标文化中相似概念替代
- 泛化处理，保留核心含义

### 情况2：敏感内容
**问题**：某些内容在目标文化中可能敏感
**处理方式**：
- 识别潜在敏感点
- 调整表达方式
- 避免文化冲突

### 情况3：法规差异
**问题**：不同国家/地区法规要求不同
**处理方式**：
- 识别法规差异点
- 调整至符合目标市场法规
- 建议咨询当地法律专家

## 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理方式 |
|----------|----------|----------|
| 文化直译 | 直译后目标用户无法理解 | 提供本地化方案 |
| 冒犯性表达 | 目标文化中的禁忌表达 | 重新设计表达 |
| 法规违规 | 违反目标市场监管要求 | 调整至合规 |
| 翻译腔 | 逐字翻译导致不自然 | 重新意译 |

### 边界案例1：涉及数字/颜色的本地化

**输入**：
```
【内容类型】营销文案
【源内容】"七夕牵手价，仅限7.7折！"
【目标市场】美国
```

**处理输出**：
```
【本地化分析】
1. **七夕**：中国传统情人节，西方用户不熟悉
2. **数字7.7**：中文谐音"亲"，西方无此文化含义
3. **"牵手"**：暗示情侣/浪漫，但直接翻译不自然

【本地化方案】

方案A（保留浪漫主题，淡化中国节日）：
"Valentine's Day Special! Get 22% OFF on couples packages. Valid only this week!"

方案B（直接欧美节日替换）：
"Heart's Day Exclusive! 22% OFF when you book together. Limited time offer."

方案C（无节日关联，纯促销）：
"Special Couple's Deal! Save 22% when you book together. Don't miss out!"

【推荐】方案A，保留浪漫主题但去中国化

【说明】
- 7.7折 → 22% OFF（西方习惯用折扣百分比）
- 七夕 → 泛化为Valentine's Day或无节日
- 强调时效性，但不制造过度紧迫感
```

## FAQ（常见问题）

**Q1：本地化和翻译有什么区别？**
A：
- 翻译：把文字从一种语言转为另一种语言
- 本地化：不只是翻译，还要适配目标市场的文化、法规、用户习惯

**Q2：哪些市场需要特别注意？**
A：
- 中东（宗教文化）
- 日本（礼仪规范）
- 欧美（直接vs委婉）
- 东南亚（多元文化）

**Q3：可以一次处理多个市场吗？**
A：可以，但不同市场需要不同的本地化方案。建议一次一个市场以保证质量。

**Q4：本地化需要多久？**
A：取决于内容类型和复杂度。一般是翻译时间的1.5-2倍。

**Q5：可以提供文化培训资料吗？**
A：可以。在输出中我会包含目标市场的文化背景说明。

## 示例
**输入**: 帮我把这款中国App的推广文案本地化到美国市场
**output**:
【目标市场】美国
【文化背景】
- 直接、简洁的表达风格
- 注重个人隐私和自主性
- 不喜欢被"强迫"或"催促"
- 喜欢真实的用户评价

【本地化要点】
1. 语气调整：从"快来买！"改为更自然的推荐口吻
2. 隐私强调：美国用户非常注重隐私
3. 避免直译：中式表达如"砍价"需要重新设计
4. 单位换算：温度、距离、货币单位
5. 节日适配：用美国节日替换中国节日

【原文】
"限时特惠！双人拼团低至5折，邀请好友一起省钱，错过再等一年！"

【本地化后】
"Exclusive Deal! Get 50% OFF when you and a friend sign up together. Your friend will thank you later. Limited time offer."

【文化适配说明】
- "拼团" → "sign up together"（美国人更注重个人自主）
- "错过再等一年" → "Limited time offer"（更温和的紧迫感）
- 删除"快来买"的催促感，改为"your friend will thank you later"的推荐口吻

【另一个例子】

【原文】（中国区推广）
"中秋赏月，配月饼，更配我们的App"

【本地化后】（美国市场）
"Snuggle up with your favorite fall treats this season - our app is the perfect companion."

【说明】
中秋节和月饼文化美国人无法共鸣，替换为美国的秋季文化（南瓜香料、舒适感）
