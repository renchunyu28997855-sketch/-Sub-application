# seo-optimizer v3.0

SEO内容优化技能。2026年集成AI能力，支持智能关键词分析、内容评分、竞品分析。

## 功能说明

帮助优化网页和文章内容，提升搜索引擎排名。包括关键词研究、标题优化、内容结构优化、内链策略等。

---

## 假设声明（Assumptions）

1. **目标明确**：用户已有待优化的内容（文章/页面）或明确的主题
2. **平台限定**：用户知道目标平台（知乎、公众号、独立站等）
3. **关键词可获取**：用户可提供或授权使用目标关键词
4. **适用场景**：内容营销网站、博客文章、电商详情页、知识库
5. **不支持**：黑帽SEO、虚假内容堆砌、站群内容

---

## 输入输出（I/O Format）

### 输入格式

**场景A - 已有内容优化：**
```
【原始内容】粘贴文章/页面内容
【目标关键词】XXX（如：空气炸锅选购指南）
【竞争对手】（可选）参考页面URL
【特殊要求】XXX
```

**场景B - 新内容SEO规划：**
```
【主题】XXX
【目标关键词】主关键词 + 2-3个长尾关键词
【文章长度】建议字数
【竞争分析】（可选）是否需要竞品分析
```

### 输出格式

```
【目标关键词】XXX
【关键词分析】
- 搜索量：XXX
- 竞争度：低/中/高
- 推荐指数：★★★☆☆
【当前问题】
...
【优化建议】
1. 标题优化
2. Meta优化
3. 内容结构调整
4. 内链建议
【修改后示例】
...
```

## 2026年新功能

### AI关键词分析
- **智能扩展**：基于种子关键词自动扩展语义相关词
- **搜索意图识别**：判断导航/信息/交易意图
- **竞争度预测**：分析首页结果质量估算排名难度

### AI内容评分
- **多维度评分**：可读性、专业性、SEO友好度、独特价值
- **即时反馈**：生成优化建议列表
- **与竞品对比**：内容差距可视化

### AI竞品分析
- **Top10分析**：分析竞品内容结构、关键词布局、外链策略
- **差距识别**：找出内容薄弱点和机会点
- **差异化建议**：提供独特角度和内容机会

## 工作流程（Workflow）

### 阶段一：关键词研究
1. 确认目标关键词（核心+长尾+语义相关）
2. AI分析关键词价值（搜索量、竞争度、转化潜力）

### 阶段二：内容分析
3. 评估现有内容（如有）- 关键词密度、结构、可读性
4. AI识别优化点（标题、Meta、内容结构、内链）

### 阶段三：优化方案
5. 制定优化策略
6. AI竞品分析（如需要）
7. 输出具体建议和修改示例

### 检查点
- [ ] 关键词选择合理
- [ ] 优化建议可执行
- [ ] 不采用黑帽SEO手段
- [ ] 保持内容自然可读

---

## 质量验证清单（Quality Checklist）

### SEO效果检查
- [ ] 标题包含核心关键词且有吸引力
- [ ] Meta描述包含关键词且诱导点击
- [ ] H标签层级清晰（H1→H2→H3）
- [ ] 关键词密度在2-3%之间
- [ ] 图片有Alt文本描述

### 内容质量检查
- [ ] 内容有价值、解决用户问题
- [ ] 结构清晰、段落分明
- [ ] 重要信息有高亮/加粗
- [ ] 链接布局合理

### 合规性检查
- [ ] 无关键词堆砌
- [ ] 无隐藏文本/小字
- [ ] 无虚假夸张宣传
- [ ] 遵循搜索引擎指南
