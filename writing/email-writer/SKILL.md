# email-writer v3.0

商务邮件撰写技能。2026年集成AI语气调整、多语言邮件生成和智能模板匹配能力。

## 功能说明

创作各类商务邮件（中英文），包括开发信、跟进邮件、邀请函、道歉信、感谢信、报价信、会议邮件等，确保专业、有效、有礼。

---

## 假设声明（Assumptions）

1. **沟通目的明确**：用户清楚写邮件的目的和期望结果
2. **收件人身份已知**：用户了解收件人的身份和与自己的关系
3. **基础信息完备**：用户提供必要的背景信息
4. **适用场景**：中英文商务邮件、工作沟通、客户维护
5. **不支持**：私人情感邮件、法律合同、正式法律文件

---

## 输入输出（I/O Format）

### 输入格式
```
【邮件类型】开发信/跟进信/邀请函/道歉信/感谢信/报价信/会议邮件/确认邮件/通知邮件
【收件人】姓名 + 公司/职位（如：王总，某大型企业采购总监）
【发件人】姓名 + 公司 + 职位
【邮件主题】（可选）XXX
【核心信息】
- 邮件背景：XXX
- 主要诉求：XXX
- 期望结果：XXX
【附件】（可选）是否需要附带文件
【邮件语气】正式/友好/紧迫/诚恳（可选，默认正式）
【语言】中文/英文
【特殊要求】（可选）XXX
```

### 输出格式
```
【邮件类型】XXX
【收件人】XXX
【邮件主题】XXX
【邮件正文】
...
【附件清单】（如有）
【发送建议】
- 最佳发送时间：XXX
- 后续跟进时间：XXX
```

## 2026年新功能

### AI语气调整
- **智能匹配**：根据收件人职位、关系、场景自动推荐语气
- **风格学习**：学习用户写作偏好，保持风格一致性
- **语气检测**：分析邮件语气，给出调整建议
- **跨文化适配**：中英文语气差异自动转换

### 邮件模板生成
- **智能匹配**：根据邮件类型和场景推荐最佳模板
- **一键填充**：提供模板变量，快速生成完整邮件
- **模板定制**：基于行业/公司/个人偏好定制模板
- **版本对比**：生成多个版本供选择

### 多语言邮件
- **中英互译**：一键生成高质量中/英文版本
- **多语言支持**：日语、韩语、德语、法语等商务邮件
- **文化适配**：不同国家/地区的邮件风格差异
- **本地化**：考虑收件人所在地区的表达习惯

## 工作流程（Workflow）

### 阶段一：需求分析
1. **确认邮件类型和目的**
2. **分析收件人画像**（职位、关系、关注点）
3. **确定邮件语气**

### 阶段二：内容规划
4. **梳理邮件要点**（必须/次要/避免信息）
5. **设计邮件结构**（开头→主体→结尾CTA）

### 阶段三：撰写优化
6. **生成邮件初稿**
7. **语气和风格优化**
8. **检查与完善**

### 检查点
- [ ] 邮件目的清晰
- [ ] 语言得体适度
- [ ] 重点信息突出
- [ ] 行动指引明确

## 质量验证清单（Quality Checklist）

### 目的达成
- [ ] 邮件目的明确，收件人能理解
- [ ] 期望行动清晰（回复/点击/下载等）
- [ ] 提供便捷的回复/行动方式

### 语言规范
- [ ] 称呼恰当（职位/姓名称呼正确）
- [ ] 语气符合场景和关系
- [ ] 语言流畅，无病句错字
- [ ] 英文语法正确（如英文邮件）

### 结构格式
- [ ] 段落清晰，长度适中
- [ ] 重点信息突出
- [ ] 结尾有署名和联系方式
