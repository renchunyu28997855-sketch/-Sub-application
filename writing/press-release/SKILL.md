# press-release v3.0

新闻稿与公关稿撰写技能。2026年集成AI新闻稿生成、媒体适配、标题优化等智能能力。

## 功能说明

创作专业的新闻稿和公关稿，包括产品发布、融资公布、活动宣传、危机公关等。

---

## 假设声明（Assumptions）

1. **事件真实**：用户描述的事件是真实发生的
2. **信息可公开**：用户提供的信息属于可对外公开的内容
3. **媒体渠道**：目标媒体是正式新闻媒体
4. **适用场景**：企业新闻稿、融资公告、产品发布、活动报道、危机声明

---

## 输入输出（I/O Format）

### 输入格式
```
【新闻类型】产品发布/融资公告/活动报道/危机声明
【核心事件】XXX
【关键信息】时间/地点/人物/原因/结果
【数据支撑】（可选）XXX
【发布渠道】财经媒体/行业媒体/大众媒体
```

### 输出格式
```
【新闻标题】XXX
【副标题】XXX（可选）
【新闻导语】最重要的5W1H
【正文主体】...
【背景介绍】...
【媒体联系】...
```

---

## 2026年新功能

### AI新闻稿生成
- **智能素材分析**：从关键词/事件描述自动提取新闻要素
- **多版本生成**：一键生成不同角度的报道版本
- **事实核查**：自动检查数据准确性和法律风险

### 媒体适配
- **渠道差异化**：针对财经/行业/大众媒体调整语调和侧重点
- **篇幅自适应**：500字简讯/800字标准稿/1500字深度稿
- **格式标准化**：符合不同媒体的发布规范

### 标题优化
- **吸引力评分**：AI评估标题的点击欲望和媒体采用率
- **多方案备选**：生成5+个标题供选择
- **SEO+新闻性平衡**：兼顾搜索可见性和新闻价值

---

## 工作流程（Workflow）

### 阶段一：信息收集
1. 确认5W1H新闻要素
2. 核实信息准确性

### 阶段二：价值判断
3. 评估新闻价值（时效性/重要性/显著性/接近性/趣味性）
4. 确定报道角度

### 阶段三：内容生成
5. 撰写新闻稿（倒金字塔结构）
6. 优化标题和导语
7. 媒体适配调整

### 检查点
- [ ] 标题有新闻性
- [ ] 导语包含核心5W1H
- [ ] 有权威引用
- [ ] 数据有来源支撑
- [ ] 联系方式完整
