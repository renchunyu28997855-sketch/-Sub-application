# product-desc v3.0

电商产品描述技能。2026年集成AI产品描述生成、卖点智能挖掘、多平台智能适配能力。

## 功能说明

创作电商平台的商品详情页文案，包括标题、主图文案、详情描述、买家秀文案等，支持淘宝/天猫/京东/亚马逊/拼多多/小红书等主流平台。

---

## 假设声明（Assumptions）

1. **产品真实存在**：用户提供真实产品的准确信息
2. **平台明确**：用户指定目标电商平台
3. **合规要求**：产品符合平台和广告法规范
4. **适用场景**：电商详情页、种草内容、产品主图

---

## 输入输出（I/O Format）

### 输入格式
```
【产品名称】XXX
【目标平台】淘宝/京东/亚马逊/拼多多/小红书
【产品类别】XXX（如：数码、美妆、家居、服饰）
【核心卖点】1. XXX 2. XXX 3. XXX
【价格/规格】（可选）XXX
【目标用户】XXX（如：年轻女性、商务人士）
```

### 输出格式
```
【平台】XXX
【产品名称】XXX
【核心卖点】
1. ...
【标题备选】
1. ...
2. ...
【主图文案】
...
【详情页结构】
...
【购买引导】
...
```

## 2026年新功能

### AI产品描述生成
- **智能标题**：基于关键词和转化数据生成高点击率标题
- **卖点挖掘**：从用户评价和竞品分析中提取差异化卖点
- **多版本生成**：一键生成A/B测试版本

### 多平台智能适配
| 平台 | 风格特点 |
|------|----------|
| 淘宝/天猫 | 种草风、强调性价比 |
| 京东 | 品质风、理性数据导向 |
| 亚马逊 | SEO导向、英文专业简洁 |
| 小红书 | 生活化、真实种草感 |

### Human-AI协作
- 初稿生成 → 用户反馈 → 迭代优化 → 最终确认

## 工作流程（Workflow）

### 阶段一：产品分析
1. 理解产品核心功能和差异化卖点
2. 了解目标用户画像和购买决策因素

### 阶段二：平台适配
3. 匹配目标平台的风格和规则
4. 制定主图和详情页策略

### 阶段三：内容创作
5. 撰写标题、主图文案、详情页
6. 优化并检查违禁词

### 检查点
- [ ] 标题含核心关键词且吸引点击
- [ ] 主图突出差异化卖点
- [ ] 详情页逻辑清晰、信任完整
- [ ] 无违禁词/绝对化用语
