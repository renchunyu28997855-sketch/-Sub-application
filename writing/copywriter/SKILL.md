# copywriter

## 功能说明
创作各类营销文案，包括广告语、产品文案、促销海报、销售页等，帮助提升营销效果。

---

## 假设声明（Assumptions）

1. **用户已有明确的产品/服务**：用户提供的产品或服务信息基本完整
2. **目标受众已界定**：用户对目标客户群体有大致的描述
3. **品牌调性已知**：用户了解自己品牌的风格和语气
4. **适用场景**：B2C/B2B产品、电商、服务类、教育类等主流行业
5. **不支持**：完全虚构产品的营销方案（需要至少基础产品信息）

## 详细工作流程（Detailed Workflow）

### 阶段一：需求分析（第1-2步）
1. **接收并确认输入信息**
   - 检查产品/服务描述是否完整
   - 确认目标受众画像
   - 识别核心卖点优先级

2. **补充提问**（如信息不足）
   - 产品差异化特点是什么？
   - 主要竞争对手有哪些？
   - 品牌既往调性如何？

### 阶段二：策略制定（第3-4步）
3. **确定文案策略**
   - 选择主打的1-2个核心卖点作为钩子
   - 确定情感基调（理性/感性/混合）
   - 设计信息层次（吸引→兴趣→欲望→行动）

4. **匹配文案框架**
   - AIDA框架（注意→兴趣→欲望→行动）
   - PAS框架（问题→放大→解决）
   - FAB框架（特点→优势→利益）

### 阶段三：创作执行（第5-7步）
5. **创作广告语**
   - 提供3-5个备选方案
   - 每个方案标注角度和适用场景

6. **撰写正文文案**
   - 主图配合文案（吸引眼球）
   - 详情页结构化内容

7. **优化与调整**
   - 检查卖点是否突出
   - 核实信息准确性
   - 确保语言流畅

### 检查点
- [ ] 核心卖点是否清晰传达
- [ ] 语言风格是否符合品牌调性
- [ ] 是否有明确的行动号召（CTA）
- [ ] 字数/长度是否符合场景要求

## 边界情况（Edge Cases）

### 情况1：输入信息极少
**问题**：用户只提供"帮我写个广告语"
**处理方式**：
- 提供开放式提问模板，收集基本信息
- 或基于常见品类提供通用文案供选择
- 示例提问："请问是什么产品？目标用户是谁？主要想传达什么信息？"

### 情况2：目标受众模糊
**问题**：用户未明确目标人群
**处理方式**：
- 询问主要使用场景
- 提供不同人群的差异化文案版本
- 或按主流人群（都市白领/年轻父母/学生等）提供备选

### 情况3：多产品/多卖点堆叠
**问题**：用户希望一个文案覆盖所有卖点和产品
**处理方式**：
- 指出信息过载的问题，建议聚焦
- 提供分层文案策略（主广告语+详细说明）
- 或拆分多个独立文案

### 情况4：竞品信息缺失
**问题**：不了解竞品，难以突出差异
**处理方式**：
- 引导用户提供或假设一般竞争环境
- 侧重于产品本身优势的绝对表达（非相对竞品）

### 情况5：风格冲突
**问题**：用户要求的风格与品牌调性不符
**处理方式**：
- 礼貌指出潜在问题
- 提供两种版本：用户要求风格 + 建议风格
- 让用户选择

## 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理方式 |
|----------|----------|----------|
| 信息矛盾 | 卖点之间逻辑冲突 | 询问确认，优先保留用户后提供的信息 |
| 虚假宣传 | 卖点无法实证 | 标注风险，建议修改为可证实表达 |
| 侵权风险 | 提及品牌名/专利等 | 删除敏感信息，替换为泛化表达 |
| 法规违规 | 违禁词/绝对化用语 | 自动过滤并提供替代方案 |
| 编码错误 | 特殊符号/emoji显示异常 | 使用标准UTF-8字符 |

## 质量验证清单（Quality Checklist）

### 文案效果检查
- [ ] **吸引力**：开头能否在3秒内抓住注意力？
- [ ] **清晰度**：核心卖点是否一目了然？
- [ ] **差异化**：是否与市面上常见文案有明显区别？
- [ ] **可信度**：表达是否过于夸张？是否有数据支撑？
- [ ] **行动号召**：是否有明确的下一步引导？

### 语言规范检查
- [ ] **语法正确**：无错别字、病句
- [ ] **标点规范**：中英文标点正确使用
- [ ] **敏感词过滤**：无违禁词/绝对化用语
- [ ] **品牌一致性**：是否符合品牌既有调性

### 合规性检查
- [ ] **广告法合规**：无"最"、"第一"、"国家级"等绝对化用语
- [ ] **知识产权**：无抄袭/侵权内容
- [ ] **行业特殊规定**：如食品/医疗/金融等行业的特殊要求

### 失败案例及修正

**原始文案**（问题文案）：
"XX空气炸锅，最最好的空气炸锅，销量第一，错过后悔一辈子！"

**问题分析**：
1. "最最好的" - 语法错误
2. "销量第一" - 绝对化用语，违反广告法
3. "错过后悔一辈子" - 夸大其词，降低可信度

**修正后**：
```
【广告语备选】
1. "少油80%，却一样酥脆！"
2. "炸鸡炸薯条，少油更尽兴"
3. "厨房小白也能变大厨"

【产品详情开头】
还在为想吃炸食又怕胖而纠结？
空气炸锅让你告别油烟，享受美味无负担！

（对比原文案：去除了绝对化用语，保留了夸张的适度表达，增加了画面感和共鸣）
```

## FAQ（常见问题）

**Q1：广告语一般多长合适？**
A：取决于使用场景。
- 电梯广告/短视频：5-12字
- 朋友圈广告：15-25字
- 详情页主图：20-40字
- 品牌口号：4-10字

**Q2：如何判断文案是否有效？**
A：核心指标包括：
- 注意力停留时长（开头是否吸引）
- 信息传达率（卖点是否被记住）
- 行动转化率（是否有引导效果）
建议进行A/B测试验证。

**Q3：同一产品需要准备多少套文案？**
A：建议至少3套：
- 主打版本（最推荐的方案）
- 备选版本（不同角度）
- 测试版本（创新尝试）

**Q4：文案风格与品牌冲突怎么办？**
A：尊重品牌既有调性优先。如果用户要求的风格与品牌不符，提供两版对比并说明风险。

**Q5：是否可以提供方言/地区化文案？**
A：可以。需要用户明确目标地区，我会在用词、表达上做本地化适配。

**Q6：涉及价格促销的文案有什么规范？**
A：需确保：
- 价格信息准确
- 促销时间明确
- 条件说明清晰（如"限量"需真实）
- 符合电商平台促销规范

## 文案类型

- **广告语（Slogan）**：品牌口号，一句话打动人心
- **产品文案**：产品卖点挖掘与表达
- **促销文案**：节日促销、活动宣传
- **销售页文案**：完整销售页结构化写作
- **详情页文案**：电商品详情页内容

## 工作流程
1. 了解产品/服务特点和目标受众
2. 挖掘核心卖点和差异化
3. 确定文案风格和调性
4. 创作初稿
5. 优化直到满意
