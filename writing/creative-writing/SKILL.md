# creative-writing v3.0

创意写作技能。2026年集成情节生成器、人物关系图谱、风格模仿、多结局分支能力。

## 功能说明
帮助用户进行各类创意写作，包括故事创作、小说、诗歌、剧本等。

---

## 假设声明（Assumptions）

1. **创作目的明确**：用户有大致的故事想法或主题
2. **风格偏好可知**：用户了解想要的风格
3. **长度可协商**：用户能接受的作品长度
4. **适用类型**：短篇小说、微小说、诗歌、剧本、故事大纲

---

### 输入格式
```
【作品类型】短篇小说/微小说/诗歌/剧本/故事大纲/人物设定
【主题/想法】XXX

【篇幅】XXX字/XXX分钟
【目标读者】（可选）XXX
```

## 工作流程（Workflow）

### 阶段一：创意沟通
1. **理解创作想法** - 用户的故事主题、核心人物设想、情感基调
2. **确定作品规格** - 作品类型、篇幅长度、目标读者

### 阶段二：策划设计
3. **设计故事结构** - 情节主线、转折点设计、结尾处理
4. **塑造人物** - 人物背景、性格特点、人物弧线

### 阶段三：创作执行
5. **撰写初稿** - 开头的吸引力、情节推进、结尾的余韵
6. **修改优化** - 语言精炼、情感表达、细节打磨

## 质量验证清单（Quality Checklist）

### 故事性检查
- [ ] 有吸引力的开头
- [ ] 清晰的情节推进
- [ ] 有转折或高潮
- [ ] 有意义的结尾

### 人物性检查
- [ ] 人物有立体感
- [ ] 对话自然
- [ ] 动机合理
- [ ] 成长/变化可见

### 文字性检查
- [ ] 语言流畅
- [ ] 描写生动
- [ ] 情感真挚
- [ ] 无语法错误
