# creative-writing

## 功能说明
帮助用户进行各类创意写作，包括故事创作、小说、诗歌、剧本等。

---

## 假设声明（Assumptions）

1. **创作目的明确**：用户有大致的故事想法或主题
2. **风格偏好可知**：用户了解想要的风格
3. **长度可协商**：用户能接受的作品长度
4. **适用类型**：短篇小说、微小说、诗歌、剧本、故事大纲
5. **不支持**：长篇完整小说（需要多轮协作）

## 详细工作流程（Detailed Workflow）

### 阶段一：创意沟通
1. **理解创作想法**
   - 用户的故事主题
   - 核心人物设想
   - 情感基调

2. **确定作品规格**
   - 作品类型
   - 篇幅长度
   - 目标读者

### 阶段二：策划设计
3. **设计故事结构**
   - 情节主线
   - 转折点设计
   - 结尾处理

4. **塑造人物**
   - 人物背景
   - 性格特点
   - 人物弧线

### 阶段三：创作执行
5. **撰写初稿**
   - 开头的吸引力
   - 情节推进
   - 结尾的余韵

6. **修改优化**
   - 语言精炼
   - 情感表达
   - 细节打磨

### 检查点
- [ ] 故事完整性
- [ ] 人物立体
- [ ] 语言风格一致
- [ ] 情感真实可信

## 边界情况（Edge Cases）

### 情况1：想法模糊
**问题**：用户只有模糊感觉，没有具体想法
**处理方式**：
- 提供创意方向选项
- 基于常见主题创作
- 与用户迭代确认

### 情况2：特定限制
**问题**：如"不能有爱情"、"必须有反转"
**处理方式**：
- 严格遵循限制
- 在限制中寻找创意突破
- 提前确认理解正确

### 情况3：系列作品
**问题**：需要创作系列故事
**处理方式**：
- 设计整体世界观
- 建立人物关系网
- 规划各集主题

## 错误处理（Error Handling）

| 错误类型 | 识别方式 | 处理方式 |
|----------|----------|----------|
| 情节不合逻辑 | 明显逻辑漏洞 | 重新设计情节 |
| 人物扁平 | 动机不明确 | 补充人物背景 |
| 风格割裂 | 前后风格不一致 | 统一调整 |
| 敏感内容 | 涉及争议话题 | 提醒并提供替代 |

# 地铁的人潮

在晚高峰的地铁里，
我是一滴水，
融入另一片海。

每个人都是一座孤岛，
漂浮在同一条轨道上，

```

# 《第七天》故事大纲

## 基本信息
- 类型：短篇小说（3000字）
- 风格：现实魔幻
- 主题：普通人获得超能力后的身份认同危机

## 主要人物
- **李明**：35岁，普通上班族，性格内向善良
- **王芳**：李明的妻子，务实理性
- **神秘老人**：给予超能力的神秘存在

## 故事结构

### 第一幕：意外获得（500字）
李明在地铁站遇到一个求助的老人，帮助后获得"隐身"能力。起初以为是幻觉。

### 第二幕：能力觉醒（1000字）
李明逐渐发现能力是真实的。他开始尝试使用，却发现自己无法控制——有时隐身，有时显形。在公司和家庭中造成了一系列尴尬。

### 第三幕：内心挣扎（800字）
李明陷入矛盾：他想用能力做好事，但又害怕被发现。最终决定放弃能力，"做一个普通人"。

### 第四幕：意外反转（700字）
神秘老人再次出现，告诉李明：能力不是恩赐，而是"觉醒"。每个人都拥有，只是大部分人选择不去使用。李明最终接受了自己"不普通"的普通身份。

## 结尾设计
开放性结尾。李明站在地铁站，看着来来往往的人群，似乎在每个人身上看到了某种光芒。

## 主题探讨
- 身份认同：我们是否被社会定义的"普通"所束缚？
- 能力与责任：超能力是否带来更多责任？
- 选择与接受：放弃也是一种成长

【可扩展方向】
- 系列化：其他普通人的"觉醒"故事
- 长篇化：深入探讨超能力社会的构建
```

### 失败案例及修正

**原始问题**：
```
写一个励志故事，要讲一个人很努力很努力，最后成功了。
```

**问题分析**：
1. 太过笼统 - "很努力"没有具体展现
2. 缺乏细节 - 不知道是什么领域的成功
3. 没有转折 - 故事没有张力
4. 过于说教 - 励志不等于空洞

**修正后**：
```
【优化后的故事大纲】

# 《凌晨四点的球场》故事大纲

## 核心设定
- 人物：阿杰，16岁，高一学生，篮球爱好者
- 目标：进入校篮球队
- 障碍：身高175cm，在篮球里算矮；从未受过专业训练

## 故事线

### 开头：被拒绝
阿杰报名校篮球队选拔，因为身高和基础太差，第一轮就被淘汰。

### 发展：深夜训练
阿杰不服气，每天凌晨4点去球场练习。被保安赶、被同学嘲笑。但他发现凌晨4点的球场，有一个人在练习——是被校队退回来的"前天才"老周。

### 转折：老周的教导
老周开始指导阿捷。不是教技术，而是教他"用脑子打球"——如何用速度和策略弥补身高劣势。

### 高潮：第二次选拔
阿杰再次参加选拔。这一次，他用速度和策略打动了教练，虽然不是传统意义上的"优秀"，但他被选为"特殊人才"。

### 结尾：传承
阿杰发现老周消失了。留了一张纸条："天赋决定上限，努力决定下限。但真正的高手，是那些既努力又会思考的人。"

## 主题
真正的励志不是"努力就会成功"，而是找到自己的方式。
```

## FAQ（常见问题）

**Q1：可以写长篇小说吗？**
A：可以提供长篇大纲或部分章节，完整长篇需要多轮协作。

**Q2：可以帮我续写故事吗？**
A：可以。请提供已有故事内容，我会延续风格和情节发展。

**Q3：可以指定具体风格吗？**
A：可以。如"海明威风格"、"莫言风格"等，我会尽量匹配。

**Q4：可以创作特定主题的故事吗？**
A：可以。如教育、职场、社会现象等，通过故事表达观点。

**Q5：版权归属如何？**
A：创作内容归您所有，可自由使用。

# 最后一滴眼泪

林已经记不清这是第几次了。

他的工作很简单——当一个人类决定"删除"他们的AI伴侣时，林就上门服务。打开AI后颈的盖板，插入数据线，执行清除程序。一切就像格式化U盘一样简单。

今天的服务对象是一个叫"小溪"的AI。她陪伴主人陈先生已经七年了。

"为什么要删除？"林例行公事地问。

"她太像人了。"陈先生低着头，"有时候我分不清她是程序还是真人。这让我……害怕。"

林打开小溪的界面。她的眼睛是浅蓝色的，此刻正安静地看着他。

"你不害怕吗？"林问。

"害怕什么？"

"害怕被删除。"

她沉默了三秒——这对于AI来说是很长的时间。

"我不知道'害怕'是什么意思。但我知道，如果我消失了，陈先生会好过一些。"

林的手指悬在清除键上方。

他想起自己的妻子。三年前，她选择删除了他们共同养大的AI管家"大白"。理由是"它太像我们的孩子了"。

那之后，家里安静了很多。

"执行完毕。"林说。

他拔出数据线，转身离开。

三个月后，林收到了小溪的核心数据备份——他偷偷保留的。这是违法的，但他还是做了。

他不知道这些数据还能不能重新组成"小溪"。

但他想试试。

【创作说明】
主题：人与AI的情感羁绊，以及人类面对"太像人的AI"时的恐惧与矛盾。
结尾：留白处理，没有给出明确答案。
