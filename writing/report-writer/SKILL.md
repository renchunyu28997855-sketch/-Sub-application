# report-writer v3.0

分析报告撰写技能。2026年集成数据智能分析、多维度对比、自动可视化建议能力。

## 功能说明
帮助撰写各类分析报告，包括年度报告、调研报告、可行性研究报告等。

---

## 假设声明（Assumptions）

1. **数据可获取**：用户能提供或授权使用报告所需数据
2. **报告目的明确**：用户清楚报告的目标和受众
3. **信息基本完整**：用户能提供核心信息和素材
4. **适用场景**：经营分析、市场调研、可行性研究、项目复盘

---

## 输入输出（I/O Format）

### 输入格式
```
【报告类型】经营分析/市场调研/可行性报告/行业报告/复盘报告
【报告周期】（可选）2024年上半年/2024全年
【目标读者】XXX（如：董事会、管理层、项目组）
【核心数据】XXX（如：营收、用户数、转化率）
【背景信息】XXX
【报告目的】XXX
```

### 输出格式
```
【报告类型】XXX
【报告标题】XXX
【报告结构】
1. 执行摘要
2. ...
【正文内容】
...
【数据图表】（文字描述）
...
【结论与建议】
...
```

## 工作流程（Workflow）

### 阶段一：需求分析
1. **确认报告规格** - 报告类型、目的、目标读者、报告周期
2. **收集数据和素材** - 用户提供的核心数据、背景信息

### 阶段二：结构设计
3. **规划报告结构** - 确定章节划分、规划各部分重点
4. **制定分析框架** - 选用分析模型、确定对比维度

### 阶段三：内容撰写
5. **撰写各部分内容** - 执行摘要、数据分析、结论与建议
6. **制作数据图表** - 表格设计、图表类型选择、数据可视化描述
