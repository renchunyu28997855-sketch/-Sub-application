# chart-designer v3.0

图表设计技能。2026年集成智能图表推荐、自动布局优化、色盲友好配色能力。

## 功能说明
帮助设计数据可视化图表，提供图表类型选择、布局设计、配色方案等专业建议。

---

## 假设声明（Assumptions）

1. **数据已准备好**：用户有需要可视化的数据
2. **目的明确**：用户知道图表的用途（PPT/报告/网页）
3. **受众可知**：用户了解图表面向的观众
4. **适用场景**：商务报告、PPT演示、数据分析、网页图表

---

## 输入输出（I/O Format）

### 输入格式
```
【图表用途】PPT/报告/网页/文章
【数据类型】趋势/对比/占比/关系/分布
【数据内容】XXX（数据描述或具体数据）
【目标受众】XXX（如：管理层、客户、技术团队）
【图表数量】单个/多个
【特殊要求】（可选）配色偏好/尺寸要求
```

### 输出格式
```
【推荐图表类型】XXX
【图表设计】
- 类型：XXX
- 布局：XXX
- 配色：XXX
- 标签：XXX
【图表描述】
...
【制作建议】
...
```

## 工作流程（Workflow）

### 阶段一：需求理解
1. **确认图表用途** - PPT/报告/网页/文章
2. **分析数据特征** - 数据类型、数据量、维度数量

### 阶段二：图表设计
3. **选择图表类型** - 根据数据特征推荐最佳类型
4. **设计布局排版** - 单图/多图、标题位置、图例位置

### 阶段三：视觉优化
5. **配色方案** - 品牌色/主题色、色盲友好
6. **标签与注释** - 标题、轴标签、注释、来源标注
