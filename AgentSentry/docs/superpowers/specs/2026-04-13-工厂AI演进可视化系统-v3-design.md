**状态**: 待批准

## 一、项目概述

### 1.1 目标
在现有数字孪生 Dashboard 基础上，新增工厂 AI 演进可视化模块，展示工厂从"全人力"逐步演进到"人机协作"的完整过程。

### 1.2 核心概念
- **审查Agent**：居中，发出审查光波，监控各部门KPI
- **进化光波**：从中央向全场扩散，所有部门同时被波及
- **演进机制**：被波及的部门开始演进，人变少、光点变多
- **人机排列**：人形在上、机器人在下，左右分开
- **颜色变化**：部门颜色根据演进百分比从橙色渐变到绿色

## 三、Agent 设计

### 3.1 审查 Agent
- **位置**：画布中央 (x: 0.5, y: 0.5)
- **形态**：机器人 + 紫色光环
- **职责**：发出审查光波，监控各部门KPI
- **颜色**：紫色 #9b59b6

### 3.2 光波扩散机制
- **进化光波**：从中央向全场扩散的圆形波纹，青绿色渐变
- **审查光波**：从中央向全场扩散的圆形波纹，紫色
- **扩散方式**：同时扩散到所有部门，所有部门同时被波及
- **视觉效果**：波纹经过部门时，部门有闪烁/光圈效果

---

## 四、光波扩散机制

### 4.1 进化流程
```
审查Agent发出进化光波
        ↓
光波从中央向全场扩散
        ↓
所有部门同时被波及
        ↓
各部门显示"进化中"效果
        ↓
各部门开始演进动画：
  - 人减少（从数个到1个）
  - 光点增加（围绕机器人）
  - 颜色渐变（橙色→绿色）
```

### 4.2 审查机制
- 审查光波先于进化光波发出
- 被审查光波波及的部门显示"被审查中"效果
- KPI达标后进入演进

---

## 五、部门演进动画

### 5.1 人机排列
- **人形在上方**：显示人形图标，初始5个，随演进减少，最少1个
- **机器人在下方**：单个机器人固定在下方
- **左右分开**：人形在左侧，机器人在右侧

### 5.2 演进动画
- **演进开始**：被进化光波波及
- **人形减少**：从5个逐步减少到1个
- **光点增加**：⚡和🔧围绕机器人，人减少几个，光点增加几个
- **颜色变化**：部门整体颜色从橙色根据百分比渐变到绿色

### 5.3 颜色渐变规则
- **橙色 #ff6b35**：演进0-20%
- **黄色 #ffcc00**：演进20-50%
- **青色 #00d4ff**：演进50-80%
- **绿色 #00ff88**：演进80-100%

## 六、光波可视化

### 6.1 进化光波
- **颜色**：青绿色渐变 #00d4ff → #00ff88
- **形态**：从中央扩散的圆形波纹
- **宽度**：波纹线条宽度渐变
- **速度**：3-5秒扩散到全场

### 6.2 审查光波
- **颜色**：紫色 #9b59b6
- **形态**：从中央扩散的圆形波纹
- **先于进化光波发出**

### 6.3 部门被波及效果
- **进化中**：绿色/青色光圈包围部门，脉冲闪烁
- **审查中**：紫色光圈包围部门，脉冲闪烁

## 七、技术实现

### 7.1 光波扩散类
```javascript
class ExpansionWave {
    constructor(centerX, centerY, maxRadius, color, duration) {
        this.centerX = centerX;
        this.centerY = centerY;
        this.radius = 0;
        this.maxRadius = maxRadius;
        this.color = color;
        this.duration = duration;
        this.elapsed = 0;
        this.alive = true;
    }
    update(dt) {
        this.elapsed += dt;
        this.radius = (this.elapsed / this.duration) * this.maxRadius;
        if (this.elapsed >= this.duration) this.alive = false;
    }
    draw(ctx) {
        if (!this.alive) return;
        ctx.save();
        ctx.strokeStyle = this.color;
        ctx.lineWidth = 4;
        ctx.globalAlpha = 1 - (this.radius / this.maxRadius);
        ctx.beginPath();
        ctx.arc(this.centerX, this.centerY, this.radius, 0, Math.PI * 2);
        ctx.stroke();
        ctx.restore();
    }
    // 检查部门是否被波及
    isAffected(deptX, deptY) {
        const dx = deptX - this.centerX;
        const dy = deptY - this.centerY;
        return Math.sqrt(dx*dx + dy*dy) <= this.radius;
    }
}
```

### 7.2 部门状态
```javascript
{
    id: 'production',

    x: 0.55, y: 0.2,
    humanCount: 5,      // 初始人形数量
    aiRate: 10,        // AI替代率 0-100
    evolutionProgress: 0, // 演进进度 0-100
    inEvolution: false,   // 是否在演进中
    underReview: false,  // 是否在被审查中

}
```

### 7.3 演进计算
- **人形减少**：`max(1, humanCount - (aiRate / 100) * (humanCount - 1))`
- **光点增加**：`Math.round((aiRate / 100) * 5)` 个
- **颜色**：`getMaturityColor(aiRate)`
