# 工厂 AI 演进可视化系统设计规范

**版本**: v1.0
**日期**: 2026-04-13
**状态**: 已批准

---

## 一、项目概述

### 1.1 目标
在现有数字孪生 Dashboard 基础上，新增工厂 AI 演进可视化模块，展示工厂从"全人力"逐步演进到"人机协作"的完整过程，包括 Agent 工作流、Skill/MCP 调用、人机协作等元素的可视化。

### 1.2 核心概念
- **演进 Agent**：核心驱动力，悬浮顶部俯视，发射进化光芒推动演进
- **AI 替代率**：每个部门的 AI 化程度（0-100%）
- **磨合期**：新替代后的适应阶段，颜色渐变反映成熟度
- **事件触发**：成熟度达标后自动触发下一波演进

## 三、可视化元素设计

### 3.1 人员与 Agent 形态

| 元素 | 形态 | 说明 |
|------|------|------|
| 人力 | 火柴人 | 基础工作节点，线条人形 |
| Agent | 机器人 | 几何体机器人，六边形身体 |
| 演进 Agent | 大型机器人 + 光环 | 悬浮顶部，核心控制器 |

### 3.2 颜色系统

**磨合期颜色渐变：**
- 橙色 (#ff6b35)：新替代，生疏
- 黄色 (#ffcc00)：适应中
- 青色 (#00d4ff)：熟练
- 绿色 (#00ff88)：完全融合，成熟

**其他颜色：**
- 人力基础色：#8b9cb3（灰蓝）
- Agent 基础色：#00d4ff（青）
- Skill 装备：#00ff88（绿）
- MCP 工具：#ff6b35（橙）
- 进化流：渐变青绿色粒子

### 3.3 节点大小
- 人力节点：50px
- Agent 节点：60px
- 演进 Agent：100px（核心节点）

## 四、演进机制

### 4.1 磨合期规则

```
新替代触发磨合期
    ↓
橙色 (0-20%) → 黄色 (20-50%) → 青色 (50-80%) → 绿色 (80-100%)
    ↓
成熟度达到 80% 以上
    ↓
触发下一波演进事件
```

### 4.2 演进 Agent 行为

- **监听**：持续监控各部门成熟度
- **决策**：当某部门成熟度达标，发出进化流
- **执行**：发射迭代粒子，感染相邻人力节点

### 4.3 迭代流可视化

- 从演进 Agent 发射
- 粒子颜色：渐变青绿
- 流经路径高亮
- 触碰到人力节点时，人开始"变形"动画（0.5秒）

---

## 五、Skill 与 MCP 可视化

### 5.1 装备系统

每个 Agent 可挂载多个 Skill/MCP 装备：

| 类型 | 图标 | 颜色 |
|------|------|------|
| Skill | ⚡ | 绿色 |
| MCP | 🔧 | 橙色 |

### 5.2 装备装配动画

1. 装备图标从左侧库飞向 Agent
2. 接触 Agent 时"咔嗒"一声
3. Agent 颜色闪烁一下
4. 装备固定在 Agent 身上

### 5.3 调用链路

当 Agent 调用 Skill/MCP 时：
- 虚线从 Agent 指向目标
- 装备图标沿线移动到目标
- 目标接收后反馈信号（绿色脉冲）

---

## 六、人机协作可视化

### 6.1 协作连接

- 人力和 Agent 协作时，用**虚线**连接
- 虚线上有小圆点流动，表示信息交换
- 协作紧密时连线发光

### 6.2 分工表示

- 人力处理：创意性/决策性任务
- Agent 处理：重复性/数据分析任务
- 虚线两端分别标注"人"或"AI"

---

## 七、动画规范

### 7.1 进化流动画

```
粒子数量：20-30
粒子速度：100-200px/s
粒子大小：3-6px
颜色：从青色渐变到绿色
路径：从演进 Agent 到目标部门
```

### 7.2 人力变形动画

```
时长：500ms
效果：火柴人轮廓 → 几何化 → 机器人形态
颜色：从灰蓝 → 橙色（进入磨合期）
```

### 7.3 磨合期脉冲

```
周期：2s
效果：节点发光强度在 50%-100% 之间脉动
```

### 7.4 协作虚线流动

```
粒子数量：3-5
速度：50px/s
方向：双向
```

## 八、交互控制

### 8.1 部门配置面板

点击任意部门节点，打开配置面板：
- AI 替代率滑块（0-100%）
- 磨合期开关
- 强制触发演进按钮

### 8.2 全局控制

- 演进速度：快（2s）/ 中（5s）/ 慢（10s）
- 暂停/继续按钮
- 重置按钮

### 8.3 事件日志

右侧面板显示演进事件：
```
[12:34:56] 🔵 质检部门磨合期完成，触发下一波演进
[12:35:02] 🟢 质检部门 AI 替代率: 25% → 40%
[12:35:08] ⚡ 质检 Agent 装配 Skill: 视觉检测
```

## 十、技术实现

### 10.1 Canvas 渲染

- 使用 Canvas 2D API 绘制所有动画
- 60 FPS 动画循环
- 节点缓存优化

### 10.2 动画系统

- requestAnimationFrame 驱动
- 缓动函数处理过渡
- 粒子系统管理大量粒子

### 10.3 状态管理

```javascript
// 部门状态
{
  id: 'quality',
  name: '质检部门',
  aiRate: 25,          // AI 替代率 0-100
  maturityLevel: 0,     // 成熟度 0-100
 磨合期: true,          // 是否在磨合期
  nodeType: 'agent',    // 'human' | 'agent' | 'hybrid'
  skills: [],           // 装配的 Skill
  mcps: [],             // 装配的 MCP
  collaborators: []      // 协作的人力节点
}
```
