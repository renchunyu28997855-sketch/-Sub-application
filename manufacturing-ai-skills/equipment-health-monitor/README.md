# Equipment Health Monitor

设备健康监控技能应用 - 智能设备状态监测、实时健康评分、异常告警

## 功能特性

- 设备状态监测：实时监控设备运行参数
- 健康评分：基于多维度指标计算设备健康评分
- 异常告警：智能识别异常并提供告警
- 维护建议：提供立即处理、短期计划、长期规划建议

## 技术栈

- Next.js 16.2.3
- React 19.2.4
- TypeScript
- MiniMax API

## 环境变量

```bash
MINIMAX_API_KEY=your_api_key_here
MINIMAX_GROUP_ID=your_group_id_here
```

## 开发

```bash
npm install
npm run dev
```

访问 http://localhost:3000/equipment-health-monitor

## 部署

项目部署在 /equipment-health-monitor 路径下，basePath 已配置。
